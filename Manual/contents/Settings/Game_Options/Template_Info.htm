Informações do modelo

Esta seção permite definir as informações para seu projeto que são utilizadas quando é exportado como um modelo.

Aqui você pode atribuir uma imagem, um ícone e uma descrição para o projeto modelo, porém isto só será usado quando for realmente exportado como um modelo. Você pode fazer isso abrindo o Menu Arquivo, pairando em "Export Project" e selecionando "As Template".
