O Editor Room

O editor room é onde você cria suas salas de jogos. Cada jogo requer pelo menos um room para ser executado, e no room você pode colocar instâncias, sprites, tiles, paths e fundos. Cada um destes diferentes assets pode ser colocado em sua própria camada única que pode então ser encomendada como você desejar no editor de camadas. Devido à complexidade do editor room, primeiro lhe daremos uma breve visão geral das características mais importantes, e depois você poderá encontrar detalhes mais detalhados nos links da seção listados abaixo.

Ao criar um recurso room, você pode clicar com o botão direito do mouse sobre ele no navegador Asset para abrir o menu de contexto, o que lhe permitirá criar a criança rooms (veja a página sobre Herança para mais informações), abrir o room para edição, adicionar um novo grupo de recursos para organizar melhor o rooms, renomear o room ou excluí-lo. Note que para alterar a ordem room e/ou herança você precisa usar o Gerente de Sala, que você pode abrir usando o menu no canto superior direito do navegador Asset.

O editor room é em si um espaço de trabalho e como tal você pode clicar na aba e arrastá-lo da janela principal para uma nova janela própria - talvez em outra tela, por exemplo. Você também pode colocá-lo de volta na janela principal arrastando a aba para a parte superior do IDE e soltando o botão do mouse.

A interface do usuário para o editor room é simples de navegar e dividida em várias seções discretas. As partes do editor que estão acopladas - as propriedades da sala e do editor de camadas, assim como as diferentes seções de propriedades de camadas - também podem ser removidas da doca simplesmente arrastando-as para fora no workspace, e podem ser adicionadas novamente nas docas arrastando-as para os lados ou para o fundo do workspace.

Abaixo está uma breve visão geral de cada uma das seções do editor Room:

Réguas / Guias

As réguas ao longo da borda da tela mostram a posição das coisas que são colocadas dentro dela, e são marcadas de (0, 0) que é o centro da tela e a origem para o room. Você pode clicar e arrastar sobre as réguas para puxar uma guia horizontal ou vertical para o room, e esta guia pode então ser usada para posicionar com precisão os diferentes assets que estão sendo usados, já que movendo um asset próximo a um o "estalará" para a guia. Ao posicionar assets dentro do room, as guias inteligentes também serão mostradas temporariamente indicando a distância entre assets, bem como a distância do limite ou ponto central do room.

A distância que assets "snap" para guias, assim como outras propriedades, podem ser definidas nas Preferências do Editor de Sala.

Editor de camadas

O editor room coloca as coisas em camadas dentro do room. Cada camada está em uma discreta "profundidade", onde as que aparecem na parte inferior da lista na janela da camada serão desenhadas sob as que aparecem perto da parte superior.

IMPORTANTE! Há uma profundidade mínima e máxima de camada de -16000 a 16000. Qualquer coisa colocada em uma camada fora dessa faixa não será desenhada, embora todos os eventos ainda funcionem normalmente.

As camadas são criadas clicando no botão apropriado para o tipo de camada que você deseja criar, que são:

Instâncias Esta camada é para colocar instâncias que serão utilizadas no room.
Mapas de azulejos Esta camada é para colocar tiles de conjuntos de azulejos comomapas de azulejos.
Caminhos Esta camada é para mostrar e adicionar caminhos no site room.
Ativos Esta camada é para que o visual assets seja adicionado ao room, como sprites e seqüências.
Filtro/efeito Este tipo de camada define um filtro/efeito que é aplicado a todas as camadas abaixo dele.
Antecedentes Este tipo de camada define um fundo, que é essencialmente uma única cor ou uma imagem sprite que pode ser azulejada sobre o conjunto room.
UI This creates a global UI layer that is shared among rooms.

Você também pode criar uma pasta de camadas usando o botão de pasta onde você pode agrupar as camadas selecionadas, assim como excluir as camadas selecionadas com o botão excluir . Cada um destes tipos de camadas é discutido aqui com mais detalhes.

Observe que você pode alternar herança para o editor de camadas que afetará a ordem e a visibilidade das camadas. A visibilidade em si pode ser definida clicando no ícone do olho ao lado de cada uma das camadas, ou você pode clicar na que se encontra no topo para ativar/desativar todas as camadas de uma só vez. Você também tem a possibilidade de "bloquear" uma camada (ou todas elas) de modo que não possam ser alteradas clicando no ícone do cadeado . Note que as camadas que estão marcadas como invisíveis não serão mostradas quando o jogo for executado.

Propriedades da camada

A janela de propriedades da camada room mudará dependendo da camada atualmente selecionada no editor de camadas.

Cada janela terá as diferentes propriedades e listas associadas ao tipo de camada atualmente selecionado, e você poderá editar detalhes fundamentais sobre como a camada é exibida e o que está realmente sobre a camada. Veja aqui para mais detalhes.

Propriedades dos quartos

Esta seção do editor room é onde você pode definir todas as propriedades básicas para o room, como seu tamanho, se usa física, ou qualquer câmera e viewports que necessite. Esta seção é ditada aqui com mais detalhes.

Caixa de ferramentas em camadas

Alguns tipos de camadas terão ferramentas adicionais adicionadas ao topo do IDE na caixa de ferramentas de camadas (por exemplo, tile camadas ou path camadas). As ferramentas exatas mudarão com base no tipo de camada que está sendo editada atualmente, e assim são explicadas na página que explica cada camada, aqui.

Caixa de ferramentas de sala

Na parte superior da tela principal do editor Room você tem alguns controles para lidar com a forma como as coisas são exibidas. Eles são:

  • Toggle Canvas Grid: Isto irá ligar/desligar a grade de tela do editor Room. Esta é uma grade que o GameMaker desenha sobre a tela principal para dividi-la em seções, e por padrão é definida para 32x32px em tamanho. Entretanto, se você clicar no ícone do Menu Grid , você abrirá as opções da grade:Estas opções permitem que você defina a cor e o alfa da grade, assim como os valores das células da grade ao longo dos eixos X e Y. Você também tem uma opção para ativar ou desativar o snapping da grade aqui (ativado por padrão). Você pode usar os atalhos de teclado "G" e + "G" para alternar a visibilidade da grade e o snapping da grade, respectivamente.
  • Controles de zoom de lona: Estes botões controlam o nível atual de zoom da lona. Você pode aumentar ou diminuir o zoom e clicar no botão irá redefinir a tela para 1:1 com o room sendo editado. Você também pode clicar no botão Ajustar Janela para fazer com que toda a tela room caiba dentro do editor atual workspace (isto fará com que o zoom in/out seja o apropriado para fazê-lo caber). Note que você também pode aumentar e diminuir o zoom usando o / e a Roda do Rato , e ao pressionar / + a tela room será configurada para ser 1:1 com as configurações room.
  • Mostrar vistas: Clicando nisto, o retângulo de visualização será ativado ou desativado. Quando ativado, você verá uma área em destaque que significa as diferentes visões que estão habilitadas para o room.
  • Reproduzir Animações: Ao adicionar sprites, sequences ou tiles animado, você pode querer ter uma idéia de como eles ficarão dentro do próprio room sem ter que compilar o projeto inteiro, e assim você pode clicar nisto para iniciar todas as diferentes animações tocando. Clicando novamente, a animação será interrompida.
  • Selecione uma tela de qualquer camada: Por padrão quando você clica em um elemento na tela room você só poderá selecionar aqueles assets que estão na camada que está sendo editada atualmente, porém se você ativar esta opção você pode então clicar em qualquer elemento e ele será selecionado, mudando a camada alvo para aquela em que o elemento está. Você pode usar o atalho de teclado " P" para habilitar temporariamente isto (segure "P" para habilitar e liberar para desabilitar)

Lona de sala

O centro da janela do editor room é ocupado com a área onde toda a edição propriamente dita acontece. Aqui é onde você estará colocando suas instâncias e assets, desenhando seu tiles, ou posicionando seu paths. Você pode aumentar e diminuir o zoom usando a roda do mouse ou os controles room na parte superior, e você pode girar ao redor segurando o botão central do mouse para baixo e movendo o mouse ao redor ou usando + enquanto se movimenta.

Ao criar asset camadas ou instâncias, você pode colocar o asset ou instância simplesmente arrastando-o do navegador Asset e depois soltando-o onde você quer que ele seja posicionado. Alternativamente, você pode selecionar um asset ou instância do navegador Asset e então pressionar e segurar para visualizar o recurso "in-situ", e se você clicar adicionalmente no botão esquerdo do mouse você pode adicionar a instância e até "pintá-la" na camada room mantendo o botão pressionado e arrastando o mouse. Note que por padrão as instâncias e sprites que são adicionadas a um room desta forma serão encaixadas nas configurações definidas da grade room, mas você pode manter pressionado / para que a instância siga o mouse precisamente e não encaixe (liberando / permitirá encaixar novamente).

Para paths você pode criar uma nova camada path e um recurso path diretamente do editor Room - ou criar um novo recurso path e depois arrastá-lo para a janela principal do editor como você faria por exemplo - e depois editar o path e suas conexões na janela do editor também. Para tile sets, você pode "pintá-los" a partir do editor tile set.

Observe que você pode selecionar e mover ou excluir vários assets da mesma camada segurando / + nos que desejar, e a maioria dos recursos do editor room lhe permitirá girá-los e escalá-los (individualmente ou em grupo) clicando e segurando ao longo das bordas ou cantos externos e arrastando o mouse. Você também pode clicar duas vezes em em qualquer asset (tiles, instâncias, sprites) para que o editor faça a rotação e abra a janela de propriedades para o asset selecionado.

Barra de status

A barra de status é usada para lhe mostrar informações específicas do contexto. A barra de status sempre mostrará onde no room está o cursor do mouse, mas também mostrará informações adicionais baseadas na camada que está sendo editada, a ferramenta que está sendo usada e o estado dessa ferramenta.

Menu Sala

Além das ferramentas de edição, você também verá que um novo menu suspenso foi adicionado ao topo da página IDE.

Estes menus são explicados abaixo:

  • Vista em camadas: Isto reabrirá a janela do Layer Editor, caso você a tenha fechado a qualquer momento.
  • Propriedades do quarto: Isto reabrirá a janela de propriedades Room caso você a tenha fechado a qualquer momento.
  • Propriedades de camadas: Isto reabrirá a Janela de Propriedades da Camada, caso você a tenha fechado a qualquer momento.
  • Ordem de criação da instância: Isto abrirá a Janela de Ordem de Criação de Instância (que também pode ser aberta usando o botão na janela Propriedades da Sala ):
    Esta janela lista todas as instâncias no room na ordem em que serão criadas (de cima para baixo). Se você precisar que uma instância específica seja criada antes de qualquer outra, você pode simplesmente clicar em e arrastá-la para a posição que você precisar.
  • Reinicialização Windows Na área de trabalho atual: Isto irá redefinir o layout da janela do editor room para seus valores padrão para o desktop workspace atualmente focado.

  • Edição de Ladrilhos: Ao trabalhar com uma camada tile, esta opção será destacada no menu suspenso. Ela tem o seguinte sub-menu:
    Ao editar a camada tile você pode selecionar vários tiles clicando, segurando e arrastando o botão esquerdo do mouse , ou você pode selecionar e agrupar o indivíduo tiles usando / + , ou remover o indivíduo tiles da seleção usando + . A partir deste menu você pode então limpar a seleção ou invertê-la, bem como cortar, copiar ou colar a seleção para outra camada (ou mesmo a mesma camada, mas em outra posição). A opção final abrirá novamente a janela acoplada do editor tile set caso você a tenha fechado.
  • Converter imagem para Tile Mapa: A opção Converter Imagem para Tile Mapa é uma ferramenta poderosa que pode ser usada para importar uma única imagem e depois extrair o tiles usado da imagem e recriá-lo como uma camada de mapa tile no editor room, criando os tile set e sprite necessários como parte do processo. Ao selecionar esta ferramenta, você será solicitado a fornecer um arquivo de imagem não comprimido (PNG, GIF ou BMP) que será então carregado. Ao ser carregado, você será apresentado com a ferramenta de importação de imagens:
    Aqui você terá opções sobre como dividir o tile ao criar o tile set, e você poderá definir a largura e altura da célula, bem como quaisquer deslocamentos necessários em torno das bordas. Você também pode definir a largura aproximada (em pixels) para o sprite final que é criado. Por exemplo, se o tile set for composto de 64x64 células tile e você definir uma largura aqui de 200, o sprite final que é criado para uso no tile set terá 192 pixels de largura (ou seja: 3 células tile ). Se você deixá-lo no valor padrão de 0, então o GameMaker tentará fazer um sprite o mais "quadrado" possível com aproximadamente o mesmo número de células tile horizontais e verticais.

    Após definir o modo como a imagem deve ser dividida, clicando na amostra ao lado da opção Remover cor abrirá o seletor de cores e permitirá selecionar uma cor que será removida do final sprite. Esta é geralmente uma cor de fundo que você deseja remover e ajustar esta amostra para qualquer outra cor que não seja 100% transparente (alfa 0) removerá a cor selecionada na importação.

    Finalmente, você tem a opção de nomear a camada do mapa sprite, tile set e tile que será criada para você por esta ferramenta. Ao finalizar a importação, será criado um sprite com todas as imagens dispostas em uma grade, assim como um tile set a partir deste sprite. No room, será criada uma nova camada de mapa tile e a imagem será reproduzida usando o tile set gerado. Note que a ferramenta não duplicará tiles e reconhecerá quando uma célula tiver dados de imagem que coincidam com outra célula (isto inclui tiles rotacionado).
