Código Snippets

Uma ferramenta muito útil que você tem à sua disposição quando edita seu código é o uso de Snippets de Código. Ao utilizar o editor script, você pode pressionar F4 para abrir o pop-up de trechos de código, o que lhe permite selecionar um dos métodos de código comumente utilizados a partir de uma lista:

A partir desta janela você pode selecionar com o mouse o snippet a ser usado, ou pressionar a tecla de atalho associada listada à direita. Isto adicionará o snippet ao seu código para edição.

Você também pode definir seus próprios trechos de código, se desejar. Antes de fazer isso, você precisará criar um arquivo chamado "snippets.txt" em um dos seguintes diretórios:

  • Janelas: %ProgramData%\GameMaker\User\
  • macOS: /Users/Shared/GameMaker/User/

Esta pasta não é modificada durante qualquer atualização do GameMaker, portanto, seu arquivo editado permanecerá intacto (mas qualquer edição do arquivo de trechos básicos no diretório de instalação será revertida).

Uma vez criado o arquivo no diretório do usuário, você pode editá-lo com qualquer editor de texto seguindo estas regras:

  • Cada trecho tem que estar em uma linha separada (não deve haver linhas em branco)
  • Cada trecho deve começar com a tecla de atalho a ser usada seguida por "-" e depois o nome do trecho (que é o que é mostrado no menu) seguido por dois pontos ":", por exemplo:

I - Instância Criar:

Depois dos dois pontos você adiciona o trecho de código

I - Instance Create:instance_create_layer(x, y, |layer|, object);

O código que você adiciona também deve seguir um formato específico onde:

  • Todos os códigos estão em uma única linha
  • Você usa "$" para uma nova linha.
    • OBSERVAÇÃO Ao criar várias novas linhas, é preciso haver um espaço entre cada sinal "$". Por exemplo, você escreveria $ $ $ $ para adicionar três novas linhas.
  • Você coloca o texto que precisa ser destacado entre os símbolos "|". O Editor de Código selecionará automaticamente este texto para edição quando o snippet for adicionado.

Veja os exemplos que já estão no arquivo de snippets base para ver como ele é montado seguindo as regras acima. Você pode encontrar o arquivo base no diretório de instalação do GameMaker:

  • Janelas: GameMaker TextoEditor\\snippets.txt
  • macOS: /Applications/GameMaker.app/Contents/MonoBundle/TextEditor/snippets.txt
