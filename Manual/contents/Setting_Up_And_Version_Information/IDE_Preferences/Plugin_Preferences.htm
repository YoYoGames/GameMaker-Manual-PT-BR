Preferências de Plugin

As Preferências de Plugins são usadas para controlar quaisquer plugins extras que tenham sido adicionados ao GameMaker IDE. Por padrão, este tem uma seção dedicada ao plugin Source Control para integrar projetos GameMaker com o Git.

Controle da fonte (Git)

Esta seção trata das preferências necessárias para que o Git SCM plugin funcione com o GameMaker.

  • Git caminho executável: Esta é a localização do executável Git que GameMaker usa para todos os comandos Source Control. Como mostrado na imagem acima, este será localizado na pasta cmd de sua instalação do Git.

Abaixo disso, você pode configurar a Ferramenta de fusão e a Ferramenta Diff para Source Control com estas configurações(opcional):

  • Merge (Localização da ferramenta): Aqui você especifica o arquivo completo path para a ferramenta merge Git.
  • Fundir (Opções de ferramentas): Neste campo você pode adicionar qualquer comando extra a ser executado sempre que utilizar a Ferramenta de fusão.
  • Diff (Localização da ferramenta): Aqui você especifica o arquivo completo path para a ferramenta de difusão Git.
  • Diff (Opções de ferramentas): Neste campo você pode adicionar qualquer comando extra a ser executado sempre que você usar a ferramenta de difusão.

Finally there are two more options that affect how Git is set up in new projects: 

  • Add skeleton .git defaults to new projects: When this setting is enabled, GameMaker will add .gitignore and .gitattributes Files to new projects that you create. It is enabled by default.
  • Add skeleton .git defaults to imported projects: When this setting is enabled, GameMaker will add .gitignore and .gitattributes Files to projects that you import from a .yyz file. It is enabled by default.
