Preferências gerais

As Preferências Gerais são usadas para definir o comportamento geral da IDE GameMaker. As mais gerais estão listadas na página principal enquanto muitas outras estão divididas em várias subcategorias, explicadas mais abaixo nesta página.

  • Linguagem IDE: Esta opção permite que você selecione o idioma para o IDE a partir da lista de opções fornecidas. O idioma padrão é o inglês.
  • Fonte IDE: Esta opção permite alterar o font utilizado em todo o IDE. O padrão é Open Sans.
  • Framerate máximo do Editor GameMaker: Esta opção permite que você defina o frame máximo que o GameMaker IDE irá atualizar, permitindo que você defina o frame para corresponder à velocidade de atualização de seu monitor. O valor padrão é 60.
  • Pele IDE: Aqui você pode selecionar entre as diferentes peles que o IDE pode utilizar. Por padrão o GameMaker vem com duas peles, uma clara e uma escura, e a configuração inicial utiliza a pele escura.
  • Dispositivo de áudio padrão: Aqui você pode escolher o dispositivo de áudio padrão para o GameMaker IDE a ser usado. Você pode atualizar a lista clicando no Botão Atualizar .
  • Resposta Automática à Confirmação de Saída: Esta opção permite que você defina se uma mensagem de confirmação deve ser mostrada ao sair do GameMaker. Por padrão, isto é definido para "mostrar mensagem", mas pode ser definido para ignorar a mensagem como se você tivesse clicado OK.
  • Resposta automática ao sistema Font Mudança de diálogo: Esta opção permite definir se uma mensagem de confirmação deve ser mostrada quando um sistema font foi adicionado, removido ou editado. Por padrão, isto é definido para "mostrar mensagem", mas pode ser definido para pular a mensagem como se você tivesse clicado OK.
  • Habilitar notificações: Ao verificar isso, o IDE mostrará notificações sobre o novo runtimes, etc... Isto está ativado por padrão.
  • Notificação de tempo limite (ms): Define o período de tempo (em milissegundos) para o qual uma notificação é mostrada. O tempo padrão é 6000.
  • Logout na saída: Se esta opção for marcada, então o IDE sairá automaticamente da sessão do usuário quando o GameMaker for fechado. Isto é direcionado principalmente aos usuários de Educação, pois - quando verificado - isto significa que sempre que o programa fecha, o usuário sai do programa e assim libera o assento da sessão para que outros usuários a utilizem. Isto está desativado por padrão.
  • Mostrar tela de boas-vindas na inicialização: Por padrão, isto está ligado e simplesmente diz ao GameMaker para mostrar uma breve tela de boas-vindas quando você iniciar o programa. Desmarcando isto evitará que a tela de início apareça.
  • Permitir o fechamento de abas vazias através de atalho: Você pode fechar qualquer aba aberta usando o atalho de teclado / + W, mas se você desativar esta opção, o atalho não funcionará mais. Isto é ativado por padrão.
  • Abrir novos recursos para edição no arraste de arquivos para o IDE: Alguns recursos (como sprites ou sons) podem ser arrastados para o principal Workspace no GameMaker IDE e serão detectados e adicionados como um novo recurso no Asset Browser. Por padrão, o editor de recursos correspondente também será aberto para edição quando isto acontecer, mas a desmarcação desta opção impedirá que os editores sejam abertos e você será obrigado a abri-los manualmente.
  • Abrir novos recursos para edição na criação: Se isto for verificado, toda vez que você criar um novo recurso no navegador Asset, o editor de recursos correspondente será aberto automaticamente. Isto é verificado por padrão e desmarcado significa que você terá que abrir manualmente todos os recursos recém-criados.
  • Recarregar automaticamente os arquivos alterados: Se isto for verificado, o IDE verificará automaticamente os arquivos para ver se algum deles foi alterado fora do GameMaker e os recarregará se tiverem sido alterados.
  • Desativar IDE animações de transição: O GameMaker IDE tem uma série de transições animadas para coisas como abrir janelas etc... no entanto, elas não são do gosto de todos e você pode experimentar um aumento de desempenho se elas forem desativadas em máquinas mais antigas ou com baixo consumo de energia se elas forem desativadas. Você pode desativá-las aqui, e elas estão ligadas por padrão.
  • Opt-in para pesquisas de feedback: Quando esta opção é marcada, você está dizendo ao GameMaker que deseja participar de pesquisas de feedback, o que significa que ocasionalmente enquanto trabalha uma mensagem será mostrada sobre a área em que você está trabalhando e solicitará uma avaliação e feedback. Esta opção está ativada por padrão.
  • Habilitar o DPI Override: Esta opção permite que você sobreponha a configuração padrão de DPI com a sua própria. Você deve primeiro ativar a opção e então definir o valor Raw DPi Override (o valor padrão é 96) ou definir a porcentagem do DPi nativo (valor padrão 100%), e uma vez alterado você será solicitado a reiniciar o IDE, o que pode exigir que você salve qualquer trabalho que esteja sendo feito no projeto atualmente aberto. Observe que o DPi tem uma porcentagem mínima de 50% e máxima de 500%, e se você passar o mouse sobre o ícone de informação, você poderá ver a configuração máxima recomendada para a tela que está sendo usada:

    Observe que se você for acima da configuração DPi recomendada para o monitor, será exibida uma notificação que lhe permite reiniciar o DPi.

As opções acima são as principais opções disponíveis nas preferências gerais , mas há uma série de opções extras disponíveis nas subcategorias listadas abaixo:

  • Antecedentes
  • Compilação
  • Confirmar Diálogos
  • Depuração
  • Vigilante de arquivos
  • Ir para
  • Ajuda
  • Entrada
  • Janelas de saída
  • Caminhos
  • Economia de energia
  • Janelas recentes
  • Recursos
  • Espaços de trabalho
