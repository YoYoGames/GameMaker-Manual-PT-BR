Obtenha Tile Dados na Célula

Com esta ação, você pode obter os dados tile para uma célula tile em uma camada de mapa de azulejos. Os dados tile são simplesmente um valor que reflete o índice do tile junto com os valores de inverter, espelhar e girar, assim como quaisquer máscaras personalizadas tile que você tenha usado. Com esta ação você primeiro fornece a camada a ser alvo usando o nome da camada (a string) como definido no editor room, e depois a coluna e linha da camada do mapa tile para obter os dados da telha para. A "célula" é a área na grade do mapa tile que contém o tile do qual você quer obter os dados, portanto se seu tiles for 16x16, por exemplo, e o room for 1024x768 o mapa de azulejos terá 64 colunas e 48 fileiras.

NOTA: A camada selecionada deve ter sido definida como um Tile Map Layer no editor room, caso contrário, você poderá receber erros.

O valor retornado será então armazenado na variável de destino que pode ter sido criada anteriormente ou pode ser uma nova variável temporária (se você marcar a caixa de seleção "Temp"). Para mais informações sobre os dados tile, consulte a seção GML sobre Funções do Mapa de Telha.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada para obter o tile de
Column A coluna (da esquerda para a direita) da célula
