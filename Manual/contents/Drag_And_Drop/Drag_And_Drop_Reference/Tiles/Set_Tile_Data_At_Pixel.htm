Conjunto Tile Dados em Pixel

Com esta ação você pode configurar os dados tile para uma célula tile encontrada em uma posição específica em uma camada de mapa de azulejos. Os dados de tile são simplesmente um valor que reflete o índice do tile junto com o inverter, espelhar e girar valores, assim como quaisquer máscaras personalizadas tile que você tenha usado. Com esta ação você primeiro fornece a camada a ser alvo usando o nome da camada (a string) como definido no editor room, e depois os eixos x e y na sala para definir os dados de tile. Note que os dados de tile serão definidos para a "célula" do mapa tile na qual a posição dada aterrissa. Por exemplo, se seu tile set tiles for 16x16, então a camada do mapa tile conterá tiles em células de 16x16 pixels, portanto se você usar esta ação para direcionar a posição room (8, 8), você estará mudando o tile na célula (0, 0) do mapa tile.

NOTA: A camada selecionada deve ter sido definida como um Tile Map Layer no editor room, caso contrário, você poderá receber erros.

O argumento final para a ação é os próprios dados tile. Você normalmente recuperaria os dados tile para a célula do mapa tile usando a ação Obter Tile Data At Pixel e depois manipulá-los usando o conjunto de ação Tile Data Transform antes de configurá-lo novamente com esta ação. Para mais informações sobre os dados tile, consulte a seção GML sobre as funções do mapa de azulejos.

Sintaxe de ação:

Argumentos:

Argumento Descrição
Layer A camada com o mapa tile ao alvo
X A posição ao longo do eixo horizontal (X) para definir os dados da telha para
Y A posição ao longo do eixo vertical (Y) para definir os dados da telha
Data Os dados de azulejos a serem usados para colocar a célula na posição

Exemplo:

O código de bloqueio de ação acima testa para um clique do mouse e quando um é recebido recupera os dados do azulejo para a célula que coincide com a posição do mouse. Estes dados são então modificados e retornados para a camada do mapa tile, girando o tile naquela célula.
