Cor da Instância

Este bloco de ação estabelece a cor image_blend para "misturar" com a instância sprite. O valor padrão é o código hexadecimal $FFFFFFFF (que representa a cor branca usando o formato ARGB ) e isto irá desenhar o sprite usando nenhuma mistura e alfa completa. Qualquer outro valor (incluindo constantes de cor internas como c_red, ou c_aqua) irá misturar a cor especificada com o original sprite bem como definir o image_alpha componente, por exemplo, mas somente quando a caixa de seleção Usar Alfa estiver marcada. Isto sobregravará qualquer valor alfa definido anteriormente usando a ação Definir instância alfa.

Favor observar que para que as mudanças nesta ação sejam visíveis, a instância não deve ter nenhum evento de sorteio (e, portanto, o GameMaker irá, por padrão, sortear o sprite) ou ser sorteada usando a ação do Draw Self. É a seguinte Também é importante observar que você deve tentar limitar a mistura na plataforma HTML5 (a menos que utilize WebGL), pois cada mistura sprite tem que ser colocada em cache separadamente e, portanto, ter muitas misturas sprites pode afetar negativamente o desempenho.
