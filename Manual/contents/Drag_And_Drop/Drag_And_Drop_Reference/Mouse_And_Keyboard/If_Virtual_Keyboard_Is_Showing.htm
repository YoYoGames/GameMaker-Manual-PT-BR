Se o Virtual Keyboard estiver aparecendo

Esta ação pode ser usada para verificar se o teclado virtual do sistema operacional está sendo mostrado ou não. A ação será avaliada para true quando o teclado estiver aberto ou abrindo, ou false se o teclado virtual estiver fechado ou fechando. Se você ativar o modificador not, esta ação será revertida e você poderá armazenar o valor de retorno true / false em uma variável alvo, que pode ser sinalizada como temporária (local).
