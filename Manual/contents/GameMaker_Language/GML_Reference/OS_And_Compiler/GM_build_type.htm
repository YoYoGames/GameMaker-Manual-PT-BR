Tipo GM_build_type

Esta constante string contém o tipo de construção: "exe" para um executável criado ou "run" para uma execução de teste iniciada a partir do IDE.

Sintaxe:

GM_build_type;

Devoluções:

String

Exemplo:

draw_text(32, 32, GM_build_type);

O código acima desenhará o tipo de construção do executável atualmente em execução.
