matrix_stack_push

Esta função é usada para empurrar uma nova matriz para o topo da pilha matriz atual. Primeiro multiplicará a matriz dada com a matriz atualmente no topo da pilha, e depois empurrará a matriz resultante para a pilha.

Esta função é útil para aplicar múltiplas transformações de matriz ao seu visual sem ter que multiplicar manualmente várias matrizes juntas.

Por exemplo, você pode ter uma matriz base empurrada para a pilha que desenha gráficos em uma determinada área retangular (digamos, uma janela). Depois de desenhar alguns gráficos usando essa matriz, você pode empurrar outra matriz na pilha para desenhar gráficos dentro de uma sub-área (como um botão na janela). Depois de desenhar algo nessa sub-área, você pode chamar matrix_stack_pop() para remover sua matriz da pilha e continuar desenhando visuais na área da janela principal.

Sintaxe:

matrix_stack_push(matrix);

ArgumentoTipo Descrição
matrix The matrix to push to the stack

Devoluções:

Exemplo:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

O código acima construirá duas novas matrizes e depois as empurrará para a pilha de matrizes. A matriz do mundo é então colocada no topo da pilha, um sprite é desenhado e as transformações são então retiradas da pilha.
