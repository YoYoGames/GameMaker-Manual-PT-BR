buffer_save

Com esta função você pode salvar o conteúdo de um buffer em um arquivo, pronto para ser lido de volta à memória usando o buffer_load() função.

NOTA: Em HTML5 o conteúdo do buffer será salvo como base64 codificado strings ao usar esta função.

Sintaxe:

buffer_save(buffer, filename);

ArgumentoTipo Descrição
buffer The index of the buffer to save.
filename The name of the file to save as.

Devoluções:

Exemplo:

buffer_save(buff, "Player_Save.sav");

Salva o conteúdo atual do buffer com o id armazenado na variável "buff" em um arquivo.
