buffer_compress

Com esta função você pode comprimir parte (ou a totalidade) de uma buffer usando a compressão zlib. Você fornece o ID do buffer para comprimir (como retornado por buffer_create()), o offset dentro do buffer para usar em bytes, e o tamanho dos dados buffer para comprimir (também em bytes). A função retornará um novo valor de ID buffer para o buffer comprimido, ou um valor inferior a 0 se ele falhar por qualquer razão. Esta função não alterará o original buffer.

Sintaxe:

buffer_compress(buffer, offset, size);

ArgumentoTipo Descrição
buffer The index of the buffer to compress.
offset The offset within the buffer to compress (in bytes).
size The size of the buffer area to compress (in bytes).

Devoluções:

Buffer ID

Exemplo:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

O código acima criará um buffer e depois o preencherá com os dados de um string. Este buffer é então comprimido e salvo, e tanto a fonte quanto o buffers comprimido são deletados.
