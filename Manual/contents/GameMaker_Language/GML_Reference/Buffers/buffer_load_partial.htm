buffer_load_partial

Esta função irá carregar alguns dos dados buffer que foram salvos anteriormente usando o buffer_save() funciona em um buffer existente. Você dá o id do buffer criado anteriormente para carregar, depois o arquivo buffer salvo para carregar, e depois o offset do início do buffer (em bytes) a partir do qual você deseja carregar os dados. Os seguintes argumentos são para definir o comprimento dos dados buffer (em bytes) a partir do ponto de offset inicial que você deseja carregar e o ponto de offset para carregar os dados no buffer (novamente, em bytes).

Por favor, leia a página buffer_load() para notas específicas da plataforma.

Sintaxe:

buffer_load_partial(buffer, filename, offset, src_len, dest_offset);

ArgumentoTipo Descrição
buffer The index of the buffer to load into (destination).
filename The name of the file to load from (source).
offset The offset within the destination buffer to load to (in bytes).
src_len The length of the part of the source buffer to load (in bytes).
dest_offset The offset where to start putting the partial data in the destination buffer (in bytes).

Devoluções:

N/A

Exemplo:

buff = buffer_create(256, buffer_grow, 1);
var _file = "save.dat";
var _so = 6;
var _sl = 5;
var _do= 0;
buffer_load_partial(buff, _file, _so, _sl, _do);

O código acima criará um novo "crescer" buffer e então carregará em uma parte dos dados salvos no arquivo "save.dat" para ele.
