Com esta função você pode verificar uma posição para uma colisão com outra instância ou todas as instâncias de uma object usando a máscara de colisão da instância que executa o código para a verificação. Quando você usa esta função, você está efetivamente pedindo ao GameMaker para mover a instância para a nova posição, verificar se há uma colisão, voltar atrás e dizer se uma colisão foi encontrada ou não.

Isto funcionará para colisões precisas, mas somente se tanto a instância e o object que está sendo verificado tiverem máscaras de colisão precisas selecionadas, caso contrário somente serão aplicadas colisões de caixas delimitadoras.

Esta função devolverá a instância única id do object a ser colidido, mas se isso não for necessário é ligeiramente mais rápido usar a função place_meeting(). Esta função também aceita as palavras-chave especiais all e other e devolverá a palavra-chave noone caso não ocorra nenhuma colisão, ou o valor único de identificação da instância encontrada caso ocorra uma colisão.

Sintaxe:

instance_place(x, y, obj);

Argumento Tipo Descrição
x Real The x position to check for instances.
y Real The y position to check for instances.
obj Objeto Ativo ou ID da instância The object to check for instances of.

Devoluções:

Instance ID or noone

Exemplo:

var _inst = instance_place(x, y, obj_Enemy);
if (_inst != noone)
{
    hp -= _inst.dmg;
    instance_destroy(_inst);
}

O código acima verificará se há uma colisão com instâncias de "obj_Enemy" e, se houver, reduzirá a variável "hp" pela quantidade armazenada na variável "dmg" da instância de colisão e, em seguida, destruirá a instância de colisão.
