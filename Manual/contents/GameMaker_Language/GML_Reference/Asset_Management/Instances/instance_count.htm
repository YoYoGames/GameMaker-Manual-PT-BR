exemplo_conta

Com esta variável lida apenas você pode obter uma contagem de todas as instâncias ativas que estão no room. Isto incluirá a instância que executa o código, mas não inclui as instâncias que foram desativadas usando as funções da instância desativada. Note que esta variável só lhe dará o número de instâncias no início da etapa, portanto qualquer alteração nas instâncias no room feita após o início da etapa não será levada em consideração.

Sintaxe:

instance_count;

Devoluções:

Exemplo:

if (instance_count < 100)
{
    var dif = 100 - instance_count;
    while (--dif > 0)
    {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
    }
}

O código acima criará múltiplas instâncias do "obj_Star" object até que a contagem total de instâncias atinja 100.
