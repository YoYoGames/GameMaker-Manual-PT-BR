instance_furthest

Esta função verificará todas as instâncias do dado object para ver qual está mais distante do ponto x/y dado. Todas as verificações serão da posição x/y dada até a origem (a posição x/y) das instâncias do object especificado. Caso não existam instâncias do object, a função retornará a palavra-chave noonemas, se houver casos, devolverá o id da instância encontrada. Observe que se a instância executando o código tiver o mesmo índice object que o object sendo verificado, então ele será incluído na verificação (isto inclui verificações para os pais objects se a instância chamadora for também um filho dos pais).

Sintaxe:

instance_furthest(x, y, obj);

ArgumentoTipo Descrição
x The x position to check for instances far from.
y The y position to check for instances far from.
obj The object to check for instances of.

Devoluções:

Instance ID or noone

Exemplo:

var inst;
inst = instance_furthest(x, y, object_index);
if (inst != id)
{
    draw_line(x, y, inst.x, inst.y);
}

O código acima encontrará a instância mais distante que compartilha o mesmo índice object que a instância que executa o código e armazena a identificação em uma variável. Esta variável é então verificada para ver se é a mesma que o id da instância chamadora e, se não for, uma linha é traçada entre as duas instâncias.
