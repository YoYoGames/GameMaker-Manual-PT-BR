id

Esta variável somente leitura contém o número de identificação único para a instância. Cada instância criada - seja através de código ou adicionando-os a um room no Editor Room - recebe um número que é usado internamente para identificar esta instância e a variável id é o que você pode usar para referenciá-la. O id também é retornado (e pode ser armazenado em uma variável) quando uma instância é criada usando instance_create_layer() ou instance_create_depth(), assim como outras funções da instância.

Esta variável é agora uma referência a uma instância, em vez de ser simplesmente um número real. Você pode obter o número de identificação da instância ligando para real() ou int64() sobre esta variável.

NOTA O valor desta variável não é o mesmo que o identificador que a instância é dada no editor room e também é diferente do instance_id que contém todas as ids de todas as instâncias atualmente ativas.

Sintaxe:

id;

Devoluções:

Instance ID

Exemplo:

for (var i = 0; i < instance_count; i++;)
{
    if (instance_id[i] != id)
    {
        instance_id[i].scr += 5;
    }
}

O código acima acrescenta 5 à variável "scr" para cada instância ativa no room, exceto a que executa o código. Isto é feito através de looping em TODAS as instâncias ativas (usando o array "instance_id" para retornar cada ID de instâncias ativas) e comparando-as com a variável "id" incorporada, que é o ID da instância original executando o código.
