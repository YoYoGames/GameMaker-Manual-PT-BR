camada

Esta variável incorporada é criada para cada instância em um room e contém o valor de identificação da camada à qual a instância está atribuída. Este valor pode então ser usado em outras funções como layer_get_depth() ou pode ser alterado para mover a instância para outra camada, mas observe que se a camada que está sendo atribuída não existir no atual room, então você receberá um erro que forçará seu jogo a fechar. Ao atribuir uma camada, você deve fornecer a identificação única da camada como retornada pela função layer_get_id() (ao usar camadas nomeadas room ), ou como retornado pela função layer_create() (quando você cria suas próprias camadas em tempo de execução).

IMPORTANTE Se você criou a instância usando o instance_create_depth() ou ter alterado manualmente a função depth variável, a camada atribuída à instância se torna uma camada "gerenciada", que é aquela que o GameMaker controla e gerencia automaticamente. Nesses casos, a variável layer retornará -1.

Sintaxe:

layer;

Devoluções:

Layer ID

Exemplo:

layer = layer_create(-1000);

O código acima criará uma nova camada com uma profundidade de -1000 e então definirá a variável de instância layer para o ID da camada retornada, movendo a instância da camada em que está atualmente para a nova camada que está sendo criada.
