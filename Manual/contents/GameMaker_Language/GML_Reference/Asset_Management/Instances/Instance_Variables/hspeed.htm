hspeed

hspeed é uma das propriedades "embutidas" que todas as instâncias possuem e define a velocidade de movimento horizontal (ao longo do eixo x) da instância em pixels por etapa. Portanto, um hspeed de 3 significa 3 pixels de movimento para a direita (+x) cada passo, e um hspeed de -3 significaria 3 pixels de movimento para a esquerda (-x) cada passo.

Observe que se você definir as variáveis speed e/ou direction, então o valor hspeed será atualizado automaticamente para refletem essas mudanças e, da mesma forma, a mudança do valor hspeed também afetará os valores speed e direction. Observe também que gravity, gravity_direction e friction podem todos modificar o valor desta variável quando eles são usados em seus jogos.

Sintaxe:

hspeed;

Devoluções:

(single precision floating point value)

Exemplo:

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

O código acima mudará a velocidade horizontal, dependendo de quais teclas são pressionadas.
