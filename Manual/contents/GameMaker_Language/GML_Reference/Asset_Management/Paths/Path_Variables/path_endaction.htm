path_endaction

Esta variável pode ser usada para obter ou alterar a reação de uma instância quando ela chega ao final da corrente path. Normalmente, você definiria isso ao iniciar o path usando path_start() mas você pode querer mudar este comportamento dependendo de qualquer número de eventos em seu jogo. Os valores disponíveis são expressos utilizando as seguintes constantes:

Constante de Ação de Fim de Caminho
Constante Descrição
path_action_stop Parar o caminho
path_action_restart Continue da posição inicial, pulando para o início se o path não estiver fechado
path_action_continue Comece o path novamente a partir da posição atual
path_action_reverse Reverta a velocidade do path (faça o path ao contrário)

Sintaxe:

path_endaction;

Devoluções:

Path End Action Constant

Exemplo:

if (path_endaction == path_action_stop)
{
    path_endaction = path_action_reverse;
}

O código acima verificará a ação final path e se ela estiver definida para parar, então a ação final será alterada para reverter.
