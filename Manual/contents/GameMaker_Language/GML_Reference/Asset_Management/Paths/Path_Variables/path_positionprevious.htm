caminho_posição anterior

Esta variável pode ser usada para obter ou definir a posição de uma instância ao longo de sua atual path no passo anterior, e é um valor normalizado entre 0 e 1 ou seja: 0 é a posição inicial do path e 1 seria a posição final. É semelhante ao xprevious e yprevious variáveis no seu funcionamento, somente é específico para paths. Pode ser útil para coisas como parar temporariamente um seguidor path se algo estiver no caminho (veja o código de exemplo abaixo).

Sintaxe:

path_positionprevious;

Devoluções:

Exemplo:

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if (collision_circle(_x, _y, 16, obj_player, false, true))
{
    path_position = path_positionprevious;
}

O código acima verifica uma área na frente da instância para uma colisão com o "obj_Player" object e, se houver, coloca a instância de volta à posição anterior que ocupava no atual path, conforme atribuído pela função path_start().
