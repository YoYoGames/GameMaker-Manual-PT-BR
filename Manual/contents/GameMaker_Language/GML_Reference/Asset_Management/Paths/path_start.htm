path_start

Um path é criado a partir de uma série de pontos definidores que são ligados entre si e depois utilizados para planejar os movimentos de uma instância. Eles podem ser criados com código, ou no Editor Path e são atribuídos a uma instância para serem usados no jogo. Você então usaria esta função para dizer a sua instância qual path seguir, que velocidade seguir o path (medido em pixels por passo), como se comportar quando chegar ao final do path, e se seguir a posição absoluta ou relativa path. Esta última parte significa que ou começa e segue o path exatamente como você projetou e o colocou no Path Editor (absoluto), ou começa e segue o path a partir da posição em que a instância foi criada (relativo).

Como você pode ver na imagem acima, as duas instâncias estão seguindo a mesma path iniciada com path_start(), mas uma das instâncias está seguindo o absoluto path enquanto a outra está seguindo o relativo path. Deve-se notar que quando a instância relativa é definida como falsa (absoluta), a instância "pulará" para o início do path não importa onde no room seja colocada.

As constantes usadas para definir o comportamento da instância quando ela chega ao final do path são dadas nesta página.

Sintaxe:

path_start(path, speed, endaction, absolute);

Argumento Tipo Descrição
path Ativo do caminho The path index to start.
speed Real The speed of which to follow the path in pixels per step, negative meaning going backwards.
endaction Constante de Ação de Fim de Caminho What to do when the end of the path is reached (see the constants listed above).
absolute Booleano Whether the calling instance should follow the absolute path as it is in the editor (true) or a relative path to its current position (false).

Devoluções:

N/A

Exemplo:

path_start(path, 4, path_action_reverse, 0);

Isto fará com que a instância de chamada siga o path indexado na variável "caminho" a uma velocidade de 4 pixels por passo. Ela seguirá o path em relação a sua própria posição, e quando chegar ao final, inverterá a direção e voltará ao caminho que veio.
