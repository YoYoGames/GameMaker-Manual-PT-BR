timeline_index

Esta variável contém o índice do timeline atualmente associado à instância. Você pode definir isto para uma determinada linha de tempo para usar aquela, ou defini-la como -1 para parar de usar uma linha de tempo para a instância (se nenhuma linha de tempo for definida para a instância, -1 também é retornada). Note que isto não inicia a linha de tempo - para isso use a variável timeline_running.

Sintaxe:

timeline_index;

Devoluções:

Timeline Asset

Exemplo:

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

O código acima atribuirá à instância executando o código uma linha de tempo indexada na variável "global.tl", se essa linha de tempo existir.
