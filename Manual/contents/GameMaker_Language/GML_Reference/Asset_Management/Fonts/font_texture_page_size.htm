font_texture_page_size

Esta variável embutida pode ser usada para obter ou definir o tamanho de página texture ao usar a função font_add(). Ao adicionar um font usando essa função, o GameMaker criará uma página texture cache com os glifos necessários até o tamanho definido por esta variável. O processo é o seguinte:

  • Cada personagem que você usa é colocado em cache do site font em uma página texture
  • Quando a página texture estiver cheia (porque você usa caracteres grandes ou muitos caracteres diferentes) o GameMaker removerá os caracteres do cache e os substituirá por novos caracteres que você precisa
  • O GameMaker tenta não fazer o texture muito grande porque nem todos os dispositivos podem lidar com o grande textures, e tenta não fazer o texture muito pequeno porque remover e readicionar caracteres no texture leva tempo de CPU e faz com que seu jogo funcione lentamente

Se você acha que precisa de uma página maior ou menor texture do que a 1024px padrão que o GameMaker usa, você pode definir o valor usando esta variável incorporada. Se você não estiver usando a função font_add(), então a configuração não terá efeito no desempenho de seus jogos, pois fonts adicionado no IDE estão em cache nas páginas regulares texture (que você pode controlar a partir do Editor de Fonte e das Opções de Jogo para a plataforma alvo).

Sintaxe:

font_texture_page_size;

Devoluções:

Real (maximum width/height in pixels)

Exemplo:

font_texture_page_size = 512;
newfont = font_add("Arial", 24, true, true, 32, 128);

O código acima define a página font cache texture para um tamanho máximo de 512px e depois adiciona um novo font ao jogo.
