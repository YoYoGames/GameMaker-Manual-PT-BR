font_delete

Com esta função, você pode excluir um font asset do jogo. Esta é uma remoção permanente, e mudar rooms, ou reiniciar o jogo não trará de volta o font removido. Para isso, o jogador precisaria sair do jogo e reinicie dessa forma, portanto, tenha cuidado ao usar esta função. Em geral, ela só é necessária para liberar a memória que foi utilizada por um font adicionado ao jogo através das funções font_add() ou font_add_sprite().

Sintaxe:

font_delete(ind);

ArgumentoTipo Descrição
ind The index of the font to delete.

Devoluções:

Exemplo:

font_delete(global.Font);

O código acima excluirá o font indexado na variável global "Font".
