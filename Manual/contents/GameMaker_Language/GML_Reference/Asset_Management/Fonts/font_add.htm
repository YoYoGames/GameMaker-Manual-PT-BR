font_add

Esta função pode ser usada para adicionar um font ao seu jogo a partir de um arquivo font presente nos Arquivos Incluídos. Você pode definir o tamanho do font (em pontos), assim como se o font deve ser em negrito ou itálico, e também pode definir o intervalo de caracteres a serem incluídos. A função retorna um valor de índice que deve ser armazenado em uma variável, pois isso será necessário em todos os outros códigos que se referem a este font, ou retornará -1 se a função falhar por qualquer motivo.

Note que o arquivo font deve ser um arquivo *.ttf ou *.otf, que também pode ser útil para adicionar arquivos não-padrão fonts como o asiático ou árabe fonts.

ADVERTÊNCIA! Se você incluir um font nos Arquivos Incluídos, ele deve ser licenciado para distribuição com o jogo.

Para o módulo de destino HTML5, esta função só pode ser usada para adicionar Fontes Web, e só funcionará se WebGL não estiver habilitada - se WebGL estiver habilitada, então você deve usar um recurso font geral (ou seja: usando draw_set_font() juntamente com um font do Browser Asset ) ou bitmap fonts (ver font_add_sprite()).

IMPORTANTE! Ao visar HTML5, o texto em itálico e negrito só é suportado se o próprio font asset ou web font tiver estes caracteres incluídos. Eles não serão sintetizados a partir da base font.

Ao carregar um font de um arquivo o tamanho do font está em pixels e o primeiro e último valores são ignorados, o que significa que todos os glifos no font serão adicionados. Fonts adicionados desta forma são atribuídos a sua própria página texture, portanto, deve-se ter cuidado ao usar esta função, pois ela aumentará o número de swaps texture ao desenhar. Vale notar também que fonts pode parecer ligeiramente maior quando desenhado, uma vez que os glifos podem ter partes que são desenhadas fora da caixa delimitadora definida para o font. Você também deve estar ciente de que há uma limitação de cerca de 200 glifos que podem ser renderizados em um único quadro para um determinado font.

Por padrão, adicionado fonts terá o anti-aliasing ativado. Entretanto, você pode verificar o estado do AA a qualquer momento usando a função font_add_get_enable_aa() e você pode habilitar/desabilitar o AA usando a função font_add_enable_aa().

NOTA Quando você carrega um font no GameMaker, você deve se lembrar de removê-lo novamente (com font_delete()) quando não for mais necessário, caso contrário há o risco de um vazamento de memória que irá retardar e, eventualmente, quebrar seu jogo.

Sintaxe:

font_add(name, size, bold, italic, first, last);

Argumento Tipo Descrição
name Cordão The path of the font file in the Included Files (*.ttf ou *.otf).
size Real The size of the font - points for Web Fonts, pixels for file fonts.
bold Booleano Whether the font is bold (true) ou não (false).
italic Booleano Whether the font is italic (true) ou não (false).
first Real The first character to include (if you're unsure, go for 32).
last Real The last character to include (if you're unsure, go for 128).

Devoluções:

Font Asset

Exemplo:

new_font = font_add("gamefont.ttf", 24, true, true, 32, 128);

Isto irá carregar o arquivo "gamefont.ttf" presente nos Arquivos Incluídos do jogo com um tamanho de 24pt. O font será em negrito e itálico, e a faixa de fontes incluirá letras maiúsculas e minúsculas, números e todas as pontuações comuns. Depois que o font for carregado, sua identificação será armazenada na variável new_font.
