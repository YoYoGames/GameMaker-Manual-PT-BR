sequence_instance

Esta é uma variável incorporada que faz parte das variáveis de instância criadas para cada object instância em seu jogo. Se a instância estiver sendo controlada por uma sequence, esta variável manterá a estrutura da instância em seqüência para a Sequence que controla a instância, caso contrário será undefined. Esta é uma variável somente de leitura e não pode ser alterada.

Note que esta variável se tornará undefined após o término do controle Sequence, mesmo que seu elemento Sequence ainda exista, e receberá o struct novamente se esse elemento Sequence for reproduzido.

Sintaxe:

sequence_instance

Devoluções:

Sequence Instance Struct

Exemplo:

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

O código acima verifica o in_sequence e se for true (o que significa que a instância está sendo controlada por um sequence), então mudará a escala de velocidade daquela variável Sequence para 2.
