asset_get_type

Com esta função você pode obter o tipo de asset sendo referenciado a partir de seu nome (a string). Uma das constantes listadas abaixo será devolvida.

Sintaxe:

asset_get_type(name);

Argumento Tipo Descrição
name Cordão The name of the game asset to get the type of (a string).

Devoluções:

Asset Type Constant

Constante Descrição
asset_object O nome dado refere-se a um objeto.
asset_sprite O nome dado refere-se a um duende.
asset_sound O nome dado refere-se a um som.
asset_room O nome dado refere-se a um room.
asset_tiles O nome dado refere-se a um conjunto de azulejos.
asset_path O nome dado refere-se a um caminho.
asset_script O nome dado refere-se a um script.
asset_font O nome dado refere-se a uma fonte.
asset_timeline O nome dado refere-se a uma linha de tempo.
asset_shader O nome dado refere-se a um sombreador.
asset_animationcurve O nome dado refere-se a uma Curva de Animação.
asset_sequence O nome dado refere-se a uma sequência.
asset_unknown O nome dado refere-se a um asset que ou não existe, ou não é um dos acima listados.

Exemplo:

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
    show_debug_message("Path doesn't exist!!!");
}
else
{
    path_index = asset_get_index("pth_Path_" + string(global.Game));
}

O código acima verifica um nome asset criado dinamicamente para ver se o asset é do tipo correto. Se não for, será mostrada uma mensagem de depuração, caso contrário o nome asset é usado para atribuir o asset à instância.
