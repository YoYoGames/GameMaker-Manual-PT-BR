tilemap_get_tileset

Usando esta função, você pode recuperar o valor do índice do Tile Set asset atribuído a um dado elemento do mapa tile em uma camada. Você dá o ID do elemento do mapa Tile (que você obtém quando cria um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e a função retornará o índice Tile Set asset.

Sintaxe:

tilemap_get_tileset(tilemap_element_id);

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the Tile Map element to get the tile set from

Devoluções:

Tile Set Asset

Exemplo:

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_tileset(map_id) != ts_Nighttime)
{
    tilemap_tileset(map_id, ts_Nighttime);
}

O código acima verifica o atual tile set atribuído à camada "Tiles_trees" e, se não for "ts_Nighttime", então o tile set é atribuído ao mapa tile.
