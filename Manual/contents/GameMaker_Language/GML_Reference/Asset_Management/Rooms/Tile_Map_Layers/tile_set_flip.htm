tile_set_flip

Esta função pode ser usada para definir um determinado conjunto de dados de azulejos para virar o tile ou não. Você dá os dados do azulejo, que podem ser recuperados usando a função tilemap_get()e então defina o argumento de inverter para true se você quiser que o tile seja invertido, ou false se você quiser que o tile esteja em seu estado padrão, não invertido. A função retornará um conjunto modificado de dados de azulejos que pode então ser aplicado usando o tilemap_set() função.

Sintaxe:

tile_set_flip(tiledata, flip)

ArgumentoTipo Descrição
tiledata the tile-data to set
flip Set to true para virar e false para deixar as coisas como estão

Devoluções:

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_flip(data);
data = tile_set_flip(data, bool);
tilemap_set(map_id, data, mx, my);

O código acima obtém o ID do mapa tile da camada dada e depois obtém a posição das células x e y para o tile sob o mouse. Esta posição é então usada para obter os dados do azulejo que é folheado e depois usado para definir novamente o tile.
