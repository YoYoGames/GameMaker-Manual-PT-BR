layer_tilemap_destroy

Esta função destruirá o dado elemento do mapa tile. Você fornece o ID do mapa tile (que você obtém ao criar o mapa tile usando layer_tilemap_create() ou quando você usa a identificação da camada junto com layer_get_tilemap_id()) e isto irá removê-lo. Note que isto não remove a camada, apenas o mapa tile dele, e se o mapa tile for um mapa que foi adicionado no editor room, então na próxima vez que você deixar o room e depois retornar, o mapa tile será recriado novamente. Entretanto, se o room for persistente, o mapa tile será removido, a menos que a persistência do room seja desligada novamente.

Sintaxe:

layer_tilemap_destroy(tilemap_element_id)

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map to be destroyed

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

O código acima verifica a camada "Tiles_trees" para ver se o elemento do mapa tile dado existe e se existe, então ele é destruído (mas não a camada).
