tilemap_get

Usando esta função você pode recuperar os dados tile a partir de uma célula do elemento do mapa tile. Você dá o ID do elemento do mapa tile (que você obtém quando cria um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), bem como a posição das células x e y e a função devolverá o "blob" de dados tile. Estes dados são essencialmente uma máscara de bit que contém o índice tile, as booleanas flip/rotate/mirror e quaisquer dados de máscara que tenham sido aplicados (veja tilemap_set_mask() para detalhes), e o valor dos dados resultantes pode então ser usado nas funções tile_* para alterar uma propriedade tiles. Note que estamos usando posições x/y de célula com base no tamanho da grade do mapa tile e não como uma posição no room, então "cell_x" é um valor de 0 à largura do mapa de azulejos, e "cell_y" é um valor de 0 à altura do mapa de azulejos. Se você precisa obter os dados para um tile em uma posição específica no room, então você precisa usar a função tilemap_get_at_pixel().

IMPORTANTE! Se o tiles no mapa tile não tiver sido alterado (ou seja: eles não estão girados ou invertidos etc...), então o valor de retorno dos dados do tile set será exatamente igual ao índice do tile no tile set. Assim você pode criar "mapas de colisão" de tiles usando um tile no índice 1 no tile set - por exemplo - então use esta função para verificar 1 ou 0 (um tile vazio) para calcular as colisões.

Sintaxe:

tilemap_get(tilemap_element_id, cell_x, cell_y);

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to get the tile-data of
x_cell Real The cell along the x-axis to get the tile data from
y_cell Real The cell along the y-axis to get the tile data from

Devoluções:

Tile Data (-1 if there is an error)

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

O código acima obtém a identificação para o mapa tile na camada "Tiles_Sky" e depois a usa para obter os dados do tile na célula (0, 0). Estes dados do tile são então virados antes de serem usados para definir novamente o tile no mapa tile.
