mapa_móvel_altura_do_set

Esta função pode ser usada para redimensionar um elemento do mapa tile. Você dá o ID do elemento do mapa tile (que você obtém ao criar um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e a nova altura do mapa tile em tile células.

Sintaxe:

tilemap_set_height(tilemap_element_id, height)

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to set the height of
height Real The height value (in "cells")

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_height(map_id) != room_height div 16)
{
    tilemap_set_height(map_id, room_height div 16);
}

O código acima verifica a largura de um mapa tile específico e, se não for o tamanho correto, então a largura é definida.
