tilemap_get_height

Usando esta função você pode recuperar a altura (em células) do elemento do mapa tile. Você dá o ID do elemento do mapa tile (que você obtém ao criar um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e a função devolverá a altura (número de células de largura).

Sintaxe:

tilemap_get_height(tilemap_element_id);

ArgumentoTipo Descrição
tilemap_element_id The unique ID value of the tile map element to get the height of

Devoluções:

Exemplo:

var lay_id = layer_get_id("espadrilles");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

O código acima usa o ID do mapa tile recuperado para obter a largura e altura tile do mapa tile e depois usa esses valores para definir uma variável de instância.
