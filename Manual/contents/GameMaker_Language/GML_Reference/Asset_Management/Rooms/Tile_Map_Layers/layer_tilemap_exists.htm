layer_tilemap_existe

Você pode usar esta função para verificar e ver se existe um elemento do mapa tile em qualquer camada. Você fornece o ID da camada (que você obtém ao criar a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e o ID do elemento do mapa tile (que você obtém ao criar um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()) e a função retornará ou true se o elemento existir, ou false se não existir.

NOTA: Esta função funciona dentro do escopo do alvo atual room - por padrão o room no qual a função é chamada - que pode ser definido usando a função layer_set_target_room().

Sintaxe:

layer_tilemap_exists(layer_id, tilemap_element_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target (or the layer name as a string)
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to check

Devoluções:

Boolean

Exemplo:

var lay_id = layer_get_id("tilemap_trees");
if (layer_tilemap_exists(lay_id, global.Treestilemap))
{
    layer_tilemap_destroy(lay_id, global.Treestilemap);
}

O código acima verifica a camada "tilemap_trees" para ver se o elemento do mapa tile dado existe e se existe, então ele é destruído (mas não a camada).
