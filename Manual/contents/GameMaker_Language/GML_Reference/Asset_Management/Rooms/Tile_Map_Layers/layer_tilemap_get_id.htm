layer_tilemap_get_id

Esta função pode ser usada para recuperar o valor de identificação único do elemento do mapa tile em uma camada. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a função retornará o valor de identificação associado ao elemento do mapa tile na camada. Note que esta função é especificamente projetada para uso com mapas tile que foram adicionados no IDE, como se você adicionasse um mapa tile a uma camada através de código usando a função layer_tilemap_create()então ele devolverá a identificação única para o elemento do mapa tile adicionado. Se o ID do mapa de azulejos estiver incorreto ou se o mapa de azulejos não existir, a função retornará -1.

Sintaxe:

layer_tilemap_get_id(layer_id)

Argumento Tipo Descrição
layer_id Cordão ou Identificação da camada The unique ID value of the layer to target (or the layer name as a string)

Devoluções:

Tile Map Element ID or -1

Exemplo:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

O código acima obterá o ID da camada chamada "tilemap_trees" e depois o utilizará para obter o ID do elemento do mapa tile nessa camada. Este ID é então usado para remover o mapa tile da camada.
