tile_get_empty

Esta função pode ser usada para verificar se um determinado conjunto de dados de azulejos é para um tile vazio ou não. Você dá os dados do azulejo, que podem ser recuperados usando a função tilemap_get(), e a função retornará true se o tile estiver vazio, ou false se houver um índice tile.

Sintaxe:

tile_get_empty(tiledata)

Argumento Tipo Descrição
tiledata Dados sobre os azulejos the tile-data to check

Devoluções:

Boolean

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
for (var i = 0; i < tilemap_get_width(map_id); i++;)
{
    for (var j = 0; j < tilemap_get_height(map_id); j++;)
    {
        var data = tilemap_get(map_id, i, j);
        if (!tile_get_empty(data))
        {
            data = tile_set_empty(data)
            tilemap_set(map_id, data, i, j);
        }
    }
}

O código acima obtém o ID do mapa tile da camada dada e então procede à verificação de cada célula tile no mapa para ver se ela tem ou não dados. Se tiver, o tile está configurado para esvaziar.
