mapa_get_mask do azulejo

Esta função pode ser usada para recuperar o valor da máscara de bit para o mapa tile dado. Você dá o ID do elemento do mapa tile (que você obtém ao criar um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e a função retornará o valor atual da máscara ou -1 se houver um erro ou 0 se nenhuma máscara for especificada. Para mais informações sobre as máscaras de bit de mapa tile, veja a função tilemap_set_mask().

Sintaxe:

tilemap_get_mask(tilemap_element_id);

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to get the mask of

Devoluções:

Real (0 for no mask, -1 for an error)

Exemplo:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
    tilemap_set_mask(_map_id, _new_mask);
}

O código acima obtém o ID do mapa tile da camada dada e então verifica o valor da máscara associada a ela. Se não for o mesmo que o valor definido na variável "new_mask", então ele é definido para esse valor.
