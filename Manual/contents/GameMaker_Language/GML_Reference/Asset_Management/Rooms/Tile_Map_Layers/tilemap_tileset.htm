tilemap_tileset

Usando esta função, você pode alterar o tile set asset atribuído a um determinado elemento do mapa tile em uma camada. Você dá o ID do elemento do mapa tile (que você obtém quando cria um elemento do mapa tile usando layer_tilemap_create() ou quando você usa a função layer_tilemap_get_id()), e depois fornecer um índice tile set e o mapa tile receberá o novo sprite.

Sintaxe:

tilemap_tileset(tilemap_element_id, tileset_index)

Argumento Tipo Descrição
tilemap_element_id Identificação do Elemento do Mapa de Telha The unique ID value of the tile map element to change
tileset_index Conjunto de azulejos Ativo The new tile set index to use

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_tileset(tile_id) != ts_Nighttime)
{
    tilemap_tileset(tile_id, ts_Nighttime);
}

O código acima verifica o atual tile set atribuído ao mapa tile na camada "Tiles_trees" e se não for "ts_Nighttime" então que tile set é atribuído ao mapa tile.
