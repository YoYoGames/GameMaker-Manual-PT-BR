room_first

Esta variável lida apenas retorna o índice do primeiro room do jogo (isto é definido pela ordem em que o rooms aparece no Gerente de Sala e não pela ordem em que eles foram criados).

Sintaxe:

room_first;

Devoluções:

Room Asset

Exemplo:

if lives < 1
{
    room_goto(room_first);
}

O código acima verificará a variável "vidas" e se for menos de 1, vá para o primeiro room no jogo.
