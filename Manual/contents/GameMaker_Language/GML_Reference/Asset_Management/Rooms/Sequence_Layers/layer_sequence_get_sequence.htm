layer_sequence_get_sequence

Com esta função você fornece o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e devolverá a estrutura de objetos sequence. Esta função supera a necessidade de primeiro obter a instância sequence struct e permite o acesso direto aos dados de sequence. Você pode saber mais sobre o formato do sequence object struct nesta página.

Sintaxe:

layer_sequence_get_sequence(sequence_element_id)

ArgumentoTipo Descrição
sequence_element_id The unique ID value of the sequence element to target

Devoluções:

Sequence Object Struct

Exemplo:

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_sequence(_seq);
_struct.playbackSpeedType = spritespeed_framespersecond;
_struct.playbackSpeed = 30;

O código acima cria um novo elemento sequence e depois recupera seus dados sequence struct . Isto é então usado para alterar a velocidade de reprodução e o tipo do sequence.
