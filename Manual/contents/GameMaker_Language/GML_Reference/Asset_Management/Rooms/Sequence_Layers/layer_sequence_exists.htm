layer_sequence_exists

Com esta função, você pode verificar se existe um elemento sequence na camada dada. Você fornece o ID da camada que pode ser um string do nome da camada - como definido no editor room - ou o ID exclusivo da camada - como retornado pela função layer_get_id()bem como o ID do elemento sequence - como retornado por layer_sequence_create() ou por uma das funções dos elementos da camada - e retornará true se o elemento em questão existir ou false caso contrário.

Sintaxe:

layer_sequence_exists(layer_id, sequence_element_id)

Argumento Tipo Descrição
layer_ID Identificaçãoporcorda ou camada The unique ID or name of the layer to check
sequence_element_id Identificação do Elemento de Sequência The unique ID value of the sequence element to target

Devoluções:

Boolean

Exemplo:

if !layer_sequence_exists("Background", my_seq)
{
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
}

O código acima verifica se o elemento dado sequence existe, e se não existir, então cria um novo sequence na camada dada, então o pausa.
