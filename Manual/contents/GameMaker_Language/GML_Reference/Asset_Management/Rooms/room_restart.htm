room_restart

Esta função reiniciará o atual room, como se ele tivesse acabado de ser inserido. Note que o room não reiniciará até o final do evento onde a função foi chamada, portanto, qualquer código após este ter sido chamado ainda será executado se no mesmo evento. Esta função também irá acionar o evento Room End.

Sintaxe:

room_restart();

Devoluções:

Exemplo:

if lives < 1 room_restart();

O código acima verifica se a variável "vive" é inferior a 1 e se é ela irá reiniciar o room.
