room_goto

Esta função permite que você vá a qualquer room em seu projeto de jogo, seja criado usando código ou no Asset Browser. Você fornece o índice room (armazenado na variável para o nome room, ou como uma variável retornada da função room_add()). Note que o room não mudará até o final do evento onde a função foi chamada, portanto, qualquer código após esta ter sido chamada ainda será executado se no mesmo evento. Esta função também acionará o evento Room End.

NOTA Room Os IDs não são baseados em seu pedido no Asset Browser ou no Room Manager, e assim você deve evitar fornecer um valor numérico diretamente. Em vez disso, use a constante de sala para o asset que você deseja referenciar (que é simplesmente seu nome) ou recuperá-lo através de uma função.

Sintaxe:

room_goto(index);

ArgumentoTipo Descrição
index The index of the room to go to.

Devoluções:

Exemplo:

switch (global.level)
{
    case 0: room_goto(rm_level1); break;
    case 1: room_goto(rm_level2); break;
    case 2: room_goto(rm_level3); break;
}

O código acima verificará uma variável global e mudará room com base no valor que ela contém.
