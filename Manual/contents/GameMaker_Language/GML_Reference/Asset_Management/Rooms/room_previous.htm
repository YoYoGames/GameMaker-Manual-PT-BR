room_previous

Com esta função, você pode recuperar o índice do room antes da entrada do room na função. Por exemplo, você pode usar a variável room para obter o índice do atual room e depois usar esta função para encontrar o room que vem antes dele, como listado no Room Manager. Se não houver room antes da entrada, então -1 é devolvido. Note que esta função não reconhecerá ou levará em consideração rooms que tenham sido adicionados dinamicamente usando room_add() ou room_duplicate().

Sintaxe:

room_previous(numb);

ArgumentoTipo Descrição
numb The index of the room to get the one before from.

Devoluções:

Room Asset

Exemplo:

if room_previous(room) != -1
{
    room_goto_previous();
}

O código acima verificará se o anterior room existe e, em caso afirmativo, irá até ele.
