room_goto_previous

Com esta função você pode fazer seu jogo ir para o anterior, conforme listado no Gerente de Sala no momento em que o jogo foi compilado. Se este room não existir, um erro será lançado e o jogo será forçado a fechar. Note que o room não mudará até o final do evento onde a função foi chamada, portanto, qualquer código após este ter sido chamado ainda será executado se no mesmo evento. Esta função também acionará o evento Room End.

Sintaxe:

room_goto_previous();

Devoluções:

Exemplo:

if room_exists(room_previous(room))
{
    room_goto_previous();
}

O código acima verificará se há outro room antes do atual e, se for o caso, irá para aquele room.
