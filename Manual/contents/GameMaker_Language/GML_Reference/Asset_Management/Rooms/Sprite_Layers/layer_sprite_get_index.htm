layer_sprite_get_index

Esta função pode ser usada para obter o valor atual do índice de imagem do elemento sprite. Você dá o ID do elemento sprite (que você obtém ao criar um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e a função retornará o valor real que representa o índice de imagem que está sendo mostrado para o sprite. A função retornará -1 se o elemento sprite não existir ou se o elemento não tiver um sprite válido atribuído a ele.

Sintaxe:

layer_sprite_get_index(sprite_element_id);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to get the information from

Devoluções:

Real or -1

Exemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_index(spr_id) < 4)
{
    layer_sprite_index(spr_id, 4);
}

O código acima obterá o ID da camada chamada "sprite_sky" e depois o usará para obter o ID do elemento sprite nessa camada. Este ID é então usado para verificar se o índice de imagem do elemento é inferior a 4, e se for o caso, é ajustado para 4.
