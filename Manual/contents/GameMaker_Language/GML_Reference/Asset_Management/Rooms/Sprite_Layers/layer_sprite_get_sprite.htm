layer_sprite_get_sprite

Esta função pode ser usada para obter o índice atual sprite do elemento sprite. Você dá o ID do elemento sprite (que você obtém quando cria um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e a função retornará um valor real que representa o índice sprite que está sendo mostrado. Se o elemento não tiver um sprite atribuído, a função retornará -1.

Sintaxe:

layer_sprite_get_sprite(sprite_element_id);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to get the information from

Devoluções:

Sprite Asset or -1

Exemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
    layer_sprite_change(spr_id, spr_Clouds);
}

O código acima obterá o ID da camada chamada "sprite_sky" e depois o usará para obter o ID do elemento sprite nessa camada. Este ID é então usado para verificar o sprite atribuído ao elemento, definindo-o para sprite "spr_Clouds" se ainda não o for.
