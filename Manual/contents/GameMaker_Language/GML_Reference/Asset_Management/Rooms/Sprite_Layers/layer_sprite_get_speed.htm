velocidade_escrita_escrita_por_camada

Esta função pode ser usada para obter o valor atual do multiplicador de velocidade do elemento sprite. Você dá o ID do elemento sprite (que você obtém ao criar um elemento sprite usando layer_sprite_create() ou quando você usa a função layer_sprite_get_id()), e a função retornará um valor real que representa o multiplicador de velocidade sendo usado para animar o sprite. O valor padrão é 1.

Sintaxe:

layer_sprite_get_speed(sprite_element_id);

ArgumentoTipo Descrição
sprite_element_id The unique ID value of the sprite element to get the information from

Devoluções:

Exemplo:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_speed(spr_id) > 0)
{
    layer_sprite_speed(spr_id, 0);
}

O código acima obterá o ID da camada chamada "sprite_sky" e depois o usará para obter o ID do elemento sprite nessa camada. Este ID é então usado para verificar a velocidade da animação para o elemento e se for maior que 0, é ajustado para 0.
