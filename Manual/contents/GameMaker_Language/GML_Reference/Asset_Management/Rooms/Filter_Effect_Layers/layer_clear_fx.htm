layer_clear_fx

Esta função é usada para limpar o FX struct que é aplicado na camada especificada. Você especifica ou o ID ou o nome da camada que deseja modificar e esta função removerá o FX struct que é aplicado a essa camada, desabilitando completamente seus efeitos.

Se você usar esta função para desativar temporariamente um efeito e precisar restaurá-lo na camada, você deve primeiro salvar o FX struct para a camada ligando para layer_get_fx() e depois chamar esta função para limpá-la na camada. Mais tarde, você pode aplicá-lo de volta à camada chamando layer_set_fx(), o que irá reativar o efeito.

Sintaxe:

layer_clear_fx(layer_name_or_id);

Argumento Tipo Descrição
layer_name_or_id Cordão ou Identificação da camada The name or ID of the layer to clear the FX for

Devoluções:

N/A

Exemplo:

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

O código acima verifica se a tecla "E" está pressionada, e então verifica se a camada "EffectLayer" tem um filtro/efeito aplicado a ela; nesse caso, armazena sua FX struct em uma variável temporária e limpa a camada de qualquer filtro/efeito. Quando a tecla "F" é pressionada, ela verifica se a variável temporária contém algo diferente de -1, e nesse caso aplica esse FX de volta à camada e redefine a variável temporária para -1. Tudo isso permite essencialmente ativar e desativar um efeito sempre que necessário, sem perder seus parâmetros.
