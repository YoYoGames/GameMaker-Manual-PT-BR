layer_enable_fx

Esta função habilita/desabilita o filtro/efeito atribuído a uma camada Room. Você especifica o ID ou o nome da camada que deseja modificar e, em seguida, um valor booleano informando se o FX deve ser ativado(verdadeiro) ou desativado(falso).

Passar em falso não removerá o FX da camada, mas simplesmente o tornará invisível. Use layer_clear_fx() para remover um FX de uma camada.

Da mesma forma, a passagem em true não fará nada se não houver FX aplicada sobre a camada. Use layer_set_fx() para aplicar um FX a uma camada.

Sintaxe:

layer_enable_fx(layer_name_or_id, enable);

Argumento Tipo Descrição
layer_name_or_id Cordão ou Identificação da camada The name or ID of the layer to modify
enable Booleano Whether to enable or disable the FX

Devoluções:

N/A

Exemplo:

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

O código acima permite uma camada FX Desaturate se o valor HP da instância for menor ou igual a 3, caso contrário o desativa. Isto indica ao jogador que seu HP é baixo, ao desnaturar a tela.
