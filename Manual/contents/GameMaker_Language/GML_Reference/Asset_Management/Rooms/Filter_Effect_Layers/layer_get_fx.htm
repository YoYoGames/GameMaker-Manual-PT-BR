layer_get_fx

Esta função é utilizada para recuperar o FX struct por uma camada. Você especifica ou o ID ou o nome da camada que você deseja atingir e a função retornará um struct contendo informações sobre seu efeito aplicado. Este struct será semelhante ao struct que você obtém de fx_create(), e as funções fx_get_parameter/s e fx_set_parameter/s pode ser usado nele para ler e modificar seus parâmetros.

Se a camada especificada não tiver filtros/efeitos aplicados a ela, a função retornará -1.

Sintaxe:

layer_get_fx(layer_name_or_id);

Argumento Tipo Descrição
layer_name_or_id Identificaçãoporcorda ou camada The name or ID of the layer to read

Devoluções:

FX Struct (or -1 if not found)

Exemplo:

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

O código acima executa um for percorre todas as camadas presentes no room, e verifica cada camada em busca de um FX struct. Se uma camada tem um FX struct, verifica o nome desse filtro/efeito chamando fx_get_name() sobre ele; se for igual a "_filter_tintfilter", o que significa que é um filtro "Colour Tint", ele muda sua cor de tonalidade para azul.
