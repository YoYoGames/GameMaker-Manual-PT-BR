layer_fx_is_enabled

Esta função retorna se o filtro/efeito para uma camada está habilitado.

Você especifica ou o ID ou o nome da camada que deseja alvejar, e a função retorna um valor booleano indicando se a FX da camada está habilitada(verdadeira) ou desabilitada(falsa).

Sintaxe:

layer_fx_is_enabled(layer_name_or_id);

Argumento Tipo Descrição
layer_name_or_id Cordão ou Identificação da camada The name or ID of the layer to check

Devoluções:

Boolean

Exemplo:

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

O código acima "respawns" a instância, movendo-a para sua posição original e recarregando seu HP. Em seguida, verifica se a camada FX Desaturate está habilitada e, se estiver, a desabilita.
