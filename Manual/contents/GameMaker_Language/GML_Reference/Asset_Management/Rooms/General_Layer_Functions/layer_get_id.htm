layer_get_id

Esta função pode ser usada para obter o valor de identificação único para uma determinada camada. No IDE, todas as camadas têm um nome e um tipo, e para poder editá-las ou alterá-las através do código, é preciso dar o valor de ID da camada. Esta função é usada para recuperar este ID usando o nome (a string) da camada (como escrito no IDE). Se você criar uma nova camada através de código usando a função layer_create() em vez disso, essa função retornará o valor de identificação único (camadas criadas dinamicamente não recebem nomes). Note que se você der o nome de uma camada que não existe no atual room, então você receberá um erro e o projeto irá falhar.

Sintaxe:

layer_get_id(layer_name)

ArgumentoTipo Descrição
layer_name The name of the layer (a string)

Devoluções:

Layer ID or -1 (if the layer specified doesn't exist)

Exemplo:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

O código acima obterá primeiro o índice da instância mais próxima à posição x/y dada e o armazenará em uma variável local. Em seguida, ele obtém a identificação única da instância da camada chamada "Instances Front", e move a instância encontrada para essa camada.
