layer_get_name

Você pode usar esta função para obter o nome da camada dada. Você fornece o valor único de identificação da camada e se a camada for uma das camadas nomeadas criadas no editor room, então a função retornará um string com o nome da camada. Se a camada não for uma das camadas do editor room (ou seja: foi criada usando layer_create()) então um string vazio será devolvido.

Sintaxe:

layer_get_name(layer_id)

Argumento Tipo Descrição
layer_id Identificação da camada The unique ID value of the layer to get the name of

Devoluções:

String

Exemplo:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_name(a[i]) != ""
    {
        ds_list_add(layer_list, a[i])
    }
}

O código acima obtém as identificações de todas as camadas no room e depois no loops, embora eles verifiquem se há camadas nomeadas. Se forem, então sua identificação é adicionada a uma lista.
