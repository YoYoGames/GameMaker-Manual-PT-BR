layer_x

Você pode usar esta função para definir a posição x da camada dentro do escopo atualmente escopo room. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a função moverá a camada o número dado de pixels ao longo do eixo horizontal do room.

Sintaxe:

layer_x(layer_id, x)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to set the x position of
x Real The x position in the room to set the layer to

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

O código acima verifica a posição da camada dada e se não estiver definido para (0, 0), então está definido para essa posição.
