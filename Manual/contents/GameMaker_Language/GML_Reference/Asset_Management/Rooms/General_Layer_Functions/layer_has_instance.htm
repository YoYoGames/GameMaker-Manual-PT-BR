camada_has_instance

Esta função pode ser usada para verificar se uma determinada instância está atualmente atribuída à camada em questão. Você fornece o ID da camada (que você obtém ao criar a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a identificação da instância a ser verificada. Você também pode dar um object_index (ou seja: o nome do object no navegador Asset ) e a função verificará se alguma instância desse object está na camada dada. A função retornará true se a instância estiver na camada e false se não estiver.

Sintaxe:

layer_has_instance(layer_id, instance_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target (or the layer name as a string)
instance_id Identificação da instância The unique instance ID or the object index of the instance to check for

Devoluções:

Boolean

Exemplo:

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

O código acima verificará se a camada dada contém alguma instância do object "obj_Bullet_Parent" e, caso contrário, destruirá a camada.
