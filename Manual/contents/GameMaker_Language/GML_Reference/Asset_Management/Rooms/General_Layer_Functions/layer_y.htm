camada_y

Você pode usar esta função para definir a posição y da camada dentro do escopo atual room. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a função moverá a camada o número dado de pixels ao longo do eixo vertical do room.

Sintaxe:

layer_y(layer_id, y)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to set the y position of
y Real The y position in the room to set the layer to

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

O código acima verifica a posição da camada dada e se não estiver definido para (0, 0), então está definido para essa posição.
