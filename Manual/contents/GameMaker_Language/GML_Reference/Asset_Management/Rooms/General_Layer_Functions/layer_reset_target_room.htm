layer_reset_target_room

Esta função é utilizada para redefinir o alvo da camada para a atual room. Veja a função layer_set_target_room() para maiores informações.

Sintaxe:

layer_reset_target_room()

Devoluções:

Exemplo:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
{
    layer_sprite_create(l, irandom(1000), irandom(1000), spr_Trees);
}
layer_reset_target_room();

O código acima estabelece a meta room para o room "rm_Game" e então obtém a identificação da camada chamada "SpriteAssets" naquele room. Este ID de camada é então usado para adicionar 50 sprite assets aleatórios à camada, antes que o alvo da camada seja redefinido para o room atual.
