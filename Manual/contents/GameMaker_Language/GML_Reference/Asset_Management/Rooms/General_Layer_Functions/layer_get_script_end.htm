layer_get_script_end_end

Esta função retorna o índice da função de script da função atribuída para ser executada no final da renderização da camada dada, ou retornará -1 se nenhuma função for atribuída. Você fornece o ID da camada (que você obtém ao criar a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho). Você pode atribuir funções script a uma camada com layer_script_begin() e layer_script_end().

Sintaxe:

layer_get_script_end(layer_id);

Argumento Tipo Descrição
layer_id Cordão ou Identificação da camada The unique ID value of the layer to target (or the layer name as a string)

Devoluções:

Script Function or -1 (if no function is assigned)

Exemplo:

if layer_get_script_end(layer) == -1
{
    layer_script_end(layer, scr_ResetShaderValues);
}

O código acima verificará se a camada que executa o código tem uma função script atribuída a ele e se não tem uma função atribuída.
