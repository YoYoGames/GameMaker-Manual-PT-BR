layer_get_shader

Esta função pode ser usada para verificar se a camada dada tem um shader atribuído a ela. Você fornece o ID da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho), e a função retornará ou o índice shader do shader atribuído, ou -1 se não for atribuído shader.

Sintaxe:

layer_get_shader(layer_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target (or the layer name as a string)

Devoluções:

Shader Asset or -1 (if no shader is assigned)

Exemplo:

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

O código acima verificará se a camada que a instância que está executando o código tem um shader atribuído a ela e se não tem um atribuído.
