layer_shader

Com esta função você pode atribuir um recurso shader a qualquer camada determinada e a camada será então renderizada usando esse shader. Você fornece ou o valor de identificação único da camada ou o nome da camada (como um string - isso terá um impacto no desempenho), juntamente com a identificação do shader a ser utilizado. O shader deve ter sido criado anteriormente no navegador Asset e o índice shader (o nome do recurso shader ) é então passado para esta função. Se a camada atribuída tiver instâncias adicionadas a ela, então o shader será aplicado a todos os eventos de sorteio que a instância utilizar - por exemplo, se a instância tiver um evento de início de GUI de sorteio, então o shader será aplicado automaticamente a ela. O shader também afetará quaisquer outros elementos gráficos desenhados nessa camada, como os mapas sprite assets ou tile.

Observe que a função não se destina a ser chamada em nenhum evento de sorteio ou evento de etapa, mas somente no início do room no Código de Criação de Sala ou no Evento de Criação de Sala / Evento de Início de Sala de uma instância.

Sintaxe:

layer_shader(layer_id, shader)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to target (or the layer name as a string)
shader Ativo Shader The shader index to assign to the layer

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

O código acima atribuirá o recurso shader "shd_Sepia" à camada dada para todos os desenhos.
