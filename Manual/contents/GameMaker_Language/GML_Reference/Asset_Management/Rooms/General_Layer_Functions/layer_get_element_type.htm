layer_get_element_type

Você pode usar esta função para obter o tipo de elemento para o elemento em questão. Você fornece o valor de identificação único do elemento (por exemplo, como retornado a função que criou o elemento ou do editor room ) e a função retornará uma das seguintes constantes (ou -1 se o elemento não existir ou se o valor de identificação estiver errado):

Constante do tipo de elemento de camada
Constante Descrição
layerelementtype_background O elemento é um pano de fundo.
layerelementtype_instance O elemento é um exemplo.
layerelementtype_sprite O elemento é um sprite asset.
layerelementtype_tilemap O elemento é um mapa de azulejos.
layerelementtype_particlesystem O elemento é um sistema de partículas.
layerelementtype_tile O elemento é um legado de fundo tile (isto só é válido para projetos que foram importados de versões anteriores de GameMaker).
layerelementtype_sequence O elemento é um sequence asset.

Note que esta função é mais útil quando você tem vários tipos diferentes de elementos atribuídos a uma camada, e que você pode obter uma lista de todos os elementos de uma determinada camada usando a função layer_get_all_elements().

Sintaxe:

layer_get_element_type(element_id)

Argumento Tipo Descrição
layer Identificação do elemento da camada The unique ID value of the element to get the type of

Devoluções:

Layer Element Type Constant or -1 (if element does not exist or is invalid)

Exemplo:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i]);
    }
}

O código acima obtém as identificações de todos os elementos da instância designados para a camada da instância que executa o código. O código então verifica se algum dos elementos retornados é sprite assets e se eles são destruídos.
