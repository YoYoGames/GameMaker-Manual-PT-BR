layer_get_y

Você pode usar esta função para recuperar a posição y da camada dentro do escopo atual room. Você fornece a identificação da camada (que você obtém quando cria a camada usando layer_create()) ou o nome da camada (como um string - isso terá um impacto no desempenho) e a função retorna um número real para a posição x da camada, relativo à posição (0,0) do room. O padrão é 0.

Sintaxe:

layer_get_y(layer_id)

Argumento Tipo Descrição
layer_id Identificaçãoporcorda ou camada The unique ID value of the layer to get the y position of

Devoluções:

Real

Exemplo:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

O código acima verifica a posição da camada dada e se não estiver definido para (0, 0), então está definido para essa posição.
