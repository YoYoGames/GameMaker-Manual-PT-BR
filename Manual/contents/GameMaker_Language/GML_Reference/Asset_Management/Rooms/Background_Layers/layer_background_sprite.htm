layer_background_sprite

Usando esta função, você pode definir o índice sprite do elemento de fundo. Você fornece o ID do elemento de fundo (que você obtém quando cria um elemento de fundo usando layer_background_create() ou quando você usa a função layer_background_get_id()), e depois dar um índice sprite para ser usado. A imagem do elemento de fundo será substituída pelo novo sprite. Se você der um valor -1, o elemento não terá nenhum sprite atribuído (mas ainda existirá e poderá ter um sprite atribuído novamente mais tarde).

Esta função também tem o pseudônimo layer_background_change().

Sintaxe:

layer_background_sprite(background_element_id, sprite_index)

Argumento Tipo Descrição
background_element_id Identificação do elemento de fundo The unique ID value of the background element to change
sprite_index Ativo Sprite The sprite index of the sprite to use for the background element

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_sprite(back_id) != spr_Clouds)
{
    layer_background_sprite(back_id, spr_Clouds);
}

O código acima obterá a identificação da camada chamada "Background_sky" e depois a usará para obter a identificação do elemento de fundo nessa camada. Este ID é então usado para verificar o sprite atribuído ao elemento, definindo-o para sprite "spr_Clouds" se ainda não o for.
