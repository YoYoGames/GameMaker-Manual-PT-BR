layer_background_destroy

Esta função destruirá o elemento de fundo dado. Você fornece a identificação do fundo (que você obtém quando cria o fundo usando layer_background_create() ou quando você usa a identificação da camada junto com layer_get_background_id()) e isto irá removê-lo. Note que isto não remove a camada, apenas o fundo da mesma, e se o fundo for um que foi adicionado no editor room, então na próxima vez que você deixar o room e depois retornar, o fundo será recriado novamente. Entretanto, se o room for persistente, o fundo será removido, a menos que a persistência do room seja desligada novamente.

Sintaxe:

layer_background_destroy(background_element_id)

Argumento Tipo Descrição
background_element_id Identificação do elemento de fundo The unique ID value of the background element to be destroyed

Devoluções:

N/A

Exemplo:

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if (layer_background_exists(lay_id, bck_id))
{
    layer_background_destroy(bck_id);
}

O código acima verifica a camada "Background_trees" para ver se o elemento de fundo dado existe e se ele existe, então ele é destruído (mas não a camada).
