shader_current

Esta função retornará o valor do índice ID do shader que está sendo usado atualmente para renderização, ou retornará -1 se não estiver sendo usado shader.

Sintaxe:

shader_current();

Devoluções:

Shader Asset or -1 (if no shader is assigned)

Exemplo:

if (shader_current() == -1)
{
    shader_set(sh_warp);
}

O código acima verificará o que é o atual shader e se ele retornar -1 (não shader sendo usado) então um shader é definido.
