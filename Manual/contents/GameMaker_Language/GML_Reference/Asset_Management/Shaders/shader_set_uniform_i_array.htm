shader_set_uniform_i_array

Com esta função você pode definir uma constante shader para manter um conjunto de valores. Você deve ter obtido previamente a "manipulação" da constante usando a função shader_get_uniform()e você terá que ter previamente rubricado a matriz.

NOTA: Todos os uniformes devem ser ajustados após a chamada da função shader_set()e antes de chamar shader_reset().

Sintaxe:

shader_set_uniform_i_array(handle, array);

Argumento Tipo Descrição
handle Cabo Uniforme Shader The handle of the shader constant to set.
array Array A previously initialised array of integer values.

Devoluções:

N/A

Exemplo:

shader_set(shader_Glass);
col_array[0] = 255;
col_array[2] = 255;
col_array[3] = 64;
col_array[4] = 128;
shader_params = shader_get_uniform(shader_tint, "cColourArray");
shader_set_uniform_i_array(shader_params, col_array);
draw_self();
shader_reset();

O código acima receberá o comando da constante shader "cColourArray" e então definirá essa constante para a matriz dada.
