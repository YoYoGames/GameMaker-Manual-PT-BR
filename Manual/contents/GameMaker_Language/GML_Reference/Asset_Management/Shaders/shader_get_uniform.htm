shader_get_uniform

Como você não pode alterar o valor de uma constante shader dentro do próprio shader, você tem que configurá-la antes de ligar para o shader usando uma das funções do conjunto uniforme disponível. Entretanto, para poder fazer isso, você deve primeiro chamar esta função para obter o "handle" da constante shader que você vai querer mudar.

NOTA: Embora um shader seja composto de dois programas discretos (vertex e fragmento) esta função não diferenciará entre os dois e devolverá o cabo da constante shader de qualquer um deles.

Sintaxe:

shader_get_uniform(shader, uniform);

Argumento Tipo Descrição
shader Ativo Shader The index of the shader to use.
uniform Cordão The shader constant to get the handle of (a string).

Devoluções:

Shader Uniform Handle

Exemplo:

shader_params = shader_get_uniform(shd_glass, "u_vRefractColour");

O código acima receberá o comando da constante shader "u_vRefractColour".
