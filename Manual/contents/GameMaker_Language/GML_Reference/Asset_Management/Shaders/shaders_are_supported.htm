shaders_are_supported

Esta função fará uma verificação para ver se a plataforma alvo escolhida suporta shaders, retornando true se suportarem, e false se não suportarem. É importante observar que no Android, se o projeto não tiver nenhum recurso shader definido, então a função sempre retornará false, independentemente de o dispositivo suportar shaders ou não.

Sintaxe:

shaders_are_supported();

Devoluções:

Exemplo:

global.GFX = false;
if (shaders_are_supported())
{
    if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
    {
        global.GFX = true;
    }
}

O código acima definirá uma variável global para false, e então se a plataforma suportar shaders e ambos os shaders verificados tiverem compilado corretamente, ela será definida para true.
