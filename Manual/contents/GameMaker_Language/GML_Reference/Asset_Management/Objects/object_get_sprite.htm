object_get_sprite

Esta função lhe dirá se o object que você está verificando tem um sprite ou não, e se tiver, então retornará o índice daquele sprite, ou -1 se não tiver. Note que esta não é uma função de instância! Você pode ter um object sem sprite enquanto uma instância desse mesmo object pode ter um e vice-versa, ou eles podem até ter um sprites diferente. Você pode definir uma instância individual sprite usando a função sprite_index variável de instância.

Sintaxe:

object_get_sprite(obj);

ArgumentoTipo Descrição
obj The index of the object to check

Devoluções:

Sprite Asset

Exemplo:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

O exemplo acima verificará o sprite_index da instância em relação ao sprite do object_index da instância. Se não forem os mesmos, então atribuirá o mesmo sprite do índice object à instância.
