event_perform_async

Esta função é utilizada para realizar qualquer um dos Eventos Assíncronos fornecidos no GameMaker. Você fornece a constante de eventos Async (mostrada na tabela abaixo) e um mapa DS que estará disponível no chamado evento Async na variável async_load.

Constante tipo de evento Async
Constante Descrição
ev_async_web_image_load Evento carregado de imagem
ev_async_web Evento HTTP
ev_async_dialog Evento de diálogo
ev_async_web_iap Evento de compra In-App
ev_async_web_cloud Evento em nuvem
ev_async_web_networking Evento de trabalho em rede
ev_async_web_steam Evento a vapor
ev_async_social Evento social
ev_async_push_notification Evento de Notificação Push
ev_async_save_load Salvar/Carregar Evento
ev_async_audio_recording Evento de gravação de áudio
ev_async_audio_playback Evento de reprodução de áudio
ev_async_system_event Evento do sistema

Eventos não assíncronos podem ser chamados usando event_perform().

Observe que o mapa DS especificado no segundo argumento não precisa ser destruído manualmente, pois será automaticamente destruído pela função após o evento Async ter sido realizado.

Sintaxe:

event_perform_async(type, ds_map);

Argumento Tipo Descrição
type Constante tipo de evento Async The type of event to perform (see the table above).
ds_map DS Map ID The DS map to use as async_load no evento chamado.

Devoluções:

N/A

Exemplo:

var _map = ds_map_create();

_map[? "id"] = "custom_async_event";
_map[? "result"] = true;
_map[? "data"] = { a: 13, b: 16 };

event_perform_async(ev_async_social, _map);

O código acima cria um mapa DS e o preenche com entradas personalizadas para ser lido no evento Async. Ele então realiza o evento Async Social com o mapa recém-criado, passado em async_load para o evento chamado.
