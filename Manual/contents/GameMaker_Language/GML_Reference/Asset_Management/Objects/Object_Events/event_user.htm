event_user

Com esta função você diz à instância para executar as ações ou código que foi colocado dentro de um dos 16 eventos definidos pelo usuário. Estes eventos só podem ser chamados desta forma, ou usando o event_perform função.

Sintaxe:

event_user(numb);

Argumento Tipo Descrição
numb Real The number of User Event to call, between 0 and 15.

Devoluções:

N/A

Exemplo:

event_user(4);

Isto chamaria Evento Definido pelo Usuário 4.
