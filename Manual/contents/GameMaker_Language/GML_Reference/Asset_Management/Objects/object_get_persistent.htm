object_get_persistent

Esta função lhe dirá se o object que você está verificando foi assinalado como "persistente" ou não. Um object persistente é aquele que fará com que qualquer instância dele seja realizada de room a room, a menos que seja explicitamente destruído. Por favor, note que esta não é uma função de instância! Portanto, você pode ter uma object persistente e uma instância não-persistente do mesmo object e vice-versa. Você pode definir uma instância individual persistente flag usando a função persistent variável de instância.

Sintaxe:

object_get_persistent(obj);

ArgumentoTipo Descrição
obj The index of the object to check.

Devoluções:

Exemplo:

if (!persistent) && (object_get_persistent(object_index))
{
    persistent = true;
}

O código acima verificará a instância em execução para ver se ela é persistente ou não, assim como verificará o índice object da instância para ver se ela é sinalizada como persistente ou não. Se a instância ainda não for persistente, o índice object será sinalizado como persistente, ele definirá "persistente" como verdadeiro para aquela instância.
