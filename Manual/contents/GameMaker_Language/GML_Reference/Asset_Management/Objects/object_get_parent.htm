object_get_parent

Esta função lhe dará o índice object de qualquer pai que tenha sido designado para o object especificado, ou então retornar -100 para mostrar que o object não tem nenhum pai designado para ele, ou -1 se o object sendo verificado não existir. Para mais informações sobre pais, consulte a seção sobre o Editor de Objetos.

Sintaxe:

object_get_parent(obj);

ArgumentoTipo Descrição
obj The index of the object to check

Devoluções:

Object Asset

Exemplo:

par = object_get_parent(object_index);

O código acima obterá o pai do índice object para a instância que executa o código e armazenará o valor de retorno na variável "par".
