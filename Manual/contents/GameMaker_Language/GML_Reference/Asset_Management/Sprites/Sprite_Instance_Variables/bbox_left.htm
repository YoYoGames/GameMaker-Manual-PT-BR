bbox_left

Esta variável lida somente retorna a posição (ao longo do eixo x) dentro do room da caixa de delimitação da mão esquerda para a instância, onde a caixa de delimitação é definida pela largura e altura máximas da máscara para a instância (conforme definido pelo sprite_index ou pelo mask_index). Mesmo quando um sprite tem uma máscara de colisão precisa, a caixa de delimitação existe e é usada para certas coisas, e assim você pode usar esta variável para encontrá-la. Observe que quando a instância não tem sprite atribuído, o valor retornado será o mesmo que a posição da instância X.

Sintaxe:

bbox_left;

Devoluções:

(integer)

Exemplo:

if bbox_left < 0
{
    x = sprite_xoffset;
}

O código acima verifica se a caixa de delimitação da instância está fora do room e se é ela que coloca a instância em uma nova posição.
