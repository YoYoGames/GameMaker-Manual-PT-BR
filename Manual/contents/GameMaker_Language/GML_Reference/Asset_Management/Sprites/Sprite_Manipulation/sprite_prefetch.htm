sprite_prefetch

Esta função pode ser utilizada para pré-fetar (colocar na memória texture ) uma página texture com o dado sprite. Você fornece o índice sprite conforme definido quando você criou o sprite asset, e a página texture em que ele se encontra será carregada na memória. Note que a função retornará -1 se o prefetch não for suportado para o recurso escolhido ou se a plataforma alvo for HTML5, ou retornará 0 se tudo tiver funcionado corretamente.

OBSERVAÇÃO: Há um hit de desempenho já que o texture é carregado na memória texture na maioria dos dispositivos, por isso é recomendável que você ligue para sprite_prefetch em qualquer gráfico necessário no início de um nível para evitar qualquer barraca.

Sintaxe:

sprite_prefetch(ind)

ArgumentoTipo Descrição
ind The sprite index to fetch

Devoluções:

(-1 or 0)

Exemplo:

sprite_prefetch(spr_Player_Aura);

O código acima colocará o referido sprite na memória texture pronto para uso.
