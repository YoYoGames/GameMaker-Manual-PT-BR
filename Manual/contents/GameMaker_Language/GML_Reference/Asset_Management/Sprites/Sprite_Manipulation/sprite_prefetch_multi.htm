sprite_prefetch_multi

Esta função pode ser usada para pré-fetar (colocar na memória texture ) um número de páginas texture que contêm o sprites dado. Você fornece uma matriz preenchida com os índices sprite (como definido quando você criou o sprite asset) e as páginas texture em que eles se encontram serão carregadas na memória. Note que a função retornará -1 se o prefetch não for suportado para o recurso escolhido ou se a plataforma alvo for HTML5, ou retornará 0 se tudo tiver funcionado corretamente.

OBSERVAÇÃO: Há um hit de desempenho já que o texture é carregado na memória texture na maioria dos dispositivos, por isso é recomendável que você ligue para sprite_prefetch_multi() em qualquer gráfico necessário no início de um nível para evitar qualquer barraca.

Sintaxe:

sprite_prefetch_multi

Argumento Tipo Descrição
array Conjunto de bens Sprite Array with the sprite indices to fetch

Devoluções:

Real (-1 or 0)

Exemplo:

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_prefetch_multi(spr_a);

O código acima cria uma matriz onde cada elemento contém um índice sprite. Este array é então usado para colocar esses sprite textures na memória.
