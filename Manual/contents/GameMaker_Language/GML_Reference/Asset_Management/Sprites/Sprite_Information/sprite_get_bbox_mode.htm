sprite_get_bbox_mode

Esta função retornará o "modo" atual para os cálculos da caixa delimitadora. Você fornece o índice sprite do sprite para verificar, e a função retornará uma das constantes mostradas abaixo.

Sintaxe:

sprite_get_bbox_mode(ind);

ArgumentoTipo Descrição
ind The index of the sprite to check.

Devoluções

Bounding Box Mode Constant

Constante Descrição
bboxmode_automatic Automático - A caixa de delimitação será calculada automaticamente, com base no ajuste de tolerância para o duende
bboxmode_fullimage Full Image - A caixa de delimitação será ajustada para usar toda a largura e altura do sprite, independentemente da tolerância e dos pixels "vazios".
bboxmode_manual Manual - A caixa de delimitação foi definida manualmente para valores definidos pelo usuário (ou no editor sprite, ou usando a função sprite_set_bbox())

Exemplo:

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

O acima verifica o modo bbox para o atual sprite e se não for automatic, então ele é ajustado para esse valor.
