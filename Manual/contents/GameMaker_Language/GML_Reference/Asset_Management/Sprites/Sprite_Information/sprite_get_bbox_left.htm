sprite_get_bbox_left

Esta função retorna a posição relativa da esquerda da caixa de delimitação sprite. Este valor é dado como um valor relativo baseado no canto superior esquerdo da base sprite asset sendo (0,0). é o mesmo valor que pode ser encontrado no editor sprite para as propriedades da máscara de colisão. A imagem abaixo mostra como ela é calculada:

Sintaxe:

sprite_get_bbox_left(ind);

ArgumentoTipo Descrição
ind The index of the sprite to check.

Devoluções

Exemplo:

var ww, hh;
ww = sprite_get_bbox_left(sprite_index) - sprite_get_bbox_right(sprite_index);
hh = sprite_get_bbox_bottom(sprite_index) - sprite_get_bbox_top(sprite_index);

O código acima calcula a largura e a altura da máscara de colisão com base nas posições laterais relativas da caixa de delimitação.
