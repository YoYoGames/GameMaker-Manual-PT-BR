método_get_index

Com esta função você pode recuperar o índice do Script para o script onde o método foi definido. Se o método não foi definido em um script então a função retornará -1, caso contrário, retornará o valor do índice para o script.

Sintaxe:

method_get_index(method);

Argumento Tipo Descrição
method Método The method variable to check

Devoluções:

Script Asset or Real

Exemplo:

var _index = method_get_index(light_setup);
if _index != undefined
{
    show_debug_message(script_get_name(_index));
}

O código acima verificará o índice script para o método "light_setup" e então, se não for -1, emitirá o nome script para o console.
