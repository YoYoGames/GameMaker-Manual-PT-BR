array_equals

Com esta função você pode verificar se duas matrizes são iguais (equivalente ou o mesmo). Você dá as duas arrays para verificar, e a função retornará true se forem iguais (ou equivalente ou o mesmo) ou false se não forem. Note que isto não é o mesmo que verificar se duas matrizes são iguais usando ==, o que não verificará se as duas matrizes possuem valores equivalentes, mas apenas para ver se as matrizes estão se referindo à mesma matriz inicial. Por exemplo:

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

Note que existem algumas constantes que podem não ser iguais a elas mesmas, o que pode fazer esta função falhar. Aqui está um exemplo:

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

Consulte a Tabela de Igualdade para mais informações.

Sintaxe:

array_equals(var1, var2);

Argumento Tipo Descrição
var1 Array The first array.
var2 Array The second array.

Devoluções:

Boolean

Exemplo:

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

O código acima verificará duas arrays para ver se elas possuem valores equivalentes, e se elas não possuem, então o código copiará todo o conteúdo da array "inventory_array" para a array "item_array".
