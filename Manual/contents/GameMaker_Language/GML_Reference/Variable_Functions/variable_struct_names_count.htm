struct_names_count

Com esta função você pode encontrar o número total de variáveis definidas para um struct. Você fornece o ID struct para verificar, e a função retornará um valor inteiro para o número de variáveis encontradas, ou (se não existir um struct do ID dado) -1.

 

Sintaxe:

struct_names_count(struct_id);

Argumento Tipo Descrição
struct_id Struct The unique ID value of the struct to check.

 

Devoluções:

Real

 

Exemplo:

var _num = struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

O código acima recuperará o número de variáveis no dado struct e mostrará uma mensagem de debug na saída do console com este valor.
