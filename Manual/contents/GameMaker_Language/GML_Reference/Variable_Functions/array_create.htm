array_create

Com esta função você pode criar uma matriz 1D de um determinado tamanho. Você informa à função o comprimento da matriz a ser criada, e ela retorna o "cabo" para a matriz que você pode então atribuir a uma variável. Arrays criados desta forma terão cada entrada inicializada em 0, a menos que você especifique um valor de inicialização (opcional). Se você fornecer o valor extra para inicializar o array, então todos os índices dentro do novo array serão definidos para isso ao invés de 0, mas note que a função terá uma maior sobrecarga de desempenho neste caso.

Sintaxe:

array_create(size, [value]);

Argumento Tipo Descrição
size Real The size of the array to create.
value Variável OPTIONAL O valor a ser usado para rubricar todos os índices de matriz.

Devoluções:

Array

Exemplo:

instance_array = array_create(100, noone);

O código acima criará uma nova matriz de 100 entradas, cada uma definida com a palavra-chave noone, e então a atribuirá à variável "instance_array".
