variável_struct_remove

Com esta função você pode remover uma variável de um struct. Você fornece o ID struct para remover a variável e o nome string da variável a ser removida.

Sintaxe:

variable_struct_remove(struct, name);

Argumento Tipo Descrição
struct Estrutura The struct reference to remove the variable from
name Cordão The name of the variable to remove (as a string)

Devoluções:

N/A

Exemplo:

if variable_struct_exists(mystruct, "shields")
{
    variable_struct_remove(mystruct, "shields");
}

O código acima verificará se a variável dada existe no dado struct e se ela existe, então ela é removida.
