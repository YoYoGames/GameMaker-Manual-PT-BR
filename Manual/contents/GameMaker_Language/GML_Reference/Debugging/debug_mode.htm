debug_mode

Esta variável lida retorna apenas true quando o jogo está sendo jogado no modo de depuração e false quando está sendo jogado normalmente.

Sintaxe:

debug_mode

Devoluções:

Exemplo:

if debug_mode
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

O código acima abre um arquivo ini diferente, dependendo do valor da variável somente leitura debug_mode.
