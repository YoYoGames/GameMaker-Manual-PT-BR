ini_open_from_string

Esta função criará um arquivo ini temporário a partir de um string e o abrirá para leitura/escrita. O string deve ser formatado corretamente como um arquivo ini (ou seja: com seções, chaves de quebra de linha e valores), caso contrário, o arquivo ini não será criado corretamente. Note que este arquivo ini é temporário e será removido da memória no momento em que for fechado, perdendo qualquer informação que foi armazenada nele, entretanto o ini_close() A função retorna um string do arquivo ini completo que pode então ser salvo em um server ou em disco.

Sintaxe:

ini_open_from_string(string);

ArgumentoTipo Descrição
string The string containing all the ini information.

Devoluções:

N/A

Exemplo:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

O código acima estabelece um ini temporário para guardar as informações do "str" string, depois lê a partir do ini antes de fechá-lo novamente.
