ini_close

Esta função deve ser chamada no momento em que você terminar de ler ou escrever para qualquer arquivo ini aberto. Se você não usar a função após ter usado qualquer uma das funções de gravação ini, então nada será gravado em disco, pois as informações do arquivo são mantidas na memória até que esta função seja chamada, o que força a gravação. Se você tentar abrir um ini sem ter previamente fechado outro (ou o mesmo), você também receberá um erro.

A função também retornará um string com o arquivo ini codificado nele. Este string pode então ser salvo em um server e/ou usado novamente junto com a função ini_open_from_string() para recriar o ini.

Sintaxe:

ini_close();

Devoluções:

String

Exemplo:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Isto abrirá "savedata.ini" e lerá um valor da seção "save1" com a chave "score", depois fechará o arquivo ini. Se não houver nenhum valor em "save1" chamado "score" ou se não houver um arquivo "savedata.ini" presente, 0 será retornado e um novo arquivo ini será criado.
