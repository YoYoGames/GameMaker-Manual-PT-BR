file_text_open_append

Esta função abre o arquivo de texto com o nome de arquivo indicado para escrita (se o arquivo não existir, ele é criado), devolvendo a identificação única do arquivo que deve ser armazenado em uma variável, pois ele será usado para todas as outras ações relacionadas a esse arquivo. A posição dentro do arquivo para escrita é definida para a última linha de texto que o arquivo contém, de modo que quaisquer dados existentes não sejam sobrescritos.

Observe que se o arquivo não puder ser criado (por causa de um nome de arquivo ilegal, por exemplo), a função retornará -1.

OBSERVAÇÃO: Você só pode ter um máximo de 32 arquivos abertos a qualquer momento. Você também deve sempre fechar os arquivos quando terminar, pois isto escreve as informações e libera a memória associada ao arquivo.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

file_text_open_append(fname);

Argumento Tipo Descrição
fname Cordão The name of the file to append to.

Devoluções:

Text File ID or -1

Exemplo:

file = file_text_open_append(working_directory + "save.txt");

Isto abriria "save.txt" do mesmo diretório do jogo e armazenaria a identificação do arquivo na variável "file".
