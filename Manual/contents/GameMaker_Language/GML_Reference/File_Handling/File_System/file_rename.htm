file_rename

Esta função irá renomear o arquivo especificado com o nome especificado. A função retornará true se o arquivo tiver sido renomeado com sucesso, ou false em qualquer outra circunstância.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre o Sistema de Arquivos para maiores informações.

Sintaxe:

file_rename(oldname, newname);

ArgumentoTipo Descrição
oldname The name of the file to change.
newname The new name to give the file.

Devoluções:

Exemplo:

if file_exists("level1.txt")
{
    file_rename("level1.txt", "level.txt");
}

Isso verificaria a existência de um arquivo e, se ele existir, será renomeado.
