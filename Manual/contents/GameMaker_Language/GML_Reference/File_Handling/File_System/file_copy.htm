file_copy

Esta função irá copiar o arquivo especificado, dando-lhe o novo nome que você escolher.

Se você estiver copiando um arquivo em Android de dentro de sua área de Arquivos Incluídos (que estará dentro de seu APK), ele copiará primeiro para a memória, portanto, esteja ciente de que copiar arquivos muito grandes do APK pode fazer com que os dispositivos fiquem sem memória. No alvo HTML5, usamos o HTML5 Local Storage API para salvar do lado do cliente, e as funções não suportam operações de arquivos binários, portanto, suponha que você esteja copiando um arquivo de texto.

A função falhará se o arquivo de novos nomes já existir.

ADVERTÊNCIA! Esta função pode não funcionar como você espera devido ao GameMaker ser lixado! Consulte a seção sobre Limites do Sistema de Arquivos para maiores informações.

Sintaxe:

file_copy(fname, newname);

ArgumentoTipo Descrição
fname The name of the file to copy.
newname The name of the new, copied file.

Devoluções:

N/A

Exemplo:

if file_exists("level1.txt")
{
    file_copy("level1.txt", "level2.txt");
}

Isto verificaria a existência de um arquivo e, se ele existe, é copiado.
