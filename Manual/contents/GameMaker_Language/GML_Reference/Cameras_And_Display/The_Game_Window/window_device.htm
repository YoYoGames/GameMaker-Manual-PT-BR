janela_dispositivo DEPRECADO

Esta função irá retornar o ponteiro do dispositivo d3d atual, que você pode então (por exemplo) passar para uma DLL ou Dylib em Windows e macOS.

NOTA Esta função foi depreciada no GameMaker em favor de os_get_info().

Sintaxe:

window_device();

Devoluções:

Pointer

Exemplo:

gfx_pointer = window_device();

O código acima irá armazenar o ponteiro do dispositivo d3d em uma variável.
