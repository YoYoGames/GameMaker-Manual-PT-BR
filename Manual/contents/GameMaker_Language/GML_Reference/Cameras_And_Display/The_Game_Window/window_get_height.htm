altura_da_janela

Com esta função você pode obter a altura atual (em pixels) da janela do jogo.

Sintaxe:

window_get_height();

Devoluções:

Exemplo:

hh = window_get_height();

O código acima irá armazenar a altura atual da janela na variável "hh".
