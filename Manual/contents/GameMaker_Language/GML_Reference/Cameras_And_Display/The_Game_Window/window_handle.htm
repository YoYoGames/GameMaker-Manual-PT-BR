manípulo_da_janela

Com esta função você pode obter o valor de id interno Windows (o HWND, um ponteiro). Esta função é realmente útil apenas para os escritores extension que precisam do window handle para chamar Windows API's em código DLL (o ponteiro retornado deve ser lançado em um string e, em seguida, no C++ apenas lançá-lo em um HWND). A tabela abaixo mostra as plataformas suportadas juntamente com o que elas retornam:

Plataforma Devoluções
Windows Janela HWND
macOS Classe NSWindow
Ubuntu (Linux) Punho XWindow
HTML5 Identificação de lona

Sintaxe:

window_handle();

Devoluções:

Pointer

Exemplo:

win_id = windows_handle();

O código acima armazenará o id da janela do jogo em uma variável.
