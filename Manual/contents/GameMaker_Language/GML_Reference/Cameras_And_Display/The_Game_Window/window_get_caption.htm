window_get_caption

Esta função retorna a legenda da janela (este é o texto que aparece no topo da janela, ao lado de seu ícone) e, por padrão, mostra a legenda do room em que você está atualmente.

Sintaxe:

window_get_caption();

Devoluções:

Exemplo:

if window_get_caption() != ""
{
    window_set_caption("");
}

O código acima verificará a legenda da janela para ver se ela tem texto ou não, e se a coloca em um string vazio para que nenhuma legenda seja exibida.
