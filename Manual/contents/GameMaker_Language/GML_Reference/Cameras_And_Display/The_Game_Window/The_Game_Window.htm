A Janela de Jogo

O jogo que você cria acontece em uma janela (mesmo quando em tela cheia), e esta janela tem uma série de propriedades, como posição, tamanho, se é em tela cheia, etc... Estes detalhes são normalmente definidos automaticamente para você com base no tamanho room e portas de visualização habilitadas em combinação com as configurações das Opções de Jogo para a plataforma alvo, mas você pode alterá-las durante o jogo usando as funções listadas nesta página.

NOTA Estas funções são apenas para Windows, Ubuntu (Linux), macOS e módulos alvo HTML5 e podem não funcionar em nenhum outro dispositivo.

Note que a imagem acima mostra a janela do jogo desenhada em um navegador, mas para os outros alvos, você pode substituir o navegador pelo visor, então uma função como window_get_y() retornará a posição do topo da janela do jogo em relação ao visor.
