janela_set_fullscreen

Com esta função você pode configurar a janela do jogo para ser tela cheia (true) ou não (false). Observe que para o alvo macOS, você deve ter desmarcado a opção "Start In Fullscreen" e marcado a opção "Allow the player to resize the game window" nas Opções de Jogo, caso contrário, esta função falhará. Observe também que esta função não funcionará em HTML5 a menos que seja adicionada como "clickable" (veja aqui para mais detalhes).

Sintaxe:

window_set_fullscreen(full);

ArgumentoTipo Descrição
full Whether to set the screen to fullscreen (true) or not (false).

Devoluções:

Exemplo:

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

O código acima verifica se há um botão do mouse pressionado e depois ajusta a janela para tela cheia se ainda não estiver, ou ajusta para janela se já estiver.
