window_set_caption

Com esta função você pode alterar ou definir a legenda do windows para o room em que você está atualmente. Esta legenda aparece na parte superior da janela, ao lado do ícone do jogo, quando o jogo não está no modo tela cheia.

Sintaxe:

window_set_caption(caption);

ArgumentoTipo Descrição
caption The new caption.

Devoluções:

Exemplo:

if window_get_caption() != ""
{
    window_set_caption("");
}

O código acima verificará a legenda da janela para ver se ela tem texto ou não, e se a coloca em um string vazio para que nenhuma legenda seja exibida.
