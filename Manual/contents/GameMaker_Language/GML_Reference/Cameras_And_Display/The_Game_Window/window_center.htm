window_center

Com esta função você pode centralizar a janela do jogo na tela quando o módulo alvo for Windows, Ubuntu (Linux) ou macOS, ou você pode centralizá-la no navegador se o módulo alvo for HTML5. Esta função não tem efeito em nenhum outro dispositivo.

NOTA Se você redimensionou a janela do jogo na etapa atual (por exemplo, mudando de tela cheia para janela ou usando window_set_size()), e deseja centralizar a nova janela, esta função deve ser chamada pelo menos uma etapa mais tarde (por exemplo, usando um alarm) caso contrário, não funcionará corretamente.

Sintaxe:

window_center();

Devoluções:

Exemplo:

if !window_get_fullscreen()
{
    window_center();
}

O código acima centralizará a janela do jogo dentro da tela se ela não estiver no modo tela cheia.
