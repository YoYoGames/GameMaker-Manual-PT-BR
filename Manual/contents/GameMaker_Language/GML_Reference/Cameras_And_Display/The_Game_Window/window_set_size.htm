tamanho_da_janela

Com esta função, você pode mudar o tamanho da janela do jogo. Em plataformas desktop, isto geralmente se refere à área do cliente da janela, excluindo a barra de título e as bordas:

Se você usar esta função para mudar o tamanho da janela do jogo, o conteúdo da janela será escalado para caber com uma proporção de 1:1 para a tela de fundo, o que manterá a imagem nítida e clara (embora escalada, se a janela for diferente do room ou do tamanho da vista). Por exemplo, se você tem uma janela de 200 x 150 pixels, e o jogador estica a janela arrastando as bordas, o tamanho final da janela pode ser de 400 x 300 pixels, mas a tela de desenho é STILL 200 x 150, o que dá uma imagem desfocada, pois a proporção de pixels agora é de 1:2. Entretanto, se você usar esta função para mudar o tamanho da janela, a proporção 1:1 é mantida, mesmo que a imagem seja escalonada para se ajustar ao novo tamanho da janela.

Sintaxe:

window_set_size( w, h );

ArgumentoTipo Descrição
w The new width of the window.
h The new height of the window.

Devoluções:

N/A

Exemplo:

if mouse_check_button_pressed(mb_left)
{
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
}

O código acima mudará o tamanho da janela com base no valor da variável "global.scale" quando o botão do mouse for pressionado.
