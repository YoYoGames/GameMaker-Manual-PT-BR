camera_set_view_target

Você pode usar esta função para definir a seguinte meta da câmera de visão dentro do room. Você dá o valor único de identificação da câmera (como retornado pelos diferentes camera_create() funções) e então dê a instância alvo ou object ID para a qual você deseja definir a visão da câmera. Note que se você escolher um ID object e houver mais de uma instância desse object no room, não há como o GameMaker saber qual instância você deseja seguir e, portanto, pode ser qualquer uma delas.

Sintaxe:

camera_set_view_target(camera_id, instance_id/object_id)

Argumento Tipo Descrição
camera_id Identificação da câmera The unique camera ID value returned when you created the camera
instance_id/object_id Objeto Ativo ou ID da instância Instance or object to have the camera target for following

Devoluções:

N/A

Exemplo:

camera_set_view_target(view_camera[0], id);

O código acima definirá o alvo da câmera de visualização para a câmera designada para visualizar a porta[0] para a da instância que estiver executando o código.
