camera_get_view_target

Esta função pode ser usada para recuperar o seguinte alvo da câmera em questão. O valor de retorno pode ser um índice object, um ID de instância ou -1 se nenhum alvo seguinte tiver sido definido.

Sintaxe:

camera_get_view_target(camera_id)

ArgumentoTipo Descrição
camera_id The unique camera ID value returned when you created the camera

Devoluções:

Object Asset, Instance ID or -1

Exemplo:

var target = camera_get_view_target(view_camera[0]);
if (target != obj_Player)
{
    camera_set_view_target(view_camera[0], obj_Player);
}

O código acima recupera o seguinte alvo da câmera designada para visualizar a porta[0] e então verifica se ela corresponde ao índice object "obj_Player". Caso não corresponda, a câmera de visualização é configurada para seguir uma instância do object.
