camera_set_update_script

Esta função pode ser usada para definir uma função de script que será chamada de cada frame de jogo que a câmera for atribuída a uma porta de visualização visível e ativa. Você dá o valor único de identificação da câmera (como retornado pelas diferentes funções camera_create ) e a identificação da função a ser chamada. A ordem na qual as funções anexadas às câmeras e a renderização real da visão da câmera é a seguinte:

  • As câmeras para todas as portas de visualização visíveis e ativas têm sua função de atualização chamada
  • Depois, para cada vista:
    • A função de início da câmera para essa visão é chamada
    • Os eventos do sorteio são executados para essa visão
    • A função final para a câmera é chamada
  • Passar para a próxima visão visível ativa e repetir...

A aplicação de uma função de atualização personalizada a uma câmera anula seu comportamento padrão de atualização. Por exemplo, se você configurar sua câmera para seguir um object (digamos, obj_player) ao ligar para camera_create_view(), ou configurá-la nas Propriedades Room, ela deixará de seguir esse object uma vez que uma atualização personalizada script for definida.

Sintaxe:

camera_set_update_script(camera_id, script)

Argumento Tipo Descrição
camera_id Identificação da câmera The unique camera ID value returned when you created the camera.
script Função Script The script function to run each game frame

Devoluções:

N/A

Exemplo:

camera_set_update_script(view_camera[0], updateCamera);

O código acima define a função de atualização para a câmera designada para visualizar a porta [0].
