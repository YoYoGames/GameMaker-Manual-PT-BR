textura_de_sorteio_primitivo_begin_textura

Esta função deve ser chamada antes de se definir os vértices de um primitivo texturizado. Você deve dar o tipo de primitivo a ser usado (ver draw_primitive_begin() para mais informações) e o id de um texture a ser usado, que pode ser um sprite ou uma imagem de fundo asset. Este id asset pode ser obtido a partir das funções sprite_get_texture()por exemplo (usar -1 para não texture).

NOTA Para que um texture repita deve ser um poder de dois em tamanho, ou seja: 32x32, 128x128, etc...

Sintaxe:

draw_primitive_begin_texture(kind, tex)

Argumento Tipo Descrição
kind Constante do tipo primitivo The kind of primitive you are going to draw.
tex Ponteiro de Textura The texture to use with the primitive.

Devoluções:

N/A

Exemplo:

draw_set_colour(c_white);
var _tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, _tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

O código acima desenhará uma faixa triangular 4 vertex (fazendo um retângulo) texturizada com o texture mantido na variável "tex", e o conjunto texture será usado para cobrir o primitivo completo.
