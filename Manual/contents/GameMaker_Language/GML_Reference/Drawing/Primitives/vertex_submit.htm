vertex_submit

Você pode usar esta função para submeter o conteúdo de um vertex buffer ao canal gráfico para uso com um shader. Você fornece o índice buffer para usar, o tipo primitivo básico para usar (veja as constantes abaixo) e o texture que deve ser usado. O tipo primitivo de base é usado apenas para atribuir a ordem na qual os vértices que você armazenou no buffer são desenhados e conectados, mas os dados reais usados para cada um dos vértices serão os que você definiu ao criar o vertex buffer.

NOTA: Esta função só pode ser usada no Evento de Sorteio.

Para um exemplo visual das diferentes bases primitivas disponíveis, veja a imagem abaixo:

Constante Descrição
pr_pointlist Um primitivo que consiste em uma lista de pontos.
pr_linelist Um primitivo composto de uma linha individual em uma lista.
pr_linestrip Um primitivo composto de uma faixa consecutiva de linhas.
pr_trianglelist Um primitivo composto de triângulos individuais em uma lista.
pr_trianglestrip Um primitivo composto de uma faixa consecutiva de triângulos.

Sintaxe:

vertex_submit(buffer, primitive, texture);

Argumento Tipo Descrição
buffer Vertex Buffer ID The buffer to use.
primitive Constante do tipo primitivo The primitive base type.
texture Ponteiro de Textura The texture to use (or -1 for no texture).

Devoluções:

N/A

Exemplo:

shader_set(shader_prim);
vertex_submit(buff, pr_trianglelist, sprite_get_texture(sprite_index));
shader_reset();

O código acima apresentará o vertex buffer indexado na variável "buff" para desenho com o shader "shader_prim", usando uma lista triangular como base primitiva e o texture do sprite para a instância executando o código.
