draw_primitive_begin

Esta função deve ser chamada antes que você possa definir quaisquer primitivos. Existem 6 tipos de primitivos que você pode definir como as seguintes constantes:

Constante do tipo primitivo
Constante Descrição
pr_pointlist Uma lista de pontos
pr_linelist Uma lista de linhas
pr_linestrip Uma faixa de linha
pr_trianglelist Uma lista triangular
pr_trianglestrip Uma faixa triangular
pr_trianglefan Um ventilador triangular

A imagem a seguir ilustra basicamente como estes devem ser e também a ordem na qual você deve definir os vértices:

Observe que em algumas plataformas (Windows, UWP, XBox) o tipo pr_trianglefan não é suportado nativamente e por isso o GameMaker faz uma conversão quando o jogo é compilado para fazê-los funcionar. Isto significa que nessas plataformas o tipo pr_trianglefan será muito mais lento de usar do que as outras. Isto Também note que para usar esta função em HTML5, você deve habilitar o WebGL nas Opções do Jogo.

Sintaxe:

draw_primitive_begin(kind)

Argumento Tipo Descrição
kind Constante do tipo primitivo The kind of primitive you are going to draw.

Devoluções:

N/A

Exemplo:

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
    draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

O código acima desenhará três quartos de um círculo feito de primitivos.
