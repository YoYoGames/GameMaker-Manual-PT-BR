draw_sprite_pos

Com esta função você pode desenhar um sprite distorcido sobre a área definida pelas quatro coordenadas de canto. Os dois primeiros argumentos são o sprite a desenhar e a subimagem do sprite (o mesmo que na função draw_sprite()) mas os próximos são aqueles que definem a posição de cada um dos quatro cantos da caixa delimitadora do dado sprite. Estes devem ser dados em ordem horária, portanto a primeira coordenada é a superior esquerda, depois a superior direita, depois a inferior. direita e finalmente a parte inferior esquerda. Você também pode definir um valor para o alfa do sprite para desenhá-lo com transparência. A imagem abaixo ilustra como esta função funciona:

ADVERTÊNCIA! A imagem acima é apenas para fins ilustrativos, e se você usar esta função em um sprite, você obterá resultados diferentes e poderá experimentar texture "tosquia" devido à forma como um sprite é construído a partir de um quadrante de primitivos.

NOTA: Esta função só é útil para o bitmap sprites e não funcionará com SWF ou JSON (Spine) sprites.

NOTA: Ao utilizar esta função, você deve ter o ajuste de Cultura Automática desativado para as páginas texture.

Sintaxe:

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

ArgumentoTipo Descrição
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x1 The first x coordinate.
y1 The first y coordinate.
x2 The second x coordinate.
y2 The second y coordinate.
x3 The third x coordinate.
y3 The third y coordinate.
x4 The fourth x coordinate.
y4 The fourth y coordinate.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

Devoluções:

Exemplo:

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

O código acima desenhará o sprite associado com a instância executando o código distorcido em torno da posição x/y da instância e com um alfa totalmente opaco.
