part_emitter_destroy

Esta função removerá o emissor especificado do sistema dado e o apagará da memória (isto também impedirá que quaisquer partículas sejam produzidas pelo emissor dado, mas NÃO as remove do room). Esta função deve ser sempre chamada quando o emissor dado não for mais necessário para que o sistema evite vazamentos de memória e erros.

Sintaxe:

part_emitter_destroy( ps, ind );

Argumento Tipo Descrição
ps Identificação do Sistema de Partículas The particle system to destroy the emitter from.
ind Identificação do Emissor de Partículas The index of the emitter to destroy.

Devoluções:

N/A

Exemplo:

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit);
}

O código acima verificará se o emissor de partículas indexado na variável "p_emit" existe no sistema de dar partículas e se ele o faz é destruído.
