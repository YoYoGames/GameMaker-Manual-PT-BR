part_emitter_destroy_all

Esta função removerá todos os emissores definidos do sistema dado e os limpará da memória (isto também impedirá que quaisquer partículas sejam produzidas pelo emissor dado, mas NÃO os remove do room). Esta função deve ser sempre chamada quando os emissores não forem mais necessários para que o sistema evite vazamentos de memória e erros.

Sintaxe:

part_emitter_destroy_all( ps );

ArgumentoTipo Descrição
ps The particle system to destroy all emitters from.

Devoluções:

Exemplo:

if lives == 0
{
    part_emitter_destroy_all(global.Sname);
    room_goto(rm_Menu);
}

O código acima verifica as "vidas" da variável global construída e, se for 0, destrói todos os emissores de partículas e depois muda room.
