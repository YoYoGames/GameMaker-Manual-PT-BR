part_type_colour2

Esta função pode ser usada para definir um gradiente de duas cores para cada partícula criada do tipo dado. A primeira cor é aquela com a qual todas as partículas começarão, e a segunda cor é aquela em que a partícula terminará, e uma mudança suave de gradiente ocorrerá na cor ao longo da vida útil das partículas de uma cor para a outra.

Sintaxe:

part_type_colour2(ind, colour1, colour2);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
colour1 Cor The colour the particle will start at.
colour2 Cor The colour the particle will end at.

Devoluções:

N/A

Exemplo:

part_type_colour2( part_Health, c_red, c_white);

O código acima fará com que todas as partículas criadas do tipo indexado na variável "part_Health" tenham uma mistura de cores do vermelho ao branco ao longo de sua vida útil.
