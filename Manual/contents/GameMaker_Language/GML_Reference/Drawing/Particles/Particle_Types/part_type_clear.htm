part_type_clear

Com esta função você pode "zerar" uma partícula, retornando todos os valores para cada uma das funções relacionadas à partícula (vida, cor, alfa, orientação, etc...) aos seus valores padrão. Note que esta função não remove da tela nenhuma partícula atualmente visível no room, para isso você deve estar usando part_particles_clear().

Sintaxe:

part_type_clear(ind);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to clear.

Devoluções:

N/A

Exemplo:

part_type_clear(global.explode_part);

O código acima limpará o tipo de partícula indexada na variável global "explode_part" a seus valores padrão.
