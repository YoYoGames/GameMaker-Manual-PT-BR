part_type_speed

Esta função é usada para definir a velocidade de movimento da partícula quando ela é criada e também pode ser usada para fazer a partícula aumentar ou diminuir sua velocidade ao longo de sua vida útil. Os valores mínimo e máximo de velocidade padrão em 1, o que significa que a partícula se moverá 1 pixel a cada passo, e se forem definidos para o mesmo valor, a velocidade inicial para todas as partículas será a mesma, mas se forem definidos para valores diferentes, cada partícula começará com uma velocidade aleatória de dentro da faixa determinada. Por exemplo, se você tiver um mínimo de 3 e um máximo de 6, todas as partículas serão criadas com uma velocidade entre 3 e 6 pixels por etapa.

Você também pode definir um valor de incremento que fará a partícula se mover mais rapidamente (se um número positivo) ou mais lentamente (se um número negativo) ao longo de sua vida útil, com a quantidade dada sendo adicionada à velocidade da partícula a cada passo. Este valor pode ser um mínimo de (+/-)0,01.

Finalmente, você pode definir o fator "agitar". Este é um valor que será adicionado ou subtraído aleatoriamente à velocidade de cada etapa da vida útil das partículas. Obviamente, valores maiores são mais pronunciados do que os menores, e este valor pode até ser negativo, sendo a faixa máxima entre -20 e 20.

Sintaxe:

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
speed_min Real The minimum speed the particle can start at.
speed_max Real The maximum speed the particle can start at.
speed_incr Real How much the particle speed should increase or decrease per step.
speed_wiggle Real How much should randomly be added or subtracted from the particle's speed per step.

Devoluções:

N/A

Exemplo:

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

O código acima estabelecerá vários valores de partículas, incluindo a velocidade que estabelece uma velocidade mínima de cinco e uma velocidade máxima de dez, com a velocidade diminuindo por um valor de -0,2 cada passo. A velocidade também se agitará por um valor aleatório entre 0 e 20 cada passo.
