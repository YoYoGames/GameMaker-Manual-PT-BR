part_type_colour1

Esta função é usada para definir um tipo de partícula para ser uma única cor durante a duração total da vida útil de cada partícula individual.

Sintaxe:

part_type_colour1(ind, colour1);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
colour1 Cor The single colour to make the particle type.

Devoluções:

N/A

Exemplo:

part_type_colour1(global.Snow_Part, c_white);

O código acima definirá todas as partículas criadas do tipo de partícula indexada na variável global "Snow_Part" como sendo apenas brancas.
