part_type_colour_hsv

Com esta função você pode definir uma tonalidade, saturação e faixa de valores para todas as partículas do tipo dado. Você fornece um valor mínimo e um valor máximo para cada um dos três componentes e as partículas criadas terão uma cor aleatória com base na faixa de parâmetros dada. Desta forma, você pode criar partículas da mesma tonalidade, mas com diferentes saturações, ou de tonalidades diferentes, mas com o mesmo valor (luminosidade), etc... Todos os valores devem estar entre 0 e 255.

Sintaxe:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

Argumento Tipo Descrição
ind Identificação do tipo de partícula The index of the particle type to change.
hmin Real The minimum the final colour's hue component can be.
hmax Real The maximum the final colour's hue component can be.
smin Real The minimum the final colour's saturation component can be.
smax Real The maximum the final colour's saturation component can be.
vmin Real The minimum the final colour's value component can be.
vmax Real The maximum the final colour's value component can be.

Devoluções:

N/A

Exemplo:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

O código acima estabelece que cada partícula emitida do tipo de partícula indexada na variável global "Estrelas" tenha cores e saturações diferentes, mas o mesmo valor (luminosidade).
