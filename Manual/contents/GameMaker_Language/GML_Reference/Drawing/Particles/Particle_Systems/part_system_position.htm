posição_do_sistema_parte

Com esta função você pode definir a posição base do sistema de partículas em relação à posição (0,0) do room, o que significa que todas as outras funções de partículas relacionadas a este sistema serão agora desenhadas em relação à nova posição. Por padrão, esta posição é sempre (0,0), mas em algumas circunstâncias muito especiais você pode querer mudar isto para outra coisa.

NOTA: Esta função mudará tudo dentro do sistema de partículas, portanto se você tiver um emissor na posição (100.100) e depois ajustar a posição do sistema de partículas para (0.100), o emissor irá agora puxar para (100.200). O mesmo vale se você mudar o sistema e depois criar o emissor, como se você o criasse em (100.100), ele será puxado em (100.200).

Sintaxe:

part_system_position(ind, x, y);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to change.
x Real The new x coordinate of the particle system.
y Real The new y coordinate of the particle system.

Devoluções:

N/A

Exemplo:

if mouse_check_button_pressed(mb_left)
{
    part_system_position(global.Sname, mouse_x, mouse_y);
}

O código acima verificará o apertar do botão do mouse e se detectar um, o sistema de partículas indexadas na variável global "Sname" é deslocado para a posição x/y do mouse
