part_system_update

Esta função fará o sistema de partículas avançar um passo no tempo de jogo e pode ser usada para simular o comportamento do sistema de partículas quando a atualização automática estiver desligada, ou pode ser usada para avançar um sistema de partículas para um ponto específico, tudo em uma única etapa. Por exemplo, se você tiver um sistema de partículas para neve, e quiser que pareça que a neve está caindo logo no momento em que o jogador entra no room, então você pode desejar avançar suas partículas de neve por alguns segundos no evento de criação de uma instância para que, na primeira vez em que forem desenhadas, haja partículas em toda a tela (o exemplo de código abaixo mostra isto). Você também pode usar esta função quando a atualização automática do sistema é desligada pela função part_system_automatic_update() para levar o sistema adiante.

Sintaxe:

part_system_update(ind);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to update.

Devoluções:

N/A

Exemplo:

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

O código acima fará avançar o sistema de partículas indexadas na variável global "RainSys" em 3 segundos.
