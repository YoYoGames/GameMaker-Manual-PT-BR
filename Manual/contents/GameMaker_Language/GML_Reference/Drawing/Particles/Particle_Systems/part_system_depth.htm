part_system_depth

Com esta função você pode definir a profundidade de extração para o sistema de partículas, muito da mesma forma que você pode definir a profundidade de renderização de diferentes camadas dentro do room, onde uma baixa profundidade de extração significa que ela aparecerá em cima de todas as coisas desenhadas com uma profundidade maior, e uma alta profundidade de extração colocando-a abaixo de tudo com uma profundidade de extração menor.

Sintaxe:

part_system_depth( ind, depth );

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to change.
depth Real The depth at which to set the particle system.

Devoluções:

N/A

Exemplo:

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

O código acima criará um sistema de partículas e armazenará seu índice na variável global "Sname". este sistema recebe então uma baixa profundidade de -1000, o que significa que aparecerá acima de tudo com uma maior profundidade de extração.
