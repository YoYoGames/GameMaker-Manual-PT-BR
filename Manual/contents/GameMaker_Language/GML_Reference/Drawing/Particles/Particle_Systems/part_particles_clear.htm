part_particles_clear

Com esta função você pode limpar todas as partículas atualmente criadas pelo sistema a partir do room. Ele não reinicia ou remove os tipos de partículas em si, apenas sua representação visual, e se você tiver qualquer object partículas em streaming de um emissor, estas partículas desaparecem, mas começarão a aparecer novamente no próximo passo após chamar este código.

Sintaxe:

part_particles_clear(ind);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system.

Devoluções:

N/A

Exemplo:

if (lives <= 0)
{
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
}

O código acima verificará o valor da variável "vidas" e, se for igual a 0, limpa todas as partículas do sistema e depois muda room.
