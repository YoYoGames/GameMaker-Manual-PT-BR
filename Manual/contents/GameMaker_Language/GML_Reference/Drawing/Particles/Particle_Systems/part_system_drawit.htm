part_system_update

Com esta função você pode forçar um sistema de partículas a desenhar todas as partículas atuais na tela. Se part_system_automatic_draw() é desligado, então esta função mostrará as partículas quando usadas no evento de sorteio de uma instância, ou pode ser usada quando o alvo de sorteio é definido para uma superfície (usando surface_set_target()) para atrair as partículas do sistema para aquela superfície.

Sintaxe:

part_system_drawit(ind);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to draw the particles of.

Devoluções:

N/A

Exemplo:

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

O código acima verifica se a superfície indexada na variável "surf" existe, e se existe, então define o alvo de desenho para a superfície, desenha o sistema de partículas com o ID armazenado na variável global, e então redefine o alvo de desenho para que todo o desenho normal apareça na tela mais uma vez.
