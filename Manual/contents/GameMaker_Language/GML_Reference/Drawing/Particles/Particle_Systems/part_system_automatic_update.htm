part_system_automatic_update

Esta função controla se o GameMaker deve atualizar o sistema de partículas automaticamente ou não, e normalmente você não precisaria usar esta função, pois o valor padrão de true (atualização automática está ligada) é o que você deseja que aconteça. Entretanto, para efeitos especiais ou para pausar o jogo, você pode configurá-lo para false e isso impedirá que qualquer atualização seja realizada no sistema de partículas em questão, a menos que seja explicitamente comandada pelo uso da função part_system_update() ou você usa esta função novamente para ativar as atualizações automáticas. Nenhuma atualização significa que as partículas criadas neste sistema não mudarão mais de posição, cor ou qualquer outro parâmetro e que os emissores também deixarão de funcionar, "congelando" o sistema no ponto exato em que a atualização automática foi definida para false.

Uma coisa a ser observada é que se você estiver usando os efeitos simples criados pelas funções effect_create_above() ou effect_create_below() então você pode usar os valores 0 (para efeitos abaixo) ou 1 (para efeitos acima) como o índice do sistema de partículas e assim alternar a atualização automática para estes também (isto também funcionará para alternar a atualização para os efeitos de partículas visuais GML ).

Sintaxe:

part_system_automatic_update(ind, automatic);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to change.
automatic Booleano Whether automatic updating is on (true) ou não (falso).

Devoluções:

N/A

Exemplo:

if global.Pause
{
    part_system_automatic_update(global.Sname, false);
}

O código acima irá switch fora das atualizações de partículas se a variável global "Paused" for verdadeira.
