part_particles_create

Esta função é ideal para aqueles efeitos que não exigem nenhuma das funcionalidades oferecidas pelos emissores de partículas (por exemplo, para criar fumaça de um míssil, ou um simples efeito de explosão), pois permite criar partículas de forma rápida e fácil em qualquer posição no jogo room. Note que você deve ter criado previamente o sistema de partículas e o tipo de partícula para que esta função possa ser utilizada.

Sintaxe:

part_particles_create(ind, x, y, parttype, number);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Identificação do tipo de partícula The index (type) of the particles to be created.
number Real The number of particles to create.

Devoluções:

N/A

Exemplo:

if mouse_check_button(mb_left)
{
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

O código acima verifica se o botão do mouse está sendo pressionado e se ele retornar true ele gera 5 partículas na posição do mouse.
