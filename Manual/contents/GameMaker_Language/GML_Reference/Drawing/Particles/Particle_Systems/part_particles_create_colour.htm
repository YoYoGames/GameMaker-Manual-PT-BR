part_particles_create_colour

Esta função é ideal para aqueles efeitos que não requerem nenhuma das funcionalidades oferecidas pelos emissores de partículas (por exemplo, para criar fumaça de um míssil, ou um simples efeito de explosão), pois permite criar partículas de forma rápida e fácil em qualquer posição no jogo room. Você também pode colorir as partículas "on the fly", pois elas são criadas com esta função, e esta cor irá sobrepor-se à cor predefinida da partícula, mas não mistura esta cor ao longo da vida útil das partículas. Note que você deve ter criado previamente o sistema de partículas e o tipo de partícula para que esta função possa ser utilizada.

Sintaxe:

part_particles_create_colour(ind, x, y, parttype, colour, number);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system.
x Real The x coordinate of where to create the particles.
y Real The y coordinate of where to create the particles.
parttype Identificação do tipo de partícula The index (type) of the particles to be created.
colour Cor The colour blending for the particles.
number Real The number of particles to create.

Devoluções:

N/A

Exemplo:

if (speed > 1)
{
    var _c = choose(c_aqua, c_lime, c_fuchsia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

O código acima irá gerar uma única partícula a cada passo que a instância com o código tenha uma velocidade maior que uma. Estas partículas serão de uma cor aleatória.
