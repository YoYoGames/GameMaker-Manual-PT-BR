part_system_automatic_draw

Esta função pode ser usada para switch fora do desenho de um sistema de partículas para que quaisquer atualizações feitas no sistema (automáticas ou não) não sejam vistas. Esta é uma opção puramente visual e quando configurada para false você não poderá ver as partículas, pois elas não são desenhadas, mas ainda existem e estão mudando de posição, cor, etc... como normalmente aconteceria. Quando o desenho automático estiver desativado, você pode ordenar explicitamente ao GameMaker que desenhe o estado atual do sistema de partículas usando a função part_system_drawit()e se você definir esta função para true novamente, você pode switch desenhar automaticamente de volta.

Uma coisa a ser observada é que se você estiver usando os efeitos simples criados pelas funções effect_create_above() ou effect_create_below() então você pode usar os valores 0 (para efeitos abaixo) ou 1 (para efeitos acima) como o índice do sistema de partículas e assim alternar o desenho automático para estes também (isto também funcionará para alternar o desenho para os efeitos de partículas visuais GML ).

Sintaxe:

part_system_automatic_draw(ind, automatic);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to change.
automatic Booleano Whether automatic drawing is on (true) or not (false).

Devoluções:

N/A

Exemplo:

part_system_automatic_draw(global.Sname, false);

O código acima desativa o desenho automático para o sistema de partículas indexado na variável global "Sname".
