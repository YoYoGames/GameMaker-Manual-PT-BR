part_system_clear

Com esta função você pode limpar o sistema indexado para seu estado padrão, removendo todos os emissores e redefinindo a profundidade e a posição (se eles tivessem sido alterados). Tenha cuidado ao usar esta função como se você tivesse qualquer configuração ou estouro de instância ou qualquer outra ação usando um emissor associado a este sistema, você receberá um erro, a menos que você esteja usando o part_emitter_exists() função a ser verificada. Também não há necessidade de chamar a função part_emitter_destroy(), pois esta também é cuidada automaticamente.

NOTA: esta função limpará as partículas visíveis no room, mas não limpa as propriedades das partículas, nem as remove da memória. Para isso você deve utilizar as funções part_type_clear() e part_type_destroy().

Sintaxe:

part_system_clear(ind);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to clear.

Devoluções:

N/A

Exemplo:

part_system_clear(global.Sname);

O código acima limpará o sistema de partículas indexado na variável global "Sname" para seu estado padrão.
