part_system_existe

Com esta função você pode verificar se o dado sistema de partículas indexadas existe ou não no jogo. Note que se a variável a ser verificada for uma variável não-inicializada (que de outra forma um sistema de partículas teria seu índice atribuído), isto lançará um erro.

Sintaxe:

part_system_exists(ind);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to check for.

Devoluções:

Boolean

Exemplo:

if !part_system_exists(global.Sname)
{
    global.Sname = part_system_create();
}

O código acima verifica se o sistema de partícula referenciado na variável global existe e se não é criado.
