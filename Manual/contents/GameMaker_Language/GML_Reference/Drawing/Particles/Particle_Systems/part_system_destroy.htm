part_system_destroy

Com esta função você pode destruir um determinado sistema de partículas e removê-lo da memória. Isto deve sempre ser chamado quando o sistema não é mais necessário, como no final de um room, ou no evento de destruição de uma instância, caso contrário você poderia acabar com partículas que aparecerão mais tarde em rooms e nenhuma maneira de removê-las, bem como um vazamento de memória que eventualmente irá travar seu jogo.

NOTA: Esta função também destruirá e removerá quaisquer emissores que possam ter sido criados e associados ao sistema que está sendo destruído.

Sintaxe:

part_system_destroy(ind);

Argumento Tipo Descrição
ind Identificação do Sistema de Partículas The index of the particle system to destroy.

Devoluções:

N/A

Exemplo:

if (global.player_hp <= 0)
{
    part_system_destroy(p_sys);
    room_goto_next();
}

O código acima verifica se um valor global de variável é menor ou igual a zero, e se é então destrói o sistema de partículas referenciado na variável em questão e depois vai para o próximo room.
