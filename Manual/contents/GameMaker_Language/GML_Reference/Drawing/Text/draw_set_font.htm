draw_set_font

Esta função irá configurar o font para ser usado em todos os outros desenhos de texto. Este font deve ter sido adicionado aos recursos da fonte do jogo ou ter sido criado usando ou o font_add() ou font_add_sprite().

Sintaxe:

draw_set_font(font);

ArgumentoTipo Descrição
font The name of the font to use.

Devoluções:

Exemplo:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World");

O código acima desenhará o texto dado usando o font indexado na variável "fnt_Game" e azul colorido.
