gpu_set_colorwriteenable

Com esta função você pode switch dentro ou fora dos canais de cor e do canal alfa para todos os desenhos posteriores. Por exemplo, você pode usar esta função para criar somente superfícies alfa (switch fora da escrita RGB antes de usar a superfície), ou para criar efeitos especiais enquanto desenha na tela. O valor padrão para cada um dos componentes é true, e pode ser fornecido como quatro argumentos únicos ou como uma matriz de 4 elementos 1D com os seguintes elementos, que serão ou true (habilitado) ou false (desabilitado):

  • [0] = Canal vermelho habilitado/desabilitado
  • [1] = Canal verde habilitado/desabilitado
  • [2] = Canal azul habilitado/desabilitado
  • [3] = Canal Alfa habilitado/desabilitado

Sintaxe:

gpu_set_colorwriteenable(red, green, blue, alpha);

Argumento Tipo Descrição
red Booleano Enable/disable the red channel
green Booleano Enable/disable the green channel
blue Booleano Enable/disable the blue channel
alpha Booleano Enable/disable the alpha channel

Devoluções:

N/A

Exemplo:

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

O código acima obtém os valores atuais de escrita por cor e então define o componente alfa para false.
