gpu_set_texrepeat_ext

Esta função pode ser usada para definir se um único "slot" de amostragem repete o dado texture ao utilizar Shaders no GameMaker. Definindo-o para true repetirá o texture se as coordenadas uv estiverem fora do intervalo 0-1, enquanto que uma definição de false significará que não haverá repetição. O caso provável de uso para estas funções é para repetir um texture em 3D, mas para que ele funcione e não puxar imagens do resto da página texture, o sprite usado precisará ser marcado como estando em uma "Separate Texture Page" no Editor Sprite.

NOTA Esta configuração será sobrevalorizada pelo valor definido ao chamar a função gpu_set_texrepeat().

Sintaxe:

gpu_set_texrepeat_ext(sampler_id, enable);

ArgumentoTipo Descrição
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / false)

Devoluções:

Exemplo:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texrepeat_ext(s_tex))
{
    gpu_set_texrepeat_ext(s_tex, false);
}
else
{
    gpu_set_texrepeat_ext(s_tex, true);
}

O código acima verifica se a filtragem texture está ligada ou desligada para uma identificação específica do amostrador (armazenada em uma variável local) e a troca de acordo.
