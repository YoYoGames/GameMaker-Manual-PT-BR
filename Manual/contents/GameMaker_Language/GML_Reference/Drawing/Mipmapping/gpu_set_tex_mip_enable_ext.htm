gpu_set_tex_mip_enable_ext

Com esta função você pode definir se o mipmapping está desligado, ligado para tudo ou ligado somente para grupos texture selecionados no Gerente de Grupos de Textura em um amostrador shader. Você fornece o valor do índice para o amostrador shader (como retornado pela função shader_get_sampler_index()e, em seguida, uma das constantes listadas abaixo.

Constante Descrição
mip_off O Mipmapping está desativado.
mip_on O Mipmapping para todos textures está habilitado.
mip_markedonly O Mipmapping está habilitado para textures que o tem habilitado nas opções do grupo Texture (default).

Sintaxe:

gpu_set_tex_mip_enable_ext(sampler_index, setting);

Argumento Tipo Descrição
sampler_index Amostrador de amostras de Shader Handle The index of the shader sampler to get
setting Constante de Mipmapping The mipmap setting (a constant, default: mip_markedonly)

Devoluções:

N/A

Exemplo:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on)
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

O código acima permite o mipmapping para o sampler shader texture dado, caso ainda não tenha sido habilitado.
