draw_light_define_ambient

Esta função é usada para controlar a luz ambiente de uma cena, que é a luz que você tem em uma cena mesmo sem ter definido nenhuma fonte de luz pontual ou direcional. É efetivamente a cor geral e o brilho (ou escuridão) de uma cena. A cor padrão é c_black.

Sintaxe:

draw_light_define_ambient(col);

ArgumentoTipo Descrição
col The colour to use (either a constant, a real or a hex value).

Devoluções:

Exemplo:

draw_light_define_ambient(c_white);

O código acima definirá a iluminação ambiente como sendo branca.
