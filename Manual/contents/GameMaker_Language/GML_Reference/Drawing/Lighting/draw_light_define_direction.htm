draw_light_define_direction

Esta função é para definir uma luz direcional, onde a direção e a intensidade são definidas pelos valores de entrada para o vetor de unidade x, y e z (significando que devem estar entre -1 e 1, onde 1 ou -1 é intensidade total, e 0 não é intensidade). O As luzes também podem receber uma cor, o que também afetará a intensidade percebida da luz, uma vez que certas cores parecem "mais escuras" do que outras. Você também deve dar à luz um número índice que será usado em outras funções para referência isso.

NOTA: Há apenas 8 luzes de hardware disponíveis, portanto apenas 8 luzes definidas podem ser habilitadas a qualquer momento (embora mais possam ser definidas).

Sintaxe:

draw_light_define_direction(ind, x, y, z, col);

ArgumentoTipo Descrição
ind The index number of the light (arbitrary)
x The x component of the light vector
y The y component of the light vector
z The z component of the light vector
col The colour to use for the light (either a constant, a real or a hex value).

Devoluções:

Exemplo:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

O código acima permitirá a iluminação de toda a cena, então definirá uma luz direcional branca no espaço room, e então finalmente acenderá essa luz.
