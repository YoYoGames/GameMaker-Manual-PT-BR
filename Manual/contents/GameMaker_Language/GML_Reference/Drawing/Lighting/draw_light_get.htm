draw_light_get

Esta função obterá os parâmetros de luz especificados como uma matriz com os seguintes 6 elementos -

  • [0] = habilitado / desabilitado (true / false)
  • [1] = Tipo de luz Constante (ver tabela abaixo)
  • [2] = x posição
  • [3] = posição y
  • [4] = posição z
  • [5] = raio de luz (somente para luzes pontuais, será 0 para direcional)
  • [6] = cor clara (um real)

O tipo de luz (elemento [1]) pode ser uma das duas constantes a seguir:

Tipo de luz Constante
Constante Descrição
lighttype_dir A luz é uma luz direcional
lighttype_point A luz é uma luz pontual

Sintaxe:

draw_light_get(ind);

Argumento Tipo Descrição
ind Real The index number of the light (from 0 to 7)

Devoluções:

Array

Exemplo:

light_a = draw_light_get(1);
if (light_a[5] < 200)
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

O código acima obterá os valores usados para definir a luz indexada como "1", então verifique o raio e se for inferior a 200 será aumentado e o raio de luz será ajustado para o novo valor.
