place_empty

Você pode usar esta função para verificar e ver se a instância de chamada colidiria com qualquer outra instância de um object ou todas as instâncias em seu jogo. Para que a colisão seja resolvida corretamente, a instância que executa o código deve ter uma máscara de colisão válida (ou para o próprio sprite, ou através do mask_index) e só registrará colisões com aquelas instâncias que também possuem uma máscara válida.

A função é testar se não há colisões caso a instância chamadora seja colocada em uma posição específica, e você pode fornecer um argumento opcional para refinar a verificação para verificar apenas se uma posição está livre de colisões com instâncias do tipo dado. Observe que se não for fornecido um ID opcional object, a verificação será feita contra todas as instâncias dentro do room. A verificação de colisão será precisa ou baseada na caixa de delimitação da instância, dependendo do tipo de máscara de colisão selecionada, mas para que colisões precisas funcionem corretamente, todas as instâncias na verificação devem ter máscaras de colisão precisas.

Observe que as coordenadas x/y dadas serão niveladas com o inteiro mais próximo antes da verificação ser realizada.

Sintaxe:

place_empty(x, y, [object_id]);

Argumento Tipo Descrição
x Real The x position to check.
y Real The y position to check.
[object_id] Objeto Ativo OPTIONAL O objeto a ser verificado.

Devoluções:

Boolean

Exemplo:

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
    x = mouse_x;
    y = mouse_y;
}

O código acima verificará uma colisão com qualquer outra instância do object "obj_Enemy", como se a instância de chamada fosse colocada na mesma posição que o mouse. Se não for detectada nenhuma colisão, então a instância tem suas coordenadas x/y definidas para as do mouse.
