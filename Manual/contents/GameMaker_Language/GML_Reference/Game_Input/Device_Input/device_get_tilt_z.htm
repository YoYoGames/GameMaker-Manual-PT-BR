device_get_tilt_z

Esta função retorna um valor entre -1 e 1, dependendo do ângulo de "inclinação" do dispositivo. A correlação real entre graus de inclinação e o valor retornado depende do dispositivo e do SO que ele usa, mas geralmente um valor de 1 ou -1 é o mesmo que +/-90°. A imagem abaixo mostra como cada uma se relaciona com o dispositivo, se as funções disponíveis:

Sintaxe:

device_get_tilt_z()

Devoluções:

Exemplo:

y += sign(device_get_tilt_z());

O código acima usa o valor da inclinação para mover o jogador ao longo do eixo y.
