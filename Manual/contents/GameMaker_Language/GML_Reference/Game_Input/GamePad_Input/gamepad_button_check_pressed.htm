gamepad_button_check_pressed

Esta função retornará true ou false, dependendo se o botão gamepad dado for detectado como tendo sido pressionado ou não. Note que esta função só será acionada uma vez para o botão na primeira vez que ele for pressionado. Para que ela acione novamente o botão deve primeiro ser liberado e depois pressionado novamente. Se você precisar verificar um aperto contínuo do botão, você deve estar usando a função gamepad_button_check().

Se você estiver verificando um botão analógico, então a verificação levará em consideração a configuração do limite e só retornará true quando o valor de entrada bruta ultrapassar o limite determinado (você pode obter este valor bruto usando a função gamepad_button_value()).

Sintaxe:

gamepad_button_check_pressed(device, button);

Argumento Tipo Descrição
device Real Which gamepad device "slot" to check.
button Botão Gamepad Constant Which gamepad button constant to check for.

Devoluções:

Boolean

Exemplo:

if gamepad_button_check_pressed(0, gp_start)
{
    audio_play_sound(snd_Start, 0, false);
    room_goto(rm_Level_1);
}

O código acima detectará se o botão "start" do controle de jogo conectado ao dispositivo "slot" 0 foi pressionado ou não e mudará room se o tiver sido.
