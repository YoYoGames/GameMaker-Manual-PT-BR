gamepad_get_button_threshold

Esta função pode ser usada para detectar o ajuste do limiar atual dos botões analógicos para um determinado dispositivo. O limite padrão para todos os botões analógicos é 0,5, sendo a faixa de 0 a 1. O limite define em que ponto o botão é considerado como sendo "pressionado" para jogos que exigem que eles atuem como um botão digital.

Sintaxe:

gamepad_get_button_threshold(device);

ArgumentoTipo Descrição
device Which gamepad device "slot" to check.

Devoluções:

Exemplo:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

O código acima verifica o limite do botão analógico do controle de jogo conectado ao dispositivo "slot" 0 e se não for o valor padrão de 0,5, ele é definido para esse valor.
