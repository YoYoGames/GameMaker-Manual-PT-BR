gamepad_get_mapping

Esta função pode ser usada para recuperar o mapeamento string para o gamepad. Este string será um dos dois:

  • O mapa atual string para o gamepad, por exemplo:

    "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

  • A string com "nenhum mapeamento" se não houver um conjunto de mapeamento (isto não é o mesmo que quando não há mapeamento disponível)
  • Um string com "índice de dispositivo fora de alcance" se o índice de slot não for um índice de gamepad válido
  • Um vazio string "" se não houver um mapeamento disponível para o gamepad conectado

Para mais informações sobre a formatação do mapa retornado string, por favor veja a função gamepad_test_mapping().

Sintaxe:

gamepad_get_mapping(index);

ArgumentoTipo Descrição
index Which gamepad index "slot" to get the mapping from.

Devoluções:

Exemplo:

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

O código acima obtém o mapa string para o slot gamepad dado e, em seguida, envia-o para o console para debugging.
