gamepad_get_device_count

Esta função ou lhe dá o número de game pads conectados, ou o número de "slots" disponíveis para os game pads a serem conectados.

O valor real de retorno dependerá da plataforma e da configuração interna dessa plataforma e, como tal, esta função deve ser usada em conjunto com a função gamepad_is_connected() para garantir o número exato de almofadas conectadas a qualquer momento.

Sintaxe:

gamepad_get_device_count();

Devoluções:

Real

Exemplo:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if (gamepad_is_connected(i))
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

O código acima loops através dos blocos de jogo disponíveis (ou gamepad slots) e então verifica cada um deles para um gamepad conectado. o valor retornado é então usado para definir uma matriz global para true ou false para uso em verificações futuras.
