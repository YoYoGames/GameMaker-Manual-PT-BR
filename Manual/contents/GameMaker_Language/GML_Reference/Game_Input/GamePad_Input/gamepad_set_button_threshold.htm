gamepad_set_button_threshold

Esta função pode ser usada para definir o ajuste atual do limiar dos botões analógicos para um determinado dispositivo. O limite padrão para todos os botões analógicos é 0,5, sendo a faixa de 0 a 1.

O limiar define em que ponto o botão é considerado como sendo "pressionado" para jogos que exigem que eles atuem como um botão digital.

Esta função afetará os estados de verificação, pressionada e liberada para botões analógicos, mas não afetará o valor retornado pela função gamepad_button_value()que sempre retornará o valor bruto para o botão.

Sintaxe:

gamepad_set_button_threshold(device, threshold);

Argumento Tipo Descrição
device Real Which gamepad device "slot" to check.
threshold Real The new threshold value (from 0 - 1, default 0.5).

Devoluções:

N/A

Exemplo:

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

O código acima verifica o limite do botão analógico do controle de jogo conectado ao dispositivo "slot" 0 e se não for o valor padrão de 0,5, ele é definido para esse valor.
