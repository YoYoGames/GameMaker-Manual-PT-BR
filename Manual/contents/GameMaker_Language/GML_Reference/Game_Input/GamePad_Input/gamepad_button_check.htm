gamepad_button_check

Esta função retornará true ou false dependendo se o botão gamepad dado for detectado como sendo mantido pressionado ou não. Se você estiver verificando um botão analógico, então a verificação levará em consideração a configuração do limiar e retornará somente true enquanto o valor de entrada bruta estiver acima do limiar dado (você pode obter este valor bruto usando a função gamepad_button_value()).

Sintaxe:

gamepad_button_check(device, button);

Argumento Tipo Descrição
device Real Which gamepad device "slot" to check.
button Botão Gamepad Constant Which gamepad button constant to check for.

Devoluções:

Boolean

Exemplo:

if gamepad_button_check(0, gp_face1)
{
    if (canshoot = true)
    {
        audio_play_sound(snd_Shoot, 0, false);
        instance_create_layer(x, y, "Bullets", obj_Bullet)
        canshoot = false;
        alarm[0] = room_speed / 2;
    }
}

O código acima detectará se o botão inferior da face superior (o "X" em um controlador ps3) está sendo mantido para baixo e, em caso afirmativo, disparará uma "bala" e disparará um alarme.
