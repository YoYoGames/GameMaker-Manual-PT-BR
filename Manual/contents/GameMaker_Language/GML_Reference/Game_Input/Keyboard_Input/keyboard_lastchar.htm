teclado_lastchar

Esta variável armazena um string da última tecla pressionada. Esta variável não é somente lida e você pode modificá-la, por exemplo, para configurá-la para "" (um string vazio) se você já a manipulou.

Sintaxe:

keyboard_lastchar;

Devoluções:

Exemplo:

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = vk_nokey;
}

O código acima verifica se a última variável-chave é maior que -1, e se é ela adiciona qualquer que seja a última-chave como um string à variável "str".
