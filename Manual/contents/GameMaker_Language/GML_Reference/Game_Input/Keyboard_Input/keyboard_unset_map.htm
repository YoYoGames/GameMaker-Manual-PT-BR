keyboard_unset_map

Com esta função você pode limpar todas as chaves refazidas para que retornem ao seu estado padrão, ou seja: todas as chaves para mapear a si mesmas.

Sintaxe:

keyboard_get_map();

Devoluções:

N/A

Exemplo:

if keyboard_check_pressed(vk_escape)
{
    keyboard_unset_map();
}

O código do exemplo acima irá redefinir todas as chaves mapeadas para suas configurações padrão, se o usuário pressionar a tecla "escape".
