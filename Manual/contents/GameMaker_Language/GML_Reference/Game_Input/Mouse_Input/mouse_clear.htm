mouse_clear

Esta função limpará o estado atual de determinado botão do mouse. Isto significa que a verificação de que está sendo mantida pressionada não retornará true até que o jogador solte o botão e pressione novamente (mas o estado de liberação ainda será detectado se a limpeza for feita enquanto o botão do mouse estiver sendo mantido pressionado).

Você pode fornecer uma das constantes do botão do mouse listadas nesta página.

Sintaxe:

mouse_clear(button);

Argumento Tipo Descrição
button Botão do Rato Constante Which mouse button constant to check for.

Devoluções:

Boolean

Exemplo:

mouse_clear(mb_any);

O código acima limpará o estado para baixo de todos os botões do mouse.
