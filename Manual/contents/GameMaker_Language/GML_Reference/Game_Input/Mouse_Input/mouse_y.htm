mouse_y

Esta variável somente leitura retorna a posição atual do eixo y do mouse dentro do room.

Sintaxe:

mouse_y;

Devoluções:

Exemplo:

y = median(64, mouse_y, room_height - 64);

O código acima manterá a instância na posição y do mouse enquanto estiver dentro dos limites de 64 pixels da parte superior e inferior do room.
