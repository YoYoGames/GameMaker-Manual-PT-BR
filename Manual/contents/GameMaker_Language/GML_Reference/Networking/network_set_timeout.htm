network_set_timeout

Com esta função você pode definir o tempo limite para ler e escrever dados de/para um server através do soquete dado. Note que o timeout não gera nenhum tipo de evento, portanto você mesmo terá que lidar com os timeouts usando alarmes (por exemplo). Observe que este valor afeta apenas o envio e a recepção de dados, e caso você deseje alterar o valor de timeout da conexão, então você deve estar usando a função network_set_config().

Sintaxe:

network_set_timeout(socket, read_timeout, write_timeout);

ArgumentoTipo Descrição
socket The id of the socket to use.
read_timeout The milliseconds in which a transfer from a server will timeout.
write_timeout The milliseconds in which a transfer to a server will timeout.

Devoluções:

Exemplo:

network_set_timeout(sock, 3000, 3000);

O código acima definirá o tempo limite para leitura e escrita de dados através do soquete indexado na variável "meia" a 3 segundos.
