network_connect_raw

Com esta função você pode enviar um pedido para conectar a um server. A função leva a identificação do soquete para conectar (veja network_create_socket()) e exige que você forneça o endereço IP para se conectar (a string), bem como a porta para se conectar, e se a conexão falhar um valor inferior a 0 será devolvido. A diferença entre esta função e network_connect() é que esta função pode se conectar a qualquer server e não faz nada com os dados brutos, o que significa que você mesmo tem que implementar os protocolos no final do server. Note que por padrão a função é síncrona, o que significa que seu jogo pode parecer "pendurado" conforme a conexão é feita. Você pode definir um valor de timeout para a conexão, ou alternativamente tornar a criação assíncrona, usando a função network_set_config()ou, alternativamente, usar a função network_connect_async_raw() ao invés disso.

NOTA Esta função não pode ser usada em HTML5. Para WebSockets, use a função Async.

Sintaxe:

network_connect_raw(socket, url, port);

Argumento Tipo Descrição
socket ID da tomada de rede The id of the socket to use.
url Cordão The URL or IP to connect to (a string).
port Real The port to connect to.

Devoluções:

Real or Network Socket ID

Exemplo:

client = network_create_socket(network_socket_tcp);
network_connect_raw(client, "www.macsweeneygames.com", 6510);

O código acima criará um novo soquete TCP e então tentará criar uma conexão "bruta" através dele para a URL dada na porta 6510.
