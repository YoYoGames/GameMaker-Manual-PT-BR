network_connect

Com esta função você pode enviar um pedido para conectar a um server. A função leva a identificação do soquete para conectar (veja network_create_socket()) e exige que você forneça o endereço IP para se conectar (a string), bem como a porta para se conectar, e se a conexão falhar um valor inferior a 0 será devolvido. A conexão usa um protocolo especial que garante que somente os jogos GameMaker se conectem entre si, no entanto, se você precisar se conectar a um server que não seja um jogo GameMaker, você pode usar network_connect_raw(). Note que por padrão a função é síncrona, o que significa que seu jogo pode parecer "pendurado" à medida que a conexão é feita. Você pode definir um valor de timeout para conexão, ou alternativamente tornar a criação assíncrona, usando a função network_set_config(), ou alternativamente usar a função network_connect_async() em seu lugar.

NOTA Esta função não pode ser usada em HTML5. Para WebSockets, use a função Async.

Sintaxe:

network_connect(socket, url, port);

Argumento Tipo Descrição
socket ID da tomada de rede The id of the socket to use.
url Cordão The URL or IP to connect to (a string).
port Real The port to connect to.

Devoluções:

Real or Network Socket ID

Exemplo:

client = network_create_socket(network_socket_tcp);
network_connect(client, "192.134.0.1", 6510);

O código acima criará um novo soquete TCP e então tentará se conectar através dele ao endereço IP dado na porta 6510.
