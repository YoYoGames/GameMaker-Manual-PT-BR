network_send_packet

Com esta função você pode enviar um "pacote" de dados através da rede. A função leva o id do soquete para conectar e então você deve fornecer o id do buffer que contém os dados a serem enviados (para mais informações em buffers veja Referência - Buffers) e finalmente o tamanho (em bytes) do pacote de dados. Os pacotes enviados com esta função são formatados de forma que o jogo GameMaker que recebe os dados possa "dividir" os pacotes corretamente, e a função retornará o número de bytes de dados enviados, ou um número inferior a 0 se o envio falhar. Vale notar que o tamanho final dos dados enviados que são devolvidos por esta função também incluirá as informações do cabeçalho GameMaker, que é um número adicional de 12 bytes.

Sintaxe:

network_send_packet(socket, bufferid, size);

ArgumentoTipo Descrição
socket The id of the socket to use.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

Devoluções:

Real

Exemplo:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

A informação acima carrega um dado previamente gravado buffer na memória e retorna o id buffer para ser armazenado na variável "buff". Este completo buffer é então enviado como um pacote pela rede usando o soquete identificado pela variável "sock".
