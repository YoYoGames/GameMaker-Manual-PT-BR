network_send_udp

Com esta função você pode enviar dados através da rede usando o UDP para um server. A função leva a identificação do soquete para conectar, a URL para conectar e a porta para usar. Você deve então fornecer o id do buffer que contém os dados a serem enviados (para mais informações em buffers veja Referência - Buffers) e finalmente o tamanho (em bytes) dos dados. O UDP é "sem conexão" na medida em que você não faz realmente uma conexão, você apenas envia um pacote diretamente para um IP, e o server recebe os dados recebidos de um endereço IP e tem que lidar com ele "como está". A função retornará o número de bytes de dados enviados, ou um número inferior a 0 se o envio falhar. Vale notar que o tamanho final dos dados enviados que são devolvidos por esta função também incluirá a informação do cabeçalho GameMaker, que é um número adicional de 12 bytes.

Esta função não funcionará quando utilizada em um projeto em execução no alvo HTML5, e os projetos HTML5 também não poderão receber UDP.

Sintaxe:

network_send_udp(socket, url, port, bufferid, size);

Argumento Tipo Descrição
socket ID da tomada de rede The id of the socket to use.
url Cordão The url or IP to connect to (a string).
port Real The port to connect to.
bufferid Identificação do tampão The id of the buffer to get the data from.
size Real The size (in bytes) of the data.

Devoluções:

Real

Exemplo:

network_send_udp(sock, "www.macsweeneygames.com", 6510, buff, buffer_tell(buff));

O código acima enviará um pacote UDP para o endereço server definido pela URL na porta 6510. Os dados são extraídos do buffer indexado na variável "buff".
