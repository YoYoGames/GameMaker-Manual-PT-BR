network_send_raw

Com esta função, você pode enviar um pacote de dados "bruto" através da rede. A função leva o id do soquete para conectar e então você deve fornecer o id do buffer que contém os dados a serem enviados (para mais informações em buffers veja Referência - Buffers) e finalmente o tamanho (em bytes) do pacote de dados. Os dados enviados não são formatados pelo GameMaker de forma alguma e os dispositivos receptores receberão os dados como um fluxo, o que significa que você mesmo terá que lidar com eles. A função retornará o número de bytes de dados enviados, ou um número inferior a 0 se o envio falhar.

Argumento de Opções

O último argumento é opcional, e só é usado com WebSockets. Ele permite que você escolha entre o envio de dados binários ou de texto. Qualquer uma destas constantes pode ser especificada neste argumento:

Tipo de envio de rede Constante
Constante Descrição
network_send_binary Enviar uma mensagem binária
network_send_text Enviar uma mensagem de texto

As APIs para algumas plataformas só aceitam mensagens de texto quando se utiliza WebSockets (por exemplo, Twitch), de modo que a constante network_send_text pode ser utilizada em tais casos. Se este argumento não for especificado, os dados binários são enviados por padrão.

Sintaxe:

network_send_raw(socket, bufferid, size, [options]);

Argumento Tipo Descrição
socket ID da tomada de rede The id of the socket to use.
bufferid Identificação do tampão The id of the buffer to get the data from.
size Real The size (in bytes) of the data.
options Tipo de envio de rede Constante OPTIONAL Usado para WebSockets para escolher entre texto e dados binários; se não especificado, os dados binários são enviados.

Devoluções:

Real

Exemplo:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

A informação acima carrega um dado previamente gravado buffer na memória e retorna o id buffer para ser armazenado na variável "buff". Este completo buffer é então enviado como um pacote de dados brutos pela rede usando o soquete identificado pela variável "sock".
