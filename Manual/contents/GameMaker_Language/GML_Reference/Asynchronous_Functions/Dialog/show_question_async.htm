show_question_async

Esta função abre uma janela e exibe ao usuário a pergunta que você define na função. Esta é uma função assíncrona, e como tal o GameMaker não bloqueia o dispositivo em que está sendo executado enquanto espera por uma resposta, mas sim continua a executar eventos como de costume. A função tem dois botões que mostram "Sim" e "Não", e uma vez que o usuário tenha pressionado um, um evento de Diálogo Assíncrono é acionado, o qual, somente para a duração desse evento, terá um mapa DS armazenado na variável async_load.

Este mapa conterá as duas chaves, "id" e "status". "id" é o valor que foi retornado pela função quando chamado, enquanto o "status" será true ou false para "Yes" e "No" respectivamente.

NOTA Esta função é apenas para depuração e testee não deve ser usada em jogos lançados. Para isso, você deve criar uma interface de usuário personalizada para receber informações dos jogadores, para que você tenha controle total sobre a aparência e o comportamento dos diálogos.

Sintaxe:

show_question_async(string);

Argumento Tipo Descrição
string Cordão The question to ask to the user.

Devoluções:

Async Request ID

Exemplo ampliado:

O evento de pressão do mouse esquerdo (por exemplo) do object que está mostrando a mensagem teria o seguinte código:

msg = show_question_async("Do you want to buy some armour for " + string(armour[0, 5]) + "coins?");

O acima exposto mostrará uma pergunta com o dado string, solicitando que o usuário pressione "sim" ou "não". A identificação da função é armazenada na variável "msg" e será usada no evento de Diálogo Assíncrono, como mostrado abaixo:

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if i_d == msg
{
    if ds_map_find_value(async_load, "status")
    {
        coins -= armour[0,5];
        global.protection += armour[0,0];
    }
}

O código acima verifica a chave "id" do retorno DS Map contra o valor armazenado na variável "msg". Se forem os mesmos, ele verifica se um dos dois botões foi pressionado e se retorna true, então deduz um valor de uma variável e adiciona um valor a uma variável global também.
