time_source_destroy

Esta função destruirá a fonte de tempo dada.

Opcionalmente, você pode especificar se todas as fontes de tempo infantil também devem ser excluídas.

Sintaxe:

time_source_destroy(id, [destroy_tree]);

Argumento Tipo Descrição
id Identificação da fonte de tempo The Time Source para destruir
destroy_tree Booleano OPTIONAL Se destruir as fontes de tempo infantil também, é falso por padrão

Devoluções:

N/A

Exemplo:

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

O código Criar evento cria uma nova Fonte de Tempo que expira uma vez por segundo e se repete indefinidamente.

O código do evento Clean Up destrói essa Fonte de Tempo, pois ela não será necessária depois que a instância for destruída.
