ds_grid_value_disk_x

Com esta função, você pode obter a coordenada x (dentro da região em forma de disco da grade) do valor que está sendo procurado. Você dá o índice da grade DS (como retornado por ds_grid_create()) junto com as posições x/y para a célula central do disco. Em seguida, você dá o raio (como um valor inteiro) ao redor da célula central para pesquisar, antes de fornecer o valor a ser pesquisado. Se o valor a ser pesquisado não existir, então a função retornará -1.

Sintaxe:

ds_grid_value_disk_x(index, xm, ym, r, val);

ArgumentoTipo Descrição
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to find.

Devoluções:

Exemplo:

if ds_grid_value_disk_exists(grid, 5, 5, 5, val)
{
    xpos = ds_grid_value_disk_x(grid, 5, 5, 5, val);
    ypos = ds_grid_value_disk_y(grid, 5, 5, 5, val);
}

O código acima verifica uma ds_grid para um valor específico dentro de uma região de disco. se for encontrado, ele então armazena a posição x e y do valor em duas variáveis para uso posterior.
