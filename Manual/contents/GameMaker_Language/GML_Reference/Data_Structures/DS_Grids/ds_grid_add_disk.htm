ds_grid_add_disk

Esta função pode ser usada para adicionar um determinado valor (real ou string) a todos os valores das células encontradas dentro da área definida em disco de uma grade. O valor a ser adicionado deve ser do mesmo tipo do que é mantido dentro das células da grade, ou seja: não se pode adicionar uma string a um real ou vice-versa, e para strings isto corresponde a concatenação.

Sintaxe:

ds_grid_add_disk(index, xm, ym, r, val);

ArgumentoTipo Descrição
index The index of the grid.
xm The x position of the disk on the grid.
ym The y position of the disk on the grid.
r The radius of the disk on the grid.
val The value to add to the cells within the disk.

Devoluções:

Exemplo:

ds_grid_add_disk(grid, 7, 6, 5, 2)

Isto adicionaria 2 a todos os valores mantidos nas células dentro da área definida do disco da grade DS referenciada pela variável "grid".
