ds_grid_read

Esta função pode ser usada para converter um string que foi criado anteriormente pela função ds_grid_write() de volta a uma grade DS. A grade DS deve ter sido criada anteriormente (veja o exemplo abaixo).

Observe que se a DS string especificada foi escrita pelo GameMaker: Studio 1.2.x runtime (ou mais antigo), você deve especificar o argumento opcional "legado", definindo-o para true, pois o formato string mudou depois disso.

Sintaxe:

ds_grid_read(index, string [, legacy]);

Argumento Tipo Descrição
index DS Grid ID The index of the grid to read.
string Cordão The string to read into the DS grid.
legacy Booleano OPTIONAL Pode ser true ou false ou omitido completamente.

Devoluções:

N/A

Exemplo:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

O código acima cria uma grade DS com base no tamanho do room (cada 32x32 quadrados de pixels representa uma célula da grade) e então lê um conjunto de dados da grade previamente salvos de um arquivo ini para a nova grade DS.
