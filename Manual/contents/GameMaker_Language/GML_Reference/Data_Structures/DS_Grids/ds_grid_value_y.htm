ds_grid_value_y

Com esta função você pode obter a coordenada y (dentro de uma determinada região de grade retangular) do valor que está sendo procurado. Você dá o índice da grade DS (como retornado por ds_grid_create()) junto com as posições x/y para os cantos superior esquerdo e inferior direito da região da grade para procurar, antes de fornecer o valor a ser procurado. Se o valor a ser pesquisado não existir, então a função retornará -1.

Sintaxe:

ds_grid_value_y(index, x1, y1, x2, y2, val);

ArgumentoTipo Descrição
index The index of the grid.
x1 The x position of the left of the region in the grid, from 0 to (grid width - 1).
y1 The y position of the top of the region in the grid, from 0 to (grid height - 1).
x2 The x position of the right of the region in the grid, from 0 to (grid width - 1).
y2 The y position of the bottom of the region in the grid, from 0 to (grid height - 1)
val The value to find.

Devoluções:

Exemplo:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
{
    xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
    ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
}

O código acima verifica uma ds_grid para um valor específico dentro de uma região retangular. se for encontrado, ele então armazena a posição x e y do valor em duas variáveis para uso posterior.
