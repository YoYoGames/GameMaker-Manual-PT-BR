ds_stack_size

Esta função devolverá o "tamanho" da pilha, ou seja: o número de itens que foram empurrados para ela.

Sintaxe:

ds_stack_size(id);

ArgumentoTipo Descrição
id The id of the data structure to check.

Devoluções:

Exemplo:

if !ds_stack_empty(control_stack)
{
    num = ds_stack_size(control_stack);
}

O código acima verifica uma pilha DS para ver se ela está vazia ou não, e se não estiver, recebe o número de itens que contém e armazena o valor em uma variável.
