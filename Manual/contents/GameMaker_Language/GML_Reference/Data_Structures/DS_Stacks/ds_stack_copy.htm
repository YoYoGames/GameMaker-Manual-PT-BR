ds_stack_copy

Esta função pode ser usada para copiar o conteúdo de uma pilha para outra. Note que isto NÃO remove o conteúdo da pilha original, nem destrói a pilha original. Ao utilizar esta função, a pilha para a qual está sendo copiada deve ter sido previamente criada e se continha algum item antes da cópia, então estes serão limpos primeiro (significando que esta informação será perdida).

Sintaxe:

ds_stack_copy(id, source);

Argumento Tipo Descrição
id DS Stack ID The id of the new stack.
source DS Stack ID The original stack to copy from.

Devoluções:

N/A

Exemplo:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    stack = ds_stack_create();
    ds_stack_copy(stack, other.stack);
}

A função acima cria uma nova instância e então, nessa instância, cria uma nova pilha de DS e copia o conteúdo da pilha na instância que executa o bloco de código, para a nova pilha de instância criada.
