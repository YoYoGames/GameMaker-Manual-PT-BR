ds_stack_pop

Esta função irá estourar o valor superior da pilha DS, retirando-o da pilha e retornando o valor a ser armazenado em uma variável. Se a pilha estiver vazia, então a função retornará a constante undefined, caso contrário retornará o valor real ou string contido na pilha.

Sintaxe:

ds_stack_pop(id);

ArgumentoTipo Descrição
id The id of the data structure to pop from.

Devoluções:

Variable (Data type value that is stored in the stack) or undefined

Exemplo:

if !ds_stack_empty(move_stack)
{
    var xx = ds_stack_pop(move_stack);
    var yy = ds_stack_pop(move_stack);
    move_towards_point(xx, yy, 4);
}

O código acima verifica a pilha DS indexada na variável "move_stack" para ver se ela está vazia, e se não estiver, ela então estala os dois valores superiores da pilha e os usa para definir uma direção para o movimento.
