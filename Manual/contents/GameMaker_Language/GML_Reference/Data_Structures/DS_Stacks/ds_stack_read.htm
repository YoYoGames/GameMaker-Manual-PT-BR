ds_stack_read

Com esta função você pode recriar uma pilha DS salva (uma que foi escrita anteriormente como um string usando ds_stack_write()). É preciso primeiro criar uma nova pilha de DS para ler o string, e se a pilha de DS já existir e tiver informações armazenadas nela, então isto será limpo antes da leitura. Esta função é de vital importância ao criar mecanismos de salvamento/carga para seu jogo.

Observe que se a DS string especificada foi escrita pelo GameMaker: Studio 1.2.x runtime (ou mais antigo), você deve especificar o argumento opcional "legado", definindo-o para true, pois o formato string mudou depois disso.

Sintaxe:

ds_stack_read(id, str [, legacy]);

Argumento Tipo Descrição
id DS Stack ID The id of the data structure to read into.
str Cordão The string to read from.
legacy Booleano OPTIONAL Pode ser true ou false ou omitido completamente.

Devoluções:

N/A

Exemplo:

stack = ds_stack_create();
ini_open("save.ini");
var str = ini_read_string("Stacks", "0", "");
if (str != "")
{
    ds_stack_read(stack, str);
}
ini_close();

O código acima cria uma pilha e armazena o índice na variável "pilha". Em seguida, ele abre um arquivo ini e lê um string a partir dele, verificando se o string não é devolvido como vazio primeiro. Este string é então lido na pilha DS recém-criada.
