ds_list_empty

Com esta função você pode verificar a lista DS dada para ver se ela está vazia (retorna true) ou não (retorna false).

Sintaxe:

ds_list_empty(id);

Argumento Tipo Descrição
id DS List id The id of the data structure to check.

Devoluções:

Exemplo:

if (count == 15( && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a lista DS indexada na variável "command_list", define um alarme e redefine a variável para 0.
