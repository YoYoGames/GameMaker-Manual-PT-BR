ds_list_set

Esta função pode ser usada para definir uma entrada na lista previamente adicionada. Você dá a identificação da lista (como retornada quando você criou a lista) e a posição dentro da lista a ser definida, assim como o valor a ser definido. Note que se a entrada sendo definida estiver fora dos limites da lista (ou seja, você define a entrada da lista 20 mas a lista atual contém apenas 10 entradas), então a lista será preenchida para a posição dada e cada entrada será definida a 0. Esta função é a mesma que a de usar o acessor da lista DS.

Sintaxe:

ds_list_set(id, pos, val);

Argumento Tipo Descrição
id DS List ID The id of the list to add to.
pos Variável The position within the list to set.
[val2, ... max_val] Variável The value to set in the list.

Devoluções:

N/A

Exemplo:

for (var i = 0; i < ds_list_size(list); i++;)
{
    ds_list_set(list, i, -1);
}

O código acima adicionará o valor armazenado na variável "score" à lista indexada na variável "sc_list".
