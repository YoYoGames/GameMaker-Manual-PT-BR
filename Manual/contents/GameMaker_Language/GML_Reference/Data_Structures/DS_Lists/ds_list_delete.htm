ds_list_delete

Com esta função você pode remover o valor armazenado em uma posição específica dentro da lista.

Sintaxe:

ds_list_delete(id, pos);

ArgumentoTipo Descrição
id The id of the list to change.
pos Where in the list to delete the value.

Devoluções:

Exemplo:

if (ds_list_size(sc_list) > 10)
{
    while (ds_list_size(sc_list) > 10)
    {
        ds_list_delete(sc_list, 0);
    }
}

O código acima verifica o tamanho de uma lista DS e se for maior que dez, ele loops através da lista retirando o valor superior (posição 0) até que a lista tenha apenas 10 entradas.
