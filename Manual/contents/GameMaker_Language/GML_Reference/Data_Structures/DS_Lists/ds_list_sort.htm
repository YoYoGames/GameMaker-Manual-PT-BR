ds_list_sort

Com esta função você pode ordenar todos os valores dentro de uma lista, seja em ordem ascendente ou descendente. Se a lista contiver strings, estes serão ordenados alfabeticamente, com base no alfabeto inglês de 26 letras.

Sintaxe:

ds_list_sort(id, ascend);

ArgumentoTipo Descrição
id The id of the list to sort.
ascend Whether the values should be ascending (true) ou ordem decrescente (false).

Devoluções:

Exemplo:

if newgame
{
    ds_list_sort(name_list, true);
}

O código acima ordenará a lista indexada na variável "name_list" se a variável "newgame" estiver marcada como true.
