ds_list_clear

Com esta função você pode limpar todos os dados da estrutura de dados da lista fornecida. Isto NÃO destrói a estrutura de dados (para isso você deve usar ds_list_destroy()) ele apenas limpa todos os dados e torna a lista vazia (zero em tamanho). Note que a limpeza de uma lista desreferenciará qualquer estrutura de dados nela armazenada dando um vazamento de memória, portanto, seria necessário passar pela lista e destruir todos os itens da estrutura de dados manualmente antes de limpá-la para evitar isso. A única vez que isto não é necessário é quando você tiver marcado qualquer item da lista como outra lista DS ou como um mapa DS, neste caso estes itens serão destruídos (não limpos!) e sua memória será limpa automaticamente quando o pai for limpo.

Sintaxe:

ds_list_clear(id);

ArgumentoTipo Descrição
id The id of the data structure to clear.

Devoluções:

Exemplo:

if (count == 15) && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a lista DS indexada na variável "command_list", define um alarme e redefine a variável para 0.
