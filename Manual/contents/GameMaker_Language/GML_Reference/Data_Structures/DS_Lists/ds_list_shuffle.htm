ds_list_shuffle

Com esta função você pode embaralhar uma lista, que reordenará todos os valores dos componentes em posições aleatórias daqueles em que eles foram originalmente adicionados à lista.

NOTA: Esta função embaralhará os itens da lista para as mesmas posições toda vez que o jogo for executado novamente devido ao fato de que o GameMaker gera a mesma semente inicial aleatória toda vez para tornar o código debugging uma tarefa muito mais fácil. Para evitar este comportamento, use randomise() no início de seu jogo. Isto só é verdade ao testar e debugging o jogo, pois o pacote executável final não mostrará este comportamento e será aleatório a cada jogada.

Sintaxe:

ds_list_shuffle(id);

ArgumentoTipo Descrição
id The id of the list to shuffle.

Devoluções:

Exemplo:

if restart
{
    ds_list_shuffle(card_list);
}

O código acima baralhará a lista indexada na variável "card_list" se a variável "restart" estiver marcada como true.
