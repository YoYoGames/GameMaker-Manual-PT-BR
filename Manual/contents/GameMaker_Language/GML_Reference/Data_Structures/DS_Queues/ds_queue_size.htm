ds_queue_size

Esta função devolverá o "tamanho" do queue, ou seja: o número de itens que foram enfileirados nele.

Sintaxe:

ds_queue_size(id);

ArgumentoTipo Descrição
id The id of the data structure to check.

Devoluções:

Exemplo:

if !ds_queue_empty(control_queue)
{
    num = ds_queue_size(control_queue);
}

O código acima verifica uma fila DS para ver se ela está vazia ou não, e se não estiver, recebe o número de itens que contém e armazena o valor em uma variável.
