ds_queue_dequeue

Esta função irá decifrar o valor da cabeça do DS queue, retirando-o do queue e retornando o valor a ser armazenado em uma variável. Se o queue estiver vazio, então a função retornará a constante undefined, caso contrário, retornará o valor contido no queue.

Sintaxe:

ds_queue_dequeue(id);

ArgumentoTipo Descrição
id The id of the queue to dequeue from.

Devoluções:

Variable (Data type value stored in the queue) or undefined

Exemplo:

if !ds_queue_empty(move_queue)
{
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
}

O código acima verifica a fila DS indexada na variável "move_queue" para ver se ela está vazia e, se não estiver, ela então retira os dois valores da cabeça do queue e os usa para definir uma direção para o movimento.
