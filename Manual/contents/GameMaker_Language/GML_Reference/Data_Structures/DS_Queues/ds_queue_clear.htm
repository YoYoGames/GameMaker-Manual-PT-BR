ds_queue_clear

Com esta função você pode limpar todos os dados da estrutura de dados queue fornecida. Isto NÃO destrói a estrutura de dados (para isso você deve usar ds_queue_destroy()) ele só limpa todos os dados dele e devolve um vazio queue.

Sintaxe:

ds_queue_clear(id);

ArgumentoTipo Descrição
id The id of the data structure to clear.

Devoluções:

Exemplo:

if (count = 15) && (!ds_queue_empty(command_queue))
{
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a fila DS indexada na variável "command_queue", define um alarme e redefine a variável para 0.
