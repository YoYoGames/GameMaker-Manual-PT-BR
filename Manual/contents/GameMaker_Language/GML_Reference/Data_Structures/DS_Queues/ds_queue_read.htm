ds_queue_read

Com esta função você pode recriar uma fila de DS salva (uma que foi escrita anteriormente como um string usando ds_queue_write()). É preciso primeiro criar uma nova fila de DS para ler o string, e se a fila de DS já existir e tiver informações armazenadas nela, então isto será liberado antes da leitura. Esta função é de vital importância ao criar mecanismos de salvamento/carga para seu jogo.

Observe que se a DS string especificada foi escrita pelo GameMaker: Studio 1.2.x runtime (ou mais antigo), você deve especificar o argumento opcional "legado", definindo-o para true, pois o formato string mudou depois disso.

Sintaxe:

ds_queue_read(id, str [, legacy]);

Argumento Tipo Descrição
id DS Queue ID The id of the data structure to read into.
str Cordão The string to read from.
legacy Booleano OPTIONAL Pode ser true ou false ou omitido completamente.

Devoluções:

N/A

Exemplo:

queue = ds_queue_create();
ini_open("save.ini");
var str = ini_read_string("Queues", "0", "");
if (str != "")
{
    ds_queue_read(queue, str);
}
ini_close();

O código acima cria um queue e armazena o índice na variável "fila". Em seguida, abre um arquivo ini e lê um string a partir dele, verificando se o string não é devolvido como vazio primeiro. Este string é então lido no DS queue recém-criado.
