ds_queue_head

Esta função só lerá o primeiro valor do queue (aquele que está "à cabeça"). Ela não irá decifrar o valor, o que significa que ainda poderá ser lido no futuro por esta função ou pelo ds_queue_dequeue(). Se o queue estiver vazio, então a função retornará a constante undefined, caso contrário retornará o valor real ou string contido no queue.

Sintaxe:

ds_queue_head(id);

ArgumentoTipo Descrição
id The id of the data structure to read from.

Devoluções:

(Data type value stored in the queue)

Exemplo:

num = ds_queue_head(control_queue);

O código acima lerá o valor do queue indexado na variável "control_queue" e armazenará o valor de retorno na variável "num".
