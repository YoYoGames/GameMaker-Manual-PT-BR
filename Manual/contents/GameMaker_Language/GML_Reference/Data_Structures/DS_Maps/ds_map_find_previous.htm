ds_map_find_previous

Esta função retorna a chave anterior armazenada no mapa DS antes da chave especificada na função. Isto pode ser útil se você tiver que iterar através do mapa DS procurando por algo, mas deve ser evitado se possível, pois pode ser lento. Se tal chave não existir, então a função retornará undefined. Você deve sempre verificar isto usando a função is_undefined() função.

Sintaxe:

ds_map_find_previous(id, key);

ArgumentoTipo Descrição
id The id of the map to use.
key The key to find the previous one to.

Devoluções:

Variable or undefined

Exemplo:

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

O código acima cria algumas variáveis temporárias e então obtém o tamanho ds_map e encontra a última chave como armazenada pelo computador no mapa. Ele então usa um for loop para iterar de volta através do ds_map procurando o valor da chave "ouro". Se ele o encontrar, ele rompe o loop.
