ds_map_write

Esta função transformará os dados do mapa DS do mapa especificado no formato string que pode então ser escrito em um arquivo *.ini ou *.txt para facilitar o armazenamento. Este string pode então ser lido posteriormente em um novo ds_map usando ds_map_read().

OBSERVAÇÃO: O retorno string não é uma leitura humana string, mas sim um despejo do conteúdo da estrutura de dados.

Sintaxe:

ds_map_write(id);

ArgumentoTipo Descrição
id The id of the data structure to use

Devoluções:

Exemplo:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

O código acima abre um arquivo ini pronto para ser escrito. Em seguida, ele usa ds_map_write() para gerar um string que é armazenado na variável temporária "t_string". Finalmente, ele escreve esse string para o arquivo ini antes de fechá-lo.
