ds_map_read

Esta função terá um string que foi previamente criado pela função ds_map_write() e depois leia-o em um mapa DS previamente criado. Se o mapa em que o string está sendo lido contiver qualquer par chave/valor, estes serão limpos primeiro antes que o mapa salvo seja reconstruído.

Observe que se a DS string especificada foi escrita pelo GameMaker: Studio 1.2.x runtime (ou mais antigo), você deve especificar o argumento opcional "legado", definindo-o para true, pois o formato string mudou depois disso.

Sintaxe:

ds_map_read(id, str, [legacy]);

Argumento Tipo Descrição
id DS Map ID The id of the data structure to read the string into
str Cordão The string to read
legacy Booleano OPTIONAL Pode ser true ou false ou omitido completamente.

Devoluções:

N/A

Exemplo:

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

O código acima cria um novo mapa DS e armazena seu índice de identificação na variável "inventário". Em seguida, ele abre um arquivo ini e lê um string desse arquivo para a variável temporária "t_string". Finalmente, ele verifica se o string é válido (não o valor padrão do ini "") e se é, então, lê o string no mapa DS recém-criado antes de fechar o ini novamente.
