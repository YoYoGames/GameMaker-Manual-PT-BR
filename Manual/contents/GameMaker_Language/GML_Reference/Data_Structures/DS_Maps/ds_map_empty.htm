ds_map_empty

Esta função simplesmente retornará false se o mapa DS especificado (previamente criado) contiver algum par chave/valor, ou true se não contiver.

Sintaxe:

ds_map_empty(id);

ArgumentoTipo Descrição
id The id of the data structure to check.

Devoluções:

Exemplo:

if ds_map_empty(inventory)
{
    weight = 0;
}

O código acima verifica se o mapa DS indexado na variável "inventário" tem algum par chave/valor e se ele não define a variável "peso" como 0.
