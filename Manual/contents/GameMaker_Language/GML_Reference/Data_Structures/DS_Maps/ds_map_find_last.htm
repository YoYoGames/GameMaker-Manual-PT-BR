ds_map_find_last

Esta função retorna a última chave armazenada no mapa DS. Esta não é a última chave na ordem em que você as adicionou! Os mapas DS não são armazenados de forma linear, para isso são usadas listas DS, então tudo o que isto faz é encontrar a última chave armazenada pelo computador. Isto pode ser útil se você tiver que iterar através do mapa DS procurando por algo, mas deve ser evitado se possível, pois pode ser lento.

Note que esta função retornará undefined se o mapa DS dado estiver vazio.

Sintaxe:

ds_map_find_last(id);

ArgumentoTipo Descrição
id The id of the map to use.

Devoluções:

Variable or undefined

Exemplo:

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

O código acima cria algumas variáveis temporárias e depois obtém o tamanho do mapa DS e encontra a última chave como armazenada pelo computador no mapa. Ele então usa um for loop para iterar de volta através do mapa DS procurando o valor da chave "ouro". Se ele o encontrar, ele rompe o loop.
