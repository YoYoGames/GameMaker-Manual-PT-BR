ds_map_values_to_array

Com esta função você pode recuperar todos os valores que um mapa DS contém para cada chave no mapa. Você fornece o DS map ID para obter as chaves de (como retornado por ds_map_create()) e a função retornará um array onde cada entrada no array é um valor do mapa DS (por exemplo, se o mapa tiver 10 chaves, então um array de 10 entradas será retornado com cada entrada correspondente a um valor para uma chave). A função tem um segundo argumento opcional onde você pode fornecer um array que você criou, caso em que os dados do valor da chave do mapa serão anexados a qualquer dado existente no array. Note que a função modificará o array fornecido diretamente, mas também retornará uma referência a ele (ou uma referência a um novo array se nenhum for fornecido).

Sintaxe:

ds_map_values_to_array(id, [array])

Argumento Tipo Descrição
id DS Map ID The id of the map to use.
[array] Array OPTIONAL A matriz para anexar os valores do mapa DS.

Devoluções:

Array

Exemplo:

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
var _length = array_length(_values);

for (var i = 0; i < _length; i ++)
{
    _total += _values[i];
}

draw_text(32, 32, "Total score for all levels: " + string(_total));

O acima recupera todos os valores de um mapa DS para cada chave no mapa e os armazena em uma variável local como uma matriz. Ele então loops através do array e adiciona cada um dos valores a uma variável local que é então usada para exibir o total de todos os valores.
