ds_map_existe

Esta função retornará true se a chave especificada existir no mapa DS (previamente criado) , e false se não existir.

Sintaxe:

ds_map_exists(id, key);

Argumento Tipo Descrição
id DS Map ID The id of the data structure to check
key Cordão The key to check for

Devoluções:

Boolean

Exemplo:

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

O código acima verificará o mapa DS indexado na variável "inventário" para a chave "poções" e, se não existir, o adicionará ao mapa.
