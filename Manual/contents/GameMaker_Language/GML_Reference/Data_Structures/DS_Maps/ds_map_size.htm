ds_map_size

Com esta função você pode encontrar quantos pares de chaves/valores o mapa DS (anteriormente criado) contém.

Sintaxe:

ds_map_size(id);

ArgumentoTipo Descrição
id The id of the data structure to check.

Devoluções:

Exemplo:

if (ds_map_size(inventory) > 49)
{
    full = true;
}

O código acima verificará o tamanho do mapa DS (ou seja: número de pares chave/valor) e se for maior que 49, define a variável "cheio" para true.
