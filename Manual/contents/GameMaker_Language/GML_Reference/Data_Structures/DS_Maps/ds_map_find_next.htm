ds_map_find_next

Esta função retorna a próxima chave armazenada no mapa DS após a chave especificada na função. Isto pode ser útil se você tiver que iterar através do mapa DS procurando por algo, mas deve ser evitado se possível, pois pode ser lento. Se tal chave não existir, então a função retornará undefined. Você deve sempre verificar isto usando a função is_undefined() função.

Sintaxe:

ds_map_find_next(id, key);

ArgumentoTipo Descrição
id The id of the map to use.
key The key to find the next one to.

Devoluções:

Variable or undefined

Exemplo:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if (key != "gold")
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

O código acima cria algumas variáveis temporárias e depois obtém o tamanho do mapa DS e encontra a primeira chave como armazenada pelo computador no mapa. Em seguida, ele usa um for loop para iterar através do mapa DS procurando o valor da chave "ouro". Se ele o encontrar, ele rompe o loop.
