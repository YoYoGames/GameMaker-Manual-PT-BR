ds_map_replace

Com esta função você pode alterar o valor para a chave dada dentro de um mapa DS. Você fornece o índice para o mapa (como retornado pela função ds_map_create()) e depois a chave para substituir - ou um string ou um inteiro - e o valor para substituir o valor chave por. Se a chave dada não existir, ela será criada para você, e se existir, então o valor atual será substituído pelo novo valor. valor. A função retornará true se a chave existir e o valor for substituído, e false se a chave não existir e uma nova chave tiver sido criada com o valor.

Sintaxe:

ds_map_replace( id, key, val );

ArgumentoTipo Descrição
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

Devoluções:

Exemplo:

ds_map_replace(inventory, "torso", 55);

O código acima procura no mapa DS a chave "torso" e quando ela o encontra (ou é criada se não existir) o valor atual é substituído pelo especificado.
