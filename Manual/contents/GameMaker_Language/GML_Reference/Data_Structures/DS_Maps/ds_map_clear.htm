ds_map_clear

Esta função limpará o mapa DS (previamente criado) de todos os pares chave/valor, mas não apagará o próprio mapa DS da memória. Para isso, você deve usar a função ds_map_destroy(). Note que a limpeza de um mapa desreferenciará quaisquer estruturas de dados armazenadas no mapa, dando um vazamento de memória, portanto você precisaria passar por ele e destruir todos os itens da estrutura de dados manualmente antes de limpar o mapa para evitar isso. A única vez que isto não é necessário é quando você tiver marcado qualquer item no mapa como uma lista DS ou como outro mapa DS, neste caso, estes itens serão destruídos (não limpos!) e sua memória será limpa automaticamente quando o mapa pai for limpo.

Sintaxe:

ds_map_clear(id);

ArgumentoTipo Descrição
id The id of the data structure to clear.

Devoluções:

Exemplo:

if global.new_game
{
    ds_map_clear(inventory);
}

A função acima verificará uma variável global para ver se ela é verdadeira e, se for, então o mapa DS indexado na variável de instância "inventário" será limpo de todos os pares chave/valor (mas não será destruído).
