ds_priority_empty

Com esta função você pode verificar a prioridade dada ao DS queue para ver se ele está vazio (retorna true) ou não (retorna false).

Sintaxe:

ds_priority_empty(id);

ArgumentoTipo Descrição
id The id of the data structure to check.

Devoluções:

Exemplo:

if (count == 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

O código acima verifica uma variável para ver se ela atingiu um valor específico e se ela tem a prioridade DS queue indexada na variável "command_queue", define um alarme e redefine a variável para 0.
