ds_priority_create

Esta função criará uma nova estrutura de dados prioritária queue e retornará o valor do índice. Este valor deve ser armazenado em uma variável e utilizado em todas as outras chamadas de função relacionadas com a prioridade queue.

IMPORTANTE! Quando você cria uma estrutura de dados, o valor do índice para identificá-lo é um valor inteiro começando em 0. Isso significa que estruturas de dados de diferentes tipos podem ter o mesmo valor de índice, portanto, em caso de dúvida, você deve estar usando o ds_exists() antes de acessá-los. Observe também que os índices são reutilizados, de modo que um valor de índice de estrutura de dados destruída pode ser usado por um índice recém-criado posteriormente.

Sintaxe:

ds_priority_create();

Devoluções:

DS Priority ID

Exemplo:

p_queue = ds_priority_create();

Isto criará uma nova prioridade queue e atribuirá seu id de índice à variável de instância "p_queue".
