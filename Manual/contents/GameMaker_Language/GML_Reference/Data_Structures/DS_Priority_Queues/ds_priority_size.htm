ds_priority_size

Esta função devolverá o "tamanho" da prioridade queue, ou seja: o número de itens que foram priorizados nela.

Sintaxe:

ds_priority_size(id);

ArgumentoTipo Descrição
id The id of the data structure to check.

Devoluções:

Exemplo:

if !ds_priority_empty(control_priority)
{
    num = ds_priority_size(control_priority);
}

O código acima verifica uma prioridade DS queue para ver se ela está vazia ou não, e se não estiver, recebe o número de itens que contém e armazena o valor em uma variável.
