ds_priority_change_priority

Esta função terá um determinado valor e mudará sua prioridade dentro da prioridade referenciada queue. O valor dado já deve existir na prioridade queue.

Sintaxe:

ds_priority_change_priority(id, val, priority);

Argumento Tipo Descrição
id DS Identificação prioritária The id of the priority queue to change.
val Variável The value to change the priority of.
priority Real The new priority of the value.

Devoluções:

N/A

Exemplo:

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

O código acima verifica uma variável global e, se estiver abaixo de um determinado valor, alterará então a prioridade do índice de funções script mantido na prioridade queue.
