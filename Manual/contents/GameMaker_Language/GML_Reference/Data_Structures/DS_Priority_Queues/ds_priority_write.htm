ds_priority_write

Esta função retorna um string que pode então ser armazenado ou transferido para outra estrutura de dados usando o ds_priority_read() função.

OBSERVAÇÃO: O retorno string não é uma leitura humana string, mas sim um despejo do conteúdo da estrutura de dados

Sintaxe:

ds_priority_write(id);

ArgumentoTipo Descrição
id The id of the data structure to check.

Devoluções:

Exemplo:

var str;
ini_open("save.ini");
str = ds_priority_write(p_queue);
ini_write_string("P_Queues", "0", str);
ds_priority_clear(p_queue);
ini_close();

O código acima abre um arquivo ini e então escreve um string contendo as informações armazenadas na prioridade DS queue indexadas na variável "p_queue". A prioridade queue é então limpa e o arquivo ini é fechado.
