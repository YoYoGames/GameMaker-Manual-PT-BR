ds_priority_delete_delete_min

Esta função retornará o valor que tem a menor prioridade no queue e então removerá o valor (e prioridade) da estrutura de dados. Se mais de um valor tiver a mesma prioridade, qualquer um deles poderá ser retornado em qualquer ordem, mas todos os outros valores com a mesma prioridade ainda estarão no queue.

Sintaxe:

ds_priority_delete_min(id);

ArgumentoTipo Descrição
id The id of the priority queue to use.

Devoluções:

(Data type stored in the priority)

Exemplo:

if ai_move
{
    script_execute(ds_priority_delete_min(ai_priority));
}

O código acima verifica uma variável de instância e se ela retornar true executará uma função script indexada na prioridade queue com o valor de prioridade mais baixo e então removerá esse script do queue.
