ds_priority_find_min

Com esta função você pode encontrar o valor armazenado na prioridade queue com a prioridade mais baixa, e se mais de um valor tem a mesma prioridade então qualquer um deles poderia ser devolvido em qualquer ordem. Entretanto, ao contrário de ds_priority_delete_min()esta função não removerá o valor do queue.

Sintaxe:

ds_priority_find_min(id);

ArgumentoTipo Descrição
id The id of the priority queue to use.

Devoluções:

Real or String

Exemplo:

if ai_move
{
    script_execute(ds_priority_find_min(ai_priority));
}

O código acima verifica uma variável de instância e se ela retornar true executará uma função script indexada na prioridade queue com o menor valor de prioridade.
